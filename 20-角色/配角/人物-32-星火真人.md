# 星火真人（金丹大能）

> **李雪莲之父** | **天人演武图作者** | **金丹大能回新手村** | **被万法宗"优化"**

## A. 基础档案

- **姓名**：星火真人（道号）
- **身份**：原万法宗金丹真人 → 已离职 / 退休
- **所属**：万法宗（已离职 / 培养下一代）
- **首次出场**：第 14 章（被司机介绍，作为李雪莲父亲）
- **实体登场**：**未实体登场**（仅通过电话 / 画作 / 背景信息登场）
- **年龄**：350 岁（第 17 章周天翊透露，去年已 350 岁）
- **金丹寿命上限**：400 年（已临近"优化"线）
- **外貌**：未直接描写（仅通过李雪莲的银发 + 红色火光礼服推测其"金丹世家"基因）

## B. 关键信息

### 金丹真人的"被优化"
- **金丹寿命上限**：400 年
- **星火真人去年已 350 岁**（被万法宗"优化"）
- **离职代价**：被回收**一半修为**
- **优化原因**：年龄 + 资源重新分配
- **退回一层**：无力支撑上面（昆墟上层）的消费
- **当前状态**：专心培养后代，调教弟子

### 张羽的吐槽
- "金丹大能回新手村养老？"
- 这是对金丹真人制度的黑色幽默

### 与天人演武图关系
- **作者**：星火真人亲手绘制
- **目的**：寻找一名 20 岁以下的少年天才弟子
- **筛选机制**：
  - **第一关**：道心（针对心中惊怖）
  - **第二关**：参悟（直面武道意念）
  - **第三关**：练成（最终通过）
- **隐藏武学**：天武炼心诀（以武道意念淬炼道心）

### 寻找弟子的历程
- **几年前**：一直想要寻到一名弟子来继承衣钵
- **要求颇高**：苦寻多年没有找到真正符合要求的弟子
- **天人演武图**："家父亲手绘下这幅天人演武图，并在这幅画中隐藏了一套武功"
- **收徒标准**：
  - 20 岁以下
  - 能练成图中武学
  - 全力培养至顶尖大学

## C. 性格与行为模式（推测）

- **要求高**：苦寻多年未找到符合要求的弟子
- **谨慎**：用天人演武图筛选弟子，避免直接见人
- **公正**：通过武道意念筛选思想灵活的弟子（"拜神心法能换着神拜，天勤心法既能苦修也能躺平"）
- **关爱女儿**：李雪莲继承衣钵线索 + 寻找天才弟子延续家族
- **制度受害者**：被万法宗"优化"是制度产物，反映昆墟十大宗门对修士的工具化

  - "若有 20 岁以下的少年英杰能练成其中武功，家父愿意收其为徒，并将之全力培养至顶尖大学"

### 第 19 章（道心考校）
- 李雪莲 + 仙韵集团嵩阳市总监讨论天人演武图
- 李雪莲的话代表星火真人的意志：
  - "心法万千，思想能契合心法固然好。但只有能不断调整自己的思想，适应环境变化，适应心法变化的人，才是真正的天才，不被心法所束缚"
  - "像之前就有一位学生在天人演武图的考验中不断经历考试考低分，最终克服了自己对低分的恐惧，磨砺出一种妄念思想"

### 第 20 章（参悟 + 演练）
- 多名学生参悟天人演武图
- 最终白真真获得"不错"评价（"高一学生 15 岁，半小时练成天武炼心诀"）

### 第 22 章（电话 · 拒绝张羽）
- **星火真人电话指示李雪莲**：
  - "**我已经找到了真正的天才**"
  - 指白真真
- **决策**：不收张羽为徒，但**个人名义**赠送 1 万灵币 + 天武炼心诀使用版权
- 拒绝原因推测：
  - 张羽道心等级 1 级（要求 3 级）
  - 白真真更符合"思想灵活 + 半小时练成"
- 张羽被婉拒但获得版权赠送 + 神辅推荐

## G. 视觉化要点（推测）

### 星火真人的"缺席"形象设计
- **未实体登场**：仅通过画作 + 电话 + 背景信息
- **李雪莲的遗传**：银发 + 红色火光礼服（推测来自星火真人的"火"属性道种）
- **天人演武图**：星火真人亲手绘制 → 画作中的"中年男子"身影 = 星火真人的形象
- **武道意念**：星火真人留下的一丝武道意念 → 在画作中可被参悟者感知

### 画中"中年男子"形象（第 18 章 P0 关键CG）
- **首次呈现**：模糊人影 → 接近 5 米后逐渐清晰
- **外貌**：面容刚毅的中年男子
- **动作**：演练一套深奥武学
- **关键一拳**：画中男子猛然一拳朝着参悟者隔空轰来
- **效果**：触发参悟者的道心考验幻境

### 星火真人的"金丹世家"基因（通过李雪莲推测）
- 银发：可能来自"火"属性仙道血脉
- 红色火光礼服：可能来自星火真人的道种
- 金丹家族的气质：富贵 / 威严 / 自信

## H. CG 制作要点

### P0 必做核心CG（第 18 章画中形象）
- **画中"中年男子"的视觉设计**：
  - 面容刚毅 / 体格健壮 / 道骨仙风
  - 演练武学时身体周围有金光 / 红色法力
  - 隔空一拳 → 拳风 / 冲击波
- **画作从模糊到清晰的渐变**：
  - 远观（5 米外）：模糊人影
  - 近观（5 米内）：逐渐清晰
  - 关键一拳：拳风扑面而来
- **不同参悟者看到的差异**：
  - 钱深：同学分数压迫的巨人
  - 炼天极：考最后一名 + 学渣逆袭
  - 白真真：催债天劫 + 800 万欠款
  - 张羽：双仪式倒计时

### P1 关键CG（第 22 章电话）
- **李雪莲接电话场景**：
  - 电话铃声响起
  - 李雪莲接听 → 听到父亲决定
  - 表情变化：从期待 → 失望 → 理解
- **电话内容**："我已经找到了真正的天才"

### 视觉差异化
- **画作 vs 真人的视觉转换**：
  - 画作中：模糊 → 清晰 → 隔空一拳
  - 真人未登场，但通过画作 + 电话 + 背景信息"构建"形象
- **缺席的反派设计**：星火真人作为"幕后大佬"，通过李雪莲 / 画作 / 电话间接呈现

## I. 人物伏笔

- **金丹真人的"被优化"机制**：
  - 350 岁"被优化" → 回收一半修为
  - 反映昆墟十大宗门对修士的工具化
  - 张羽吐槽："金丹大能回新手村养老？"
- **天人演武图 = 星火真人的遗愿**：
  - 寻找弟子 → 延续家族
  - 寻找"思想灵活的天才" → 避免被心法束缚
- **白真真 = 星火真人最终弟子**：
  - 第 22 章电话确认
  - 为白真真后续离开嵩阳篇（去白龙高中 / 二层大学）埋伏笔
- **张羽被婉拒**：
  - 阶级与天赋双鸿沟的体现
  - 1-25 章嵩阳篇核心冲突之一
- **星火真人的"金丹世家"基因**：
  - 李雪莲的银发 + 火光礼服
  - 推测为"火"属性道种 / 血脉

## J. CG 痛点

- **"缺席"角色的形象构建**：
  - 不能让观众觉得"没有存在感"
  - 通过画作 + 电话 + 背景信息构建"幕后大佬"形象
- **画中"中年男子"的视觉设计**：
  - 既要符合"金丹大能"威严
  - 又要体现"画作"的艺术性
  - 演练武学的动态效果
- **不同参悟者看到的差异**：
  - 钱深 / 炼天极 / 白真真 / 张羽看到的不同幻境
  - 体现"道心"主题的多元化
