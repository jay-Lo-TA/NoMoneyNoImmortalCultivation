# 人物-77-郑在明（嵩阳三中校董 / 招揽者）

> **P1**（高考前夕挖角张羽/白真真的校董代表）

## A. 基础档案
- **姓名/称谓**：郑在明
- **首次出现**：第 121-122 章（高考前 2 个月，竞赛结束后密集挖角期）
- **身份**：嵩阳三中（民办/二三梯队高中）校董
- **阵营**：嵩阳市二三梯队高中 + 16 豪族围剿穷学生阵营外围
- **修为**：未明（校董通常不直接战斗）

## B. 性格与行为模式
- **精明商人**：把"招揽"做成标准化产品（考试大礼）
- **包装诱惑**：用奖学金、免试名额把"分流"包装成"恩惠"
- **校董派头**：西装革履 + 助理随行 + 合同文件堆叠
- **目的单一**：把张羽/白真真的竞赛成绩"收编"到三中考试名单

## C. 关键事件
- **第 121 章**：
  - 嵩阳高中刚结束白龙武道竞赛
  - 玉星寒、宋海龙等金丹弟子私下接触张羽谈陪练
  - 各路势力密集接触张羽/白真真
- **第 122 章**：
  - 郑在明代表嵩阳三中正式向张羽抛橄榄枝
  - 提出考试大礼：考试名额 + 免试资格 + 高额奖学金 + 功法补贴
  - 张翩翩揭穿真相："他们要把你们分流到二三梯队高中的考试名额里消化掉"

## D. 关键台词
- "我们三中愿意给张羽同学/白真真同学……"（标准招揽开场）
- "免试名额 + 全额奖学金 + 功法补贴全包"（礼包展示）
- （内心）："这两个穷鬼的竞赛成绩太炸裂了，不能让他们直接考十大"

## E. 重要关系
- **张羽**：招揽对象（被拒绝）
- **白真真**：招揽对象（被拒绝）
- **张翩翩**：识破其目的并警告张羽
- **16 豪族**：背后指使者
- **嵩阳三中**：所属学校

## F. CG 制作要点
- **造型**：商务西装 + 名牌手表 + 助理 2-3 名
- **场景**：`[[40-场景/通用/场景SC-121-XX-嵩阳三中校董办公室]]`（待建）
- **动捕**：推合同文件的特写 + 微笑的精明表情
- **关键道具**：印有三中 logo 的合同 + 巨额奖学金支票

## G. 人物伏笔 + CG 痛点
- **伏笔 1**：郑在明背后是否还有其他二三梯队高中联合挖角？
- **伏笔 2**：被拒绝后是否会对张羽/白真真采取"围剿"行动？
- **CG 痛点**：校董办公室的"逼仄感"（区别于白龙/仙云的豪华）
- **CG 痛点**：合同文件的"金箔陷阱符文"特效
