# 仙韵集团嵩阳市总监（法骸金人）

> **仙韵集团嵩阳市总监** | **法骸改造人**（全身金箔）| 画展核心宾客 + **筑基期**压迫感代表

## A. 基础档案

- **姓名**：未公开（仅以"仙韵集团嵩阳市总监"称之）
- **身份**：仙韵教育集团嵩阳市总监
- **所属**：仙韵集团（万法宗下属）·嵩阳市分部
- **首次出场**：第 11 章（间接，通过试功教室"老师"汇报）
- **实体登场**：第 16 章（法骸金人形态首登场）
- **实力**：**筑基期**（法力 100+）
- **外貌特征**：
  - **浑身肌肤犹如贴了一层金箔**，反射着淡淡金光
  - 浑身金光闪闪（走动时反射光斑）
  - 法骸改造程度极高（全身覆盖）
- **重要事件**：第 16 章画展宾客入场盛况 + 第 18 章参与天人演武图揭幕 + 第 19 章与李雪莲探讨道心

## B. 性格与行为模式

- **上位者气场**：作为仙韵集团嵩阳市总监，自带成功人士气场
- **平易近人**：与李雪莲平等交流（"微笑着说"），与其他富豪名流融洽
- **见识广博**：第 19 章与李雪莲探讨天人演武图，对"道心"概念有深入理解
- **聚焦中心**：三人之一（仙韵总监 / 宝石眼男子 / 光环男子），成为全场焦点
- **配角定位**：
  - 衬托"筑基期"修仙者与"炼气期"修仙者的阶级鸿沟
  - 展示"法骸改造人"的高端仙道技术
  - 暗示仙韵集团对低层学生的工具化态度

## C. 关键事件

### 第 11 章（间接出场）
- 仙韵集团试功教室的"老师"（研发负责人）是嵩阳市总监下属
- "老师"冷漠态度："就算死上几百人又如何了？"（仙韵集团对低层学生的工具化）

### 第 16 章（法骸金人首登场 · **P1 关键CG**）
- 画展正式开始，宾客入场
- **仙韵集团嵩阳市总监**：浑身上下肌肤如贴金箔，反射金光
- 张羽赵天行震惊："这是什么东西？机器人？"
- 赵天行解释：**法骸**（仙道义肢升级版）：
  - "有些像是义肢，却又比义肢什么的强太多"
  - "用法宝替换掉人体的任何部位，并在保留原有功能的前提下，大幅度提升各项能力"
  - "**只有法力达到 100 以上，才能在体内植入法骸**"
- 张羽："筑基老祖大学毕业后跑来一层炸鱼"
- 三人成为全场焦点（仙韵总监 / 宝石眼男子 / 光环男子），吸引炼气仔簇拥

### 第 17 章（分数和秘辛）
- 与其他宾客交流天人演武图情报
- 与李雪莲讨论星火真人背景

### 第 18 章（天人演武图揭幕 · P0 必做核心CG）
- 李雪莲实体登场时，**仙韵集团嵩阳市总监主动迎向李雪莲**
- 周围富豪名流"恨不得能和四人聊上几句，混个脸熟"
- 出席天人演武图揭幕仪式

### 第 19 章（惊怖和道心 · P0 必做）
- 与李雪莲探讨天人演武图的道心考验
- **李雪莲的解答**：天人演武图的第一关考的是道心，针对心中惊怖
- 仙韵集团嵩阳市总监："我明白了，星火前辈是想要一个思想灵活的弟子？"
- 与李雪莲互动："思想的变化，离不开时代发展之影响"
- 推测"人人都离不开借贷，想必也会体现在天人演武图的试炼之中"

### 第 20 章（参悟 · P0 必做）
- 观察张羽的特殊行为（吐纳 + 参悟 + 健体 + 刷手机 + 吃大虾）
- "这小子一会吐纳，一会又站起来参悟武学，这是看到了什么东西？"

## D. 关键台词

> "**我明白了，星火前辈是想要一个思想灵活的弟子？一个练习拜神心法，能换着神拜，练习天勤心法，既能在苦修上卷，也能在躺平上卷的徒弟。**"

> "**思想的变化，离不开时代发展之影响。像是如今社会，人人都离不开借贷，想必也会体现在天人演武图的试炼之中吧？**"

> （与李雪莲交流时平等自信的神态）

## E. 重要关系

- **李雪莲**：画展合作方 + 平级交流
- **张羽**：作为保安远距离观察（"机器人？"的反应）
- **赵天行**：张羽的同伴，向张羽解释"法骸"概念
- **炼气期宾客**：争相簇拥三人组（仙韵总监 / 宝石眼 / 光环）

## F. 法骸改造的视觉设定

### 法骸概念
- **法骸**：仙道义肢升级版
- **范围**：可替换人体的任何部位
- **效果**：保留原有功能 + 大幅度提升各项能力
- **门槛**：**法力达到 100 以上**才能植入
- **意义**：筑基期修士的"标配"装备

### 仙韵集团嵩阳市总监的法骸
- **覆盖范围**：全身（"浑身上下肌肤如贴金箔"）
- **视觉特征**：金光闪闪（走动时反射光斑）
- **设计风格**：金箔质感 + 流动光泽
- **动静表现**：
  - 静态：金色肌肤 + 淡淡金光反射
  - 走动：动态光斑扫过周围宾客
  - 谈话：金光在面部流动

### 其他法骸改造人
- **宝石眼男子**：双眼如镶嵌宝石，反射光学迷幻效果
- **光环男子**：脑后浮现一轮光环
- **三人组**：画展焦点，炼气仔争相簇拥

## G. CG 制作要点

### P1 关键CG（第 16 章法骸金人首登场）
- **金人走入画面的低机位仰拍**
- **金光反射的动态光斑**：扫过周围宾客
- **张羽/赵天行"震撼发现"反应**
- **三人组簇拥画面**：其他富豪围拢
- **张羽内心独白**："筑基老祖大学毕业后跑来一层炸鱼"

### P0 关键CG（第 18 章揭幕仪式）
- **仙韵集团嵩阳市总监主动迎向李雪莲**
- 李雪莲的红色火光晚礼服 vs 金光法骸（双重"金丹级"视觉符号）
- 周围富豪名流簇拥的群像

### P0 关键CG（第 19 章探讨道心）
- **仙韵集团嵩阳市总监 + 李雪莲 + 天人演武图前**
- 三人组讨论道心考验机制
- 金光法骸 + 银发清冷 + 天人演武图（三种视觉元素并置）

### 视觉差异化
- **金光流动**：金箔肌肤的动态光泽
- **光斑扫描**：走动时反射周围环境
- **奢华质感**：与张羽"薄肌拉丝"形成极端对比
- **阶级符号**：筑基期 + 法骸 = 嵩阳顶层

## H. 与张羽的对比

### 阶级鸿沟（第 16 章首登场时）
- **张羽**：炼气期 + 法力 8.3 + 保安服 + 400 时薪
- **仙韵嵩阳市总监**：筑基期 + 法力 100+ + 金光法骸 + 高层地位
- **视觉冲击**：张羽"薄肌拉丝的身材，俊秀清冷的脸庞，异世来客的气质" → **输给赵天行的筋肉怪物** → **被法骸金人完全碾压**

### 法骸作为"权力符号"
- 张羽："只有法力达到 100 以上，才能在体内植入法骸"
- **100 法力 = 炼气境界上限**
- 筑基期才能稳定植入法骸
- **视觉意义**：法骸 = 阶级跨越的标志

## I. 人物伏笔

- **筑基期压迫感**：本作"阶级固化"主题的视觉符号
- **仙韵集团工具化**：第 11 章试功教室"老师"的态度反映集团文化
- **法骸改造技术**：本作"修真工业化"的世界观核心
- **三人组焦点**：画展上的阶级展示

## J. CG 痛点

- **法骸金光反射的实时渲染**：走动时的动态光斑需要粒子系统
- **金箔肌肤的次表面散射**：金属 + 皮肤的双重质感
- **三人组的视觉协调**：三种不同的法骸造型（金光 / 宝石眼 / 光环）需统一风格
- **与李雪莲的视觉对位**：红色火光礼服 vs 金光法骸（两个金丹级视觉符号）
