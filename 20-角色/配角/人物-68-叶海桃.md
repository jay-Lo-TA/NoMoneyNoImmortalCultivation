# 人物-68-叶海桃（嵩阳高中竞赛队·道心不稳）

> **嵩阳高中竞赛队"竞赛落选+道心崩溃"代表** | 第 101 章 P2 心理戏 CG

## A. 基础档案

- **姓名**：叶海桃
- **身份**：嵩阳高中高一·体育竞赛队队员（**最晚入队**）
- **所属**：嵩阳高中·体育竞赛队（高一）
- **首次出场**：第 101 章
- **肉体强度**：**2.94 级**（第 101 章·嵩阳高中第 5 名·张羽白真真加入后掉到第 7）
- **家境**：参加特殊培养计划 + 投入巨款 + **背上债务**
- **原文（首次出现第 101 章）**：原本第 5 名的她**张羽白真真加入后直接掉到第 7 名·完全失去竞赛机会**——"想到自己家为了参加竞赛队、参加特殊培养计划而投入的巨款，背上的债务……此刻却失去了参加竞赛的名额，叶海桃只感觉天都要塌了。"

## B. 性格与行为模式

- **绝望型**：家境不好+投入巨款+竞赛落选 = 心理崩溃
- **道心不稳**：第 101 章"**手脚更是忍不住地抽搐了起来，像是要失去了控制**"
- **王海急救**：王海第一时间喂**道心缓冲剂**急救
- **蓝岭点评**："叶海桃，你是最晚入队的，要抓紧了，这个数值很难拿名次"

## C. 外形与服装设定

- 高一女生·嵩阳高中校服
- **崩溃瞬间**：脚一软坐倒在地 + 手脚抽搐

## D. 关键场景动作捕捉需求

- **道心崩溃动作**：手脚抽搐+坐倒+脸色苍白
- **王海喂道心缓冲剂**：急救动作

## E. 人物弧光（成长/转变/关系）

- **起点（第 101 章）**：竞赛队最晚入队·投入巨款·期望高
- **崩溃（第 101 章）**：张羽白真真加入后掉队到第 7·竞赛资格消失
- **结局**：道心不稳·王海喂缓冲剂急救——**成为嵩阳高中竞赛落选的悲剧代表**

## F. 关键台词

- **金句**："叶海桃，你是最晚入队的，要抓紧了，这个数值很难拿名次"（蓝岭）
- **金句**："她道心不稳了，去把她的道心缓冲剂拿来灌进她嘴里"（王海急救）

## G. CG 痛点

- **道心不稳具象化**：手脚抽搐 + 坐倒慢镜头
- **贫富差距金融具象化**：投入巨款 + 背上债务 + 竞赛落选的三重打击
