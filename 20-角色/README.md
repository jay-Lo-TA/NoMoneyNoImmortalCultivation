## 🎯 收录标准

凡是符合以下任一标准的角色，归入 `20-角色/`：

1. **剧情角色**（85 个）：有名字 + 多次出场 + 推进剧情
2. **群众角色**（~95-110 场景类型）：未推进剧情 + 画面需要 + 原文有背景描述
3. **未来预留角色**：第 2 季 ch203+ 出现的角色（如师承/萧玉容/高崇光）

不属于本目录的内容：
- ❌ 通用 CG 资产 → 归 `25-道具/`
- ❌ 场景地点 → 归 `40-场景/`
- ❌ 抽象招数/功法 → 归 `10-世界观/招数/`
- ❌ 美术风格 → 归 `50-美术风格/`
