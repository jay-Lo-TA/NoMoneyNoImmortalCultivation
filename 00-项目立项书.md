# 《没钱修什么仙？》番剧制作项目立项书

> **制片人视角 P0 文档** | 项目入口

## 🎬 项目愿景

将熊狼狗所著《没钱修什么仙？》改编为 26 集 × 15 分钟/集的**TV 半年番**，目标受众为 18-35 岁年轻人 + 潮流文化爱好者，打造"修真 + 金融"的差异化视觉风格。

## 📊 项目核心数据

| 项目 | 数据 |

| 原作 | 熊狼狗《没钱修什么仙？》1-200 章（原作未完结） |

| 目标受众 | 18-35 岁年轻人 + 潮流文化爱好者 |
| 制作周期 | 预计 18-24 个月 |
| 项目状态 | **前期知识库搭建阶段**（2026-06） |

## 🌟 视觉差异化卖点（5 大支柱）

1. **修真 + 金融**：法力贷 / 灵币 / 修真金融化（核心 P0 哲学）
2. **修真 + 游戏化**：羽书系统 / "+" 号加点 / 属性面板
3. **绝育 / 变性 / 抵押器官**的黑色幽默（开篇面试荒诞）
4. **金丹弟子 / 仿妖筋肉**的商业化讽刺
5. **第 1 季季终钩子**：第 50 章无极云手 + 黑影（完美悬念）
