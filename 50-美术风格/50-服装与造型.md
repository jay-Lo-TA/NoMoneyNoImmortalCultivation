# 👔 服装与造型

## 张羽（主角）

- **日常校服**：万法大学制服（现代 + 仙道元素）
- **战斗装**：筑基期后可解锁法骸装备
- **风格**：朴素的努力型男主，**避免太华丽**
- **重要级**：P0

## 白真真（女主）

- **风格**：进化型灵根 + 轻盈飘逸
- **校服**：天剑大学（外校）
- **战斗**：剑修风格
- **重要级**：P0

## 主要角色服装

| 角色 | 服装特点 | 重要级 |
|---|---|---|
| 张羽 | 朴素、努力型 | P0 |
| 白真真 | 飘逸、剑修 | P0 |
| 高崇光 | 威严、化神级法骸 | P0 |
| 萧玉容 | 炼器系主任，气场强 | P0 |
| 链枢神君 | 金融系 + 邪教风格 | P0 |
| 道乾坤 | 仙门级造型 | P1 |
| 夜星璃 | 炼器系弟子 | P1 |
| 素云霓 | 班花造型 | P2 |

## 设计原则

- 修真元素要"年轻化"（年轻人向）
- 符咒 / 法器作为装饰而非整体风格
- 颜色与角色弧光呼应（参考 50-色彩与情绪）
