# 🎨 视觉风格猜想

## 整体调性

修真 + 现代金融 —— **冷峻几何 + 东方神秘**

## 视觉关键词

- **冷峻**：金属灰、深蓝、金融感
- **几何**：直线、棱角、极简
- **东方神秘**：符咒、太极、阴阳
- **高科技感**：HUD、UI、光效

## 配色方案

| 场景 | 主色 | 辅色 | 情绪 |
|---|---|---|---|
| 昆墟主楼 | 深蓝 + 银白 | 金 | 冷峻、秩序 |
| 地下层 | 血红 + 黑 | 紫 | 危险、压抑 |
| 战场 | 红 + 金 | 白 | 热血、激烈 |
| 日常生活 | 暖灰 + 米白 | 蓝 | 真实、压抑 |
| 修仙场景 | 金 + 紫 | 白 | 神圣、超凡 |

## 参考作品

- 《赛博朋克：边缘行者》（金融 + 城市感）
- 《灵笼》（东方 + 科幻）
- 《凡人修仙传》（修真基础）
- 《天官赐福》（角色唯美度）
- 《BLAME!》（建筑几何感）

## CG 关键挑战

1. **金融 UI 与修真符咒的融合**：羽书、法力贷 APP
2. **昆墟建筑可视化**：54 层超级建筑
3. **法术特效分层**：低阶（小光效）→ 高阶（领域级）
4. **角色唯美度 vs 修真写实**：年轻人向，造型要潮
