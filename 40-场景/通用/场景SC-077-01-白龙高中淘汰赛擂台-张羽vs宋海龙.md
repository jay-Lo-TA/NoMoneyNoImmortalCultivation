# 场景SC-077-01-白龙高中淘汰赛擂台-张羽vs宋海龙（背龙翻岳+无极云手5级突破）

> **场景元信息**
> - 重要级：**P0 必做（半决赛 + 1000万血条 + 武学突破）**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·半决赛擂台
> - 首次出现：第 77 章
> - 复用章节：第 77、78、79 章（**全作最终决战的开端**）

## 🎬 场景描述

**P0 必做半决赛核心 CG**。白龙高中武道竞赛**半决赛**第 2 场 — **张羽 vs 白龙高中宋海龙**。**本作武道竞赛篇最热血决战的开端 + "1000 万血条"的视觉化金融讽刺 + 张羽无极云手 5 级突破的"战斗极限 CG"**。

**白龙高中宋海龙**（**本作最强高一学生 + 2 米 5 泰坦巨汉 + 1000 万保险**）：
- **白龙高中年级第一** + **校服胸口和背后显示大大的"1"**
- 评分赛 300 分（**1 秒甩飞黄巾力士**）
- 装备：**象筋体**（**植入象妖筋肉** + **天灵蕴体 + 不败斗志 + 家族儿子王**）
- **象筋体 = 长期和妖兽战斗 + 负面情绪极少 + 白龙高材生 + 宋家这一届的王牌儿子**
- **蕴体灵根 = 战斗用灵根**（灵根形成的灵机经络会缠绕全身上下，全方位增加使用者的力量、速度、体能还有抗打击能力）
- **"豪华版"蕴体灵根**（**除增强肉体强度和抗打击能力外，还能不断吸收空气中的灵机，让身上的伤势渐渐自愈**）
- **1000 万医疗保险上限**（**头显数字**）— **本作"金钱即血条"的视觉化金融讽刺**
- 修炼功法：**专家级武学"背龙翻岳手" 1 级发动** — 纯粹以肉体暴力来推动擒拿、摔跤的专家级武学
- 象筋体让法力全部用来加持肉身 → 背龙翻岳手 1 级 + 强横体魄 = 足以碾压同级强者
- **白龙奴隶主** — 观众席上的白龙人对宋海龙**令行禁止**

**核心剧情**（**4 段递进**）：
- **第 77 章"1000 万血条 + 等级碾压"**：
  - 紫云人坐炼天极当板凳（**炼天极暗中为张羽加油**："张羽要是赢了宋海龙的话，我的广告合作说不定都能重新要回来……"）
  - 楚秋河偷偷诅咒宋海龙（"只有你输了，大家才都有面子"）
  - 宋海龙张开臂膀高举双手 → **白龙方阵山呼"宋海龙！第一！"**（**整场最整齐的群像表演**）
  - 张羽在擂台上显得孤立无援
  - 宋海龙刺猬头：头上出现一组数字 **1000 万**（**外置灵根自带的医保费用上限显示**）
  - 宋海龙："这是我的外置灵根自带的功能，能够显示我的医保费用上限。**我现在的医保上限有 1000 万，你能把这 1000 万治疗预算全部打光，我便认输了。**"（**本作最傲慢的"自亮血条"CG**）
  - 张羽思考：对方保险够治 1000 万，自己这边只够治 10 万 → **"哦不，现在只剩下 7 万的伤"**（**金融讽刺核心**）
  - 张羽运起天武炼心诀 → 压下负面情绪 → 提起战意
  - 宋海龙："你先手吧，我动手的话，你可能就没机会了。"（**本作最傲慢的"让先手"CG**）
  - 张羽一掌带无相云罡拍在宋海龙胸口 → **"打在了橡胶轮胎上"** → 额头 1000 万只减少 300
  - 又几掌 → 1000 万只减去 2000 多 → **且数字有了恢复的迹象**（豪华版蕴体灵根自愈）
  - 宋海龙："是不是又涨回去了？**抱歉啊抱歉，我这次用的战斗灵根不是普通的蕴体灵根，而是豪华版蕴体灵根。**"
  - "**这灵根除了能够增强我的肉体强度和抗打击能力之外，还能够不断吸收空气中的灵机，让我身上的伤势渐渐自愈。**"
  - 宋海龙："你不过来，那我就来了。"
  - 宋海龙大手一张 → 如龙爪般抓来
  - 张羽思考："数据上差太多了，单纯靠我自己的力量很难击败他。**唯一的办法就是借用他的力量击败他，防守反击才是我的战法。**"
  - 全集中状态下 → **背龙翻岳手 1 级** → 擒摔武学
  - 张羽像一条小小的游鱼 → 在宋海龙铺天盖地的网罗下一次次艰难擦边逃过
  - "**抓到你了。**" → 宋海龙猛地捏紧了张羽的胳膊
  - **张羽一声怒吼 → 原本只是覆盖在双掌部位的无相云罡突然一阵暴涨 → 蔓延到了手肘部位 → 如泥鳅一样从宋海龙的手掌中划了出去**
  - 炼天极吃惊："**在绝境之下，他的无极云手又突破了？5 级了？这个张羽还真是武学奇才。**"（**本作张羽"战斗中突破"的关键 CG**）

**关键道具**：
- **1000 万数字浮空**（**本作首次"金钱即血条"CG**）
- **象筋体的"金色筋肉" + "灵机经络缠绕"视觉化**（**本作首次"外置灵根"的完整视觉化**）
- **背龙翻岳手 1 级的"擒拿网罗"特效**
- **无相云罡从双掌蔓延到手肘**（**无极云手 4→5 级突破 CG**）

**氛围**：金色（宋海龙的灵根 + 保险数字） + 白色（张羽的无相云罡） + 暗红（伤口飞溅） — **金白对决 + 金融讽刺**。

## 📐 CG 制作要点

- **空间设计**：
  - 半决赛擂台（**整场比赛只有 1 个主擂台** — 第 76-77 章连续两场）
  - 宋海龙 2 米 5 泰坦巨汉 + 刺猬头
  - 张羽相对瘦小的身形（**视觉对比**）
  - 观众席三大方阵
- **关键帧**（**P0 必做 / 4 段递进**）：
  - **白龙方阵"宋海龙！第一！"山呼**（**整场最整齐的群像表演**）
  - **宋海龙"1000 万"血条自亮**（**本作最傲慢的"自亮血条"CG**）
  - **宋海龙"你先手吧，我动手的话，你可能就没机会了"**（**本作最傲慢的"让先手"CG**）
  - **张羽"打在了橡胶轮胎上"** + **1000 万只减少 300** + **数字又涨回**（**金融讽刺"血条不能破"CG**）
  - **宋海龙"豪华版蕴体灵根"自愈** — **灵机经络缠绕全身**（**本作首次"外置灵根"完整视觉化**）
  - **背龙翻岳手 1 级擒拿网罗** — 张羽"如游鱼"擦边逃过
  - **"抓到你了"** → 张羽**一声怒吼** → 无相云罡**从双掌蔓延到手肘** → 5 级突破
  - 炼天极吃惊："**在绝境之下，他的无极云手又突破了？5 级了？这个张羽还真是武学奇才。**"（**张羽"战斗中突破"的关键 CG**）
- **特效需求**：
  - **1000 万金色数字浮空特效**（**本作首次"金钱即血条"**）
  - **象筋体的"金色筋肉" + "灵机经络缠绕"**（**本作首次"外置灵根"完整视觉化**）
  - **背龙翻岳手的"擒拿网罗"特效**
  - **无相云罡从双掌蔓延到手肘**（**无极云手 4→5 级突破**）
  - 自愈的灵机流光（**豪华版蕴体灵根**）
- **动捕需求**：
  - 宋海龙的"龙爪"动作（**需要武打替身**）
  - 张羽的"防守反击"动作
  - 张羽"一声怒吼" + "无相云罡暴涨"动作
  - 白龙方阵"山呼"群体表演
- **音频设计**：
  - "宋海龙！第一！"的整齐山呼
  - 1000 万数字弹出时的"叮"声
  - 自愈时的灵机流光音
  - 张羽"一声怒吼"
  - 炼天极"在绝境之下"的吃惊评价
- **建议时长**：5-7 分钟（**半决赛开端 + 武学突破 CG**）
