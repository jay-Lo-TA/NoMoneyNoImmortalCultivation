# 场景SC-081-02-嵩阳市区饭店-庆功宴+预制菜+功法选择

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市市区·某预制菜饭店
> - 首次出现：第 81 章
> - 复用章节：第 81、82 章

## 🎬 场景描述

嵩阳市区的一家小饭店，主打"正宗预制菜"——没有海鲜、没有额外调味、吃的是预制菜的本味。空间不大、装修朴素、桌上有一次性桌布、塑料椅子、墙上贴着"本店不议价"的标语。**注意**：这家店是嵩阳高中的"穷人饭局"专用户，**与场景SC-014-03 城市炸鸡店（白真真的"自律日常"）是不同店家**。

**时间**：第 81-82 章衔接处，庆功宴。

**关键视觉元素**：
- **张羽进门第一句："有帝王蟹吗？来十只。"**（赤贫+小农意识反差喜剧）
- **钱深的"健康饮食"科普**："没海鲜+营养丰富+每盘一万大卡以上+妖兽肉食补"
- **雷钧的"穷老师" 6000 块预算**
- **兽蛋白奶"干杯"**（健康又营养的高中生饮品）
- **菜单上的炼体功法选择 UI 浮空**（功法列表 + 3 选 1）

**核心剧情**：
- 雷钧讲几句"教学成果" → 自我表扬 → 张羽白真真"吹牛老师"
- 张翩翩短信："弟弟恭喜第一 + 4万记得快点还"
- 钱深问"乐沐岚说的 40 万什么意思" → 张羽说 → 雷钧痛心疾首"糊涂！"
- 张羽的手机震动 → 武道竞赛通知：5万奖金到账 + 选专家级功法
- 张羽在 3 门炼体功法（龙象大力/赤髓浑元气/移筋换脉）中 → 雷钧建议龙象大力 → 张羽选**赤髓浑元气**（**一招 0→高难度**，符合其"越难越快"天赋）
- 钱深请客 + 何大有"不吃穷人餐" + 赵天行"水煮鸡肉"

**氛围**：穷人庆功 + 小农意识 + 健康饮食 + 功法选择的"知识传授"，是**主角胜利后难得的非战斗戏**，奠定第 82 章开始"赛后修整"基调。

## 📐 CG 制作要点

- **空间设计**：
  - 朴素小饭店：一次性桌布 + 塑料椅 + "本店不议价"标语
  - 墙上的菜单（妖兽肉 + 营养价值）
  - 角落里的电视（播报嵩阳新闻/武道竞赛结果）
- **关键帧（**P1 必做 CG**）**：
  - **张羽"帝王蟹 10 只"**（赤贫口吻）→ 雷钧脸一黑
  - **钱深"健康饮食"科普**（表情认真、何大有不屑）
  - **兽蛋白奶"干杯"**（五人举杯 + 钱深请客 + 雷钧的 6000 块预算）
  - **张羽手机浮空 UI 浮现**："武道竞赛 5 万到账 + 选专家级功法"（**3 门炼体功法列表**）
  - **张羽选赤髓浑元气**（雷钧的"建议 + 张羽的"天赋判断"对比蒙太奇）
  - **"这一招"UI 浮现**：赤髓浑元气一招"站功"全身上下齐齐配合
- **特效需求**：
  - 菜单上"妖兽肉"标签
  - 浮空功法列表（镀金数字 + 功法图）
  - 兽蛋白奶的高光（健康饮品）
- **动捕需求**：
  - 张羽"帝王蟹"的"小农意识"表演
  - 雷钧讲几句"教学成果"的"吹牛"表情
  - 钱深"健康饮食"的"营养学家"姿态
  - 何大有"我不吃穷人餐"的高傲
- **音频设计**：
  - 饭店嘈杂环境
  - 雷钧讲几句话的"领导式"语气
  - 手机"叮"（5 万到账 + 选功法）
- **建议时长**：3-5 分钟（贯穿 81-82 章的庆功 + 功法选择）
