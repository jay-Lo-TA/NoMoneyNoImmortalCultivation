# 场景SC-060-01-嵩阳高中校外茶室包房（张翩翩密谈）

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市·嵩阳高中校外·私密茶室包房
> - 首次出现：第 60 章
> - 复用章节：第 60 章

## 🎬 场景描述

嵩阳高中校外的私密茶室包房，是张翩翩为张羽、白真真提供庇护性密谈的场所。室内陈设走极简高端路线——原木长桌、独立茶具、淡雅插花、竹帘半垂，色调偏暖米黄，落地窗引入一束午后阳光，整体氛围"安全、可放心说话"。张翩翩扫视四周后亲口确认"这里暂时能够放心说话"，说明这个空间此前已由她确认过无监控、无窃听，是嵩阳市金融修仙世界中的"信息隔离区"。

**关键道具与场景元素**：
- **假存款余额符咒**：张翩翩指尖划动，施展改过的符咒，让头顶银行余额显示 25 亿（实际只有 25 万）—— 这是 P1 重要 CG 特效，关键叙事道具
- **箓书（张羽手中）**：价值十万，张羽和白真真讨论密谈中一直把玩
- **白真真跪地拜义母戏**：白真真戏剧性跪地、甜笑拜"义母"，是黑色幽默桥段
- **手机短信 / 张翩翩电话**（末尾）：与电话另一头的女性友人"翩宝"通话借钱

**核心剧情**：
1. 张羽激动认姐 + 白真真跪地拜义母
2. 张翩翩揭秘：25 亿是假存款符咒改的，实际 25 万（**金融讽刺重要 CG**）
3. 张翩翩传授"以钱制钱"理念："符咒是有钱人最容易被钱驱使，也最容易被钱震慑的根基"
4. 张翩翩交代：高考后她去万法大学，半年后失去庇护，张羽/白真真需自己强大
5. 张翩翩给出目标：成为嵩阳最强高一生（道心/肉体强度/武道/道术全竞赛头两名）
6. 张翩翩让白真真签巡查队实习生合同
7. 末尾张翩翩向朋友借天灵根钱

**氛围**：私密、温暖、精致、黑色幽默、师徒情深（姐弟/义母情）、金融讽刺。

## 📐 CG 制作要点

- **空间设计**：
  - 茶室包房：原木长桌 + 茶具 + 竹帘 + 插花 + 落地窗 + 半垂帘
  - 暖米黄主色调，午后的阳光斜入
  - 与"嵩阳高中"精英环境形成延续感，但又多了一分"长姐庇护"的家庭感
- **关键帧（按时间线）**：
  1. **开篇震撼**：白真真突然跪地拜义母（黑色幽默镜头）
  2. **假存款揭秘**：张翩翩指尖划动 → 头顶浮现金色虚拟数字"2,500,000,000"（25 亿）→ 张羽/白真真震惊脸 → 张翩翩切换符咒 → 数字骤变为"250,000"（25 万）—— **P1 必做 CG：金额数字缩放 + 碎裂特效**
  3. **教学场景**：张翩翩阐述"以钱制钱"理念 + 头部浮现字幕特效
  4. **合同场景**：白真真签巡查队实习生合同
  5. **张翩翩特写**：她偷偷看张羽头顶"？？？未知潜能"评价（与天眼符 UI 联动）
  6. **末尾蒙太奇**：张翩翩与朋友通电话借钱，背景是远望张羽/白真真离去的背影
- **符咒特效**：
  - 假存款符咒：金色光粒汇聚 → 数字浮现 → 数字碎裂为"假"字 → 重组成真值
  - 头顶评价 UI：与场景SC-044-02 天眼符 UI 视觉语言一致（张翩翩也使用万民部符咒）
- **动捕需求**：
  - 张翩翩指尖滑动的优雅 + 权威
  - 张羽/白真真从激动 → 震惊 → 失落的情绪曲线
  - 白真真跪地/撒娇/甜美表情
- **音频设计**：
  - 茶室环境音（淡淡的茶香/水声）
  - 符咒发动"嗡"声
  - 数字碎裂"叮"声
  - 末尾电话铃声
- **建议时长**：5-8 分钟（核心叙事场景，不能压缩）
