# 场景SC-088-01-嵩阳市摄影棚-玉星寒首登场+群拍广告

> **场景元信息**
> - 重要级：**P0 必做核心场景**
> - 物理位置：嵩阳市·某商业摄影棚（仙韵集团广告拍摄）
> - 首次出现：第 88 章
> - 复用章节：第 88 章

## 🎬 场景描述

嵩阳市某商业摄影棚——仙韵集团"补习机构"广告拍摄场地。**注意**：本场景是**商业摄影棚**，**与场景04-中心大厦自助餐区、场景02-中心大厦999层画展主厅（仙韵集团相关的中心大厦空间）是不同的物理位置**——本场景是独立商业摄影棚。

**建筑特征**：
- 巨大白色幕布 + 桁架 + 顶光
- 摄影机轨道 + 监视器墙
- 化妆区 + 候场区
- 工作人员穿梭（导演、化妆师、助理）
- 仙韵集团 logo 墙

**时间**：第 88 章初，武道竞赛结束几天后的一次群拍广告。

**关键视觉元素**：
- **玉星寒首登场**（"被十多名工作人员包围的少年"）：
  - 星火真人的关门弟子
  - "优化下来的金丹"+ "未来嵩阳市的高考状元"
  - 被助理、化妆师、导演环绕 → "众星捧月"构图
  - 仙韵集团拉来拍广告
- **宋海龙"应激的狸花猫"反应**：
  - 弓着背、炸着毛，死死盯着玉星寒
  - 暗示"未来要跟宋海龙争夺年级第一"
- **拍摄中心 vs 边缘的群像构图**：
  - 中心：玉星寒
  - 花旁绿叶：武道前三的张羽、乐沐岚、宋海龙
  - 绿叶下的根茎：第四名白真真
  - 花下的泥：炼天极等排名更靠后的人
  - 导演："金丹真人关门弟子"几个大字填满整个屏幕
- **张羽白真真的"群拍 vs 个人广告"反差**：
  - 张羽："为什么宋海龙都接到个人广告了，我们只接到群拍？"
  - 白真真："因为开学已经半年了，还有半年时间我们就到高二了。**害怕我们塌房**……塌房是成绩降低啊。"
  - 群拍："如果塌房了，把我们剪掉不就好了？"
- **乐沐岚的"反复深呼吸 + 鸡皮疙瘩"**：
  - 拍摄期间在张羽身旁 → 乐沐岚深呼吸 + 微微颤抖
  - "我最近天天看他朋友圈发的照片，已经没有之前感觉那么恶心了，正好趁着这个机会更进一步"
  - 但想到擂台上"坐在他身上又脏又热又腥的血水" → 泛起鸡皮疙瘩
  - "算了，还是先从最简单的开始"
- **张羽 vs 乐沐岚"奇怪"的近身戏**：
  - 张羽"你干嘛呢？" → 乐沐岚退后一步 → "我没事，不用你管"
  - 乐沐岚捂着鼻子："他身上的味道好难闻……这就是妈妈说的，穷人身上的地铁味吗？"
  - "这次果然还是来对了，我已经能慢慢适应了。"

**核心剧情**：
- 张羽白真真接到群拍广告 → 摄影棚
- 玉星寒被十多名工作人员包围 → 白龙高中的"未来年级第一 + 高考状元"
- 炼天极解释玉星寒身份（"星火真人的关门弟子 + 优化下来的金丹 + 转学到白龙高中"）
- 拍摄期间 → 玉星寒被所有人套近乎
- 中心位置被玉星寒占据 → 张羽乐沐岚宋海龙是"花旁绿叶" + 白真真"绿叶下的根茎" + 炼天极"花下的泥"
- 拍摄 4 小时 → 张羽 3 万 + 白真真 8000 → 各自回家

**氛围**：摄影棚商业感 + 金丹弟子的"众星捧月"vs 武竞赛前几名的"花旁绿叶" + 乐沐岚的"脱敏接触" + 阶级差的视觉化。

## 📐 CG 制作要点

- **空间设计**：
  - 商业摄影棚的白色幕布 + 桁架 + 顶光
  - 摄影机轨道 + 监视器墙
  - 化妆区 + 候场区
  - 仙韵集团 logo 墙
- **关键帧（**P0 必做核心 CG**）**：
  - **玉星寒首登场（**"众星捧月"**）**：
    - 玉星寒被十多名工作人员包围的远景
    - 中心位置 vs 周围人潮的反差构图
    - 玉星寒的面部特写（**本作首次登场**）
  - **宋海龙"应激的狸花猫"**：
    - 宋海龙弓着背、炸着毛的特写
    - 死死盯着玉星寒的眼睛
  - **拍摄中心 vs 边缘的群像构图**：
    - 中心：玉星寒（"众星捧月"）
    - 花旁绿叶：张羽、乐沐岚、宋海龙（前三）
    - 绿叶下的根茎：白真真（第四）
    - 花下的泥：炼天极等
    - 导演的"金丹真人关门弟子"几个大字填满屏幕
  - **群拍 vs 个人广告的"塌房"对话**：
    - 张羽白真真的对白（"塌房是成绩降低"）
    - 暗示高二的金融风险
  - **乐沐岚的"反复深呼吸 + 鸡皮疙瘩"**：
    - 乐沐岚站在张羽身旁的特写
    - 深呼吸 + 微微颤抖
    - 捂着鼻子"穷人身上的地铁味"
  - **张羽 vs 乐沐岚"奇怪"的近身戏**：
    - 张羽"你干嘛呢？" → 乐沐岚退后一步
    - 张羽"神经病"内心独白
- **特效需求**（**高优先级 P0**）：
  - 玉星寒周围的"众星捧月"光晕
  - 群像构图的视觉分组（不同色块）
  - 乐沐岚的"脱敏进度" UI（**与天眼符 UI 不同的"洁癖脱敏"风格**）
  - 摄影棚的"专业灯光"
  - 监视器墙的多画面
- **动捕需求**（**高优先级**）：
  - 玉星寒的"金丹弟子"气场（**首次登场**）
  - 宋海龙的"应激"反应
  - 乐沐岚的"深呼吸 + 颤抖"表演
  - 张羽的"莫名其妙"表情
  - 工作人员的"众星捧月"动作
- **音频设计**：
  - 摄影棚的嘈杂 + 工作人员的脚步声
  - "金丹真人关门弟子"的字幕音效
  - 群像的对白蒙太奇
  - 乐沐岚的"深呼吸"
- **建议时长**：3-5 分钟（涵盖玉星寒首登场 + 群像构图 + 乐沐岚脱敏接触）
