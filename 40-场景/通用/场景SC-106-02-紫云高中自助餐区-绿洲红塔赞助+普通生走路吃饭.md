# 场景SC-106-02-紫云高中自助餐区-绿洲红塔赞助+普通生走路吃饭

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市·紫云高中·室内体育场·东侧自助餐区
> - 首次出现：第 106 章
> - 复用章节：第 106 章、第 107 章

## 🎬 场景描述

紫云高中室内体育场的**东侧自助餐区**——**绿洲集团 + 红塔牧业免费提供**的高端自助餐，是张羽白真真在吃了一个月合成食物后的"人间烟火盛宴"。本场景是**"反差萌核心 CG"**——穷鬼嵩阳学生 vs 顶级名校的吃相对比 + 紫云高中"差生在厕所吃饭"魔怔校规的视觉化。

**空间氛围**：
- 自助餐区在体育场馆东侧，空间宽敞
- 各种大品牌赞助商的 logo（绿洲集团、红塔牧业）到处都是
- 紫云学生端着盘子**边走边吃**——倒数十名以外没资格坐下
- 红塔"黑色怪物"（黑龙血脉觉醒）**风卷残云**式进食
- 西装革履的公司人、拿着相机的网红（4 米以上 + 2 米不到）来来往往

**关键视觉元素**：
- **"预制烤鸡腿的眼泪"**——白真真眼含泪花咬下鸡腿（"预制烤鸡腿真是太好吃了！"）
- **张羽狼吞虎咽牛排羊排**——纯自然修行者第一次接触大量兽肉
- **红塔黑龙学生群像**——黑色油亮、黑色龙鳞、五彩斑斓的黑、小角、利爪、尾巴、猩红眼睛、狩猎者威压
- **紫云学生边走边吃**——整齐排队盛饭后立刻边走边吃，到出口处光盘放回餐具
- **乐沐岚登场 CG**——紫云人齐齐行礼 + 紫色竖瞳印记 + "板凳女生趴地"魔怔画面
- **乐沐岚伸手摸张羽 CG**——"我能摸一下你吗？我可以给钱"——**本作洁癖脱敏主题升华**

**关键剧情（按章节）**：
- **第 106 章末尾**：张羽等人到自助餐区 + 红塔黑龙学生群像登场 + 看到炼天极额头紫色圆点
- **第 107 章核心**：白真真"预制烤鸡腿"的眼泪 + 乐沐岚"摸手 2000 块"事件 + 白真真"同时摸两个"商业头脑

## 📐 CG 制作要点

- **P1 反差萌核心**：白真真咬鸡腿的眼泪特写——这是"吃了一个月合成食物"的情感爆发 CG
- **P1 视觉差异化**：红塔黑龙学生群像——黑色龙鳞反射油亮 + 猩红眼睛 + 小角 + 利爪 + 尾巴 + 狩猎者威压（与白龙的"人形妖兽"形成对比——白龙是"人变妖兽"、红塔是"妖变强人"）
- **P1 魔怔画面**：紫云学生齐齐朝乐沐岚行礼的群像 + "板凳女生"趴在乐沐岚身下的"魔怔校规"CG（为后续"紫云药业的副作用"埋伏笔）
- **P1 摸手 CG**：乐沐岚冰凉小手从指尖试探性地触摸张羽手背的特写（结合上一场景 SC-104-02 练功室的洁癖脱敏线索）
- **演出细节**：白真真"预制烤鸡腿真是太好吃了"的台词 + 慢镜头（鸡腿汁水爆开 + 眼泪掉落的双镜头）
