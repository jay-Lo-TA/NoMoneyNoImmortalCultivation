# 场景SC-091-01-嵩阳市远郊-张羽夜奔暗学帮园区

> **场景元信息**
> - 重要级：**P2**
> - 物理位置：嵩阳市·远郊（嵩阳高中→暗学帮园区途中）
> - 首次出现：第 91 章
> - 复用章节：第 91、92、97、98、99 章（通勤蒙太奇背景板）

## 🎬 场景描述

**本场景是"嵩阳市贫富反差蒙太奇"的关键CG，也是张羽"赤髓浑元气变脸" + "张翩翩隔网画符"两个视觉奇观的首次实体化。**

空间：嵩阳市的远郊，约距离嵩阳市市中心近 70 公里。**道路基建从市中心一路崩坏**：高科技中心大厦 → 摩天楼群 → 普通街道 → 烂尾楼区 → 远郊小镇 → 凡人救济摊。**张羽从科幻大都市一路跑到破旧小县城。**

氛围：夜色深沉，霓虹灯影在张羽身后逐渐稀疏。**从张羽的视角看出去，世界由明到暗、由仙到凡**——这是原作宇宙"昆墟一层阶级断层"的视觉化呈现。

关键剧情节点：
- 张翩翩视频电话中**隔网画隐匿符**（**首次"远程画符"视觉化**——张羽头顶落下金色符箓隐入发丝）
- 张羽**赤髓浑元气变脸**——镜中自己从玉树临风变成普通路人，连身高都缩了一段（**肌肉控制的极致展示**）
- 张羽从嵩阳中心大厦附近出发，**一路跑向远郊**（后续 92-99 章通用的"通勤蒙太奇"背景）

光照：
- **出租屋镜子前**：日光灯冷白 + 镜子反射，张羽看到自己"陌生化"的镜像
- **远郊夜路**：月光 + 远处小镇的零星灯火 + 张羽身影在黑暗中快速穿梭

关键道具：
- **张羽手机**（视频通话界面 + 张翩翩手指画符的指尖特写）
- **远郊公路**（双车道柏油路 → 单车道水泥路 → 土路 → 凡人步道）
- **晨光合成食物摊**（远郊凡人救济摊点，长队凡人排队领食物）

## 📐 CG 制作要点

**镜头语言**：
- **隔网画符特写**：张翩翩纤细手指在手机摄像头前画符 → 屏幕中飞出金色符箓 → 落点锁定张羽头顶发丝 → 隐入（**实拍 + CG 合成**）
- **赤髓浑元气变脸逐帧**：张羽在镜子前观察自己的脸 → 脸颊肌肉微动 → 颧骨位置调整 → 眼神改变 → 整个面部"换脸"（**类似《碟中谍》阿汤哥的面具特效，但这次是肌肉自我调整**）
- **远郊蒙太奇**：从嵩阳中心大厦天顶俯拍 → 镜头一路下摇 → 经过城市天际线 → 进入远郊小镇 → 最终停在暗学帮园区门口的招牌（**一镜到底 + 镜头时长 30 秒**）

**特效需求**：
- **隐匿符**：金色流光 + 屏幕破屏特效（参考《哈利·波特》魔法从书中飞出）
- **赤髓浑元气变脸**：面部肌肉调动的逐帧展示（参考《终结者 2》T-1000 变形）
- **远郊晨光蒙太奇**：太阳从市中心方向升起，光线扫过城市天际线、远郊农田、凡人救济摊

**动捕需求**：张羽在远郊的跑步姿势可使用轻功（跑车腿、电梯纵）的奔跑动捕——参考第 92 章铺垫，第 93 章首次实战展示。
