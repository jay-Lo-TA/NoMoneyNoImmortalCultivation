# 场景SC-059-01-嵩阳高中体育练功场-周澈尘对峙+威压符

> **场景元信息**
> - 重要级：**P0 必做核心CG**
> - 物理位置：嵩阳高中·体育练功场（周澈尘出场 + 张翩翩威压符对峙）
> - 首次出现：第 59 章
> - 复用章节：第 59 章

## 🎬 场景描述

嵩阳高中体育练功场的**高潮对峙场**。这场 CG 是**全剧动画视觉差异化最具代表性的场景之一**——也是**张翩翩完整登场的代表作**。

### 第一幕：周澈尘登场（神权挑战）

**蓝岭被张羽拘役符压制后**，突然周澈尘**踏冰霜而来**——整个练功场的温度**急剧下降**，空气中弥漫着冰霜雾气，蓝岭身上冒出热气白烟。

周澈尘：
- 高二第一、学生会副主席（高三会长忙于高考、实际掌权者）
- "学生中的学生，学长中的学长"
- 踏入练功场时，所有学生齐齐鞠躬"会长好"
- **眼神瞬间扫描全场** → 用天眼符观察张羽 → 看到"嵩阳市巡察队外包员工（展开详情）"
- 冷笑："巡查队区区一个外包，米粒大一点的职位，也敢在嵩阳高中撒野？"
- 抬起手掌，**冰蓝色寒气浮现**

**视觉重点**：
- **温度下降的物理感**（地面出现霜痕、空气出现白雾、学生们缩脖）
- **周澈尘的身后拖出冰霜长尾**（缓步踏来，气场弥漫）
- **掌中冰蓝寒气的微型风暴**（参考《火影忍者》冰遁血继限界）

### 第二幕：双幽朋登场（参考前一届的堕落）

周澈尘叫出**双幽朋**——高一曾是总分第一+实战第一的天才少年，如今面对周澈尘**畏畏缩缩**："大意轻敌，狂妄自大，不识敌我差距。"——曾经的意气风发完全消失，像一条摇尾乞怜的狗。

**视觉重点**：双幽朋的**"过去 vs 现在"对比镜头**——双手不知道往哪放、抠耳朵、抓住头发——与之前高一意气风发的他形成时空错位感。

### 第三幕：张翩翩威压符登场（**全剧最震撼 CG 之一**）

就在周澈尘掌中寒气即将爆发时——**一股暖风从四面八方吹拂而来**，不但吹散寒气、还恢复温度。**张翩翩与一名男子（嵩阳高中高三第一+学生会会长楚承宣）一同从大门外走入**。

张翩翩淡淡道："你要袭击我手下的临时工？"

**周澈尘目光一凝："张翩翩？她怎么来这里了？这是要为张羽撑腰？"**

张翩翩走向场中央，**右手食指抬起，箓书已飞至指尖之前**，高速划过——

> **威压符——发动！**

张翩翩头顶上方浮现出**两行字**：

**第一行**：「白龙高中高三年级月考总分：**699分**」——全场学生震惊："高三月考 699 分？这是神！未来的高考之神啊！"

**第二行**（震撼级视觉）：「银行余额：**2**」 → 数字每秒跳动一位 → 2万5千 → 25万 → 250万 → **2亿5千万** → **停在 25 亿**！

**全场**："光银行余额就有 25 亿？这个女人的总资产到底达到了什么地步？"

### 第四幕：天眼符的"你不配知道"

何大有忍不住用天眼符查看张翩翩底细——但只看到对方头顶**五个大字**：「**你不配知道**」。

何大有"如遭雷击"，"宛如看到了一道天堑，正横亘在他与张翩翩的面前，让他永远也无法跨越"。

**视觉重点**：
- **699 分的浮空大字**（黑底金边 + 数字发光 + 镀金特效）
- **25 亿数字的逐位滚动**（每秒跳一位 + 数字下方金光流动 + 屏幕四周的光波辐射）
- **"你不配知道"五个大字的浮空显示**（红字白底 + 警告边框 + 何大有天眼符视角的画面抖动）
- **赵天行数数字数错**的可爱表情（内心独白："太有钱了，有钱人的威压真的好强啊！")
- **张翩翩"如同国王接受臣子膜拜"的构图**——所有人弯腰鞠躬，张翩翩独自站在高处
- 周澈尘眼角抽搐、语气客气："张学姐，你误会了，我不知道张羽是你的手下……"
- 张翩翩淡淡回应："不要让我听到任何你对我手下不利的事情，不然我头上这些零花钱就用来买你的脑花。"

## 📐 CG 制作要点

- **空间设计**：练功场温度变化（冷 → 暖）、光影层次（暗 → 亮）、画面色调（蓝灰 → 金光）
- **关键帧**：
  - 镜头 1：周澈尘**踏冰霜登场**的缓推镜头（脚部特写 → 全身 → 表情）
  - 镜头 2：周澈尘掌中**冰蓝色寒气凝聚**的特写（寒气的微观风暴 + 学生们口中冒白气）
  - 镜头 3：**双幽朋畏缩登场**（与高一巅峰时期照片对比的"过去 vs 现在"叠化镜头）
  - 镜头 4：**暖风吹散寒气**的温度变化（地面霜痕融化 + 空气白雾散去 + 暖色调渗入）
  - 镜头 5：张翩翩**从大门走入的低角度镜头**（光晕 + 修长身形 + 白龙校服）
  - 镜头 6：张翩翩指尖**箓书飞跃的慢镜头**（书本旋转 + 巡察部/白龙高中徽记闪光）
  - 镜头 7：威压符发动 → **699 分浮空**（金字 + 黑底 + 镀金光晕）
  - 镜头 8：**25 亿数字逐位跳动**（每秒一位 + 数字下金光流动 + 浮空闪烁 + 全场学生面孔特写切换）
  - 镜头 9：何大有**天眼符反弹**的特写（红色大字"你不配知道"+ 何大有"如遭雷击"的五官）
  - 镜头 10：赵天行**数错 25 亿**的可爱反应（数完又数、眉头皱起、嘴里嘟囔）
  - 镜头 11：周澈尘**眼角抽搐**（面部微表情 + 嘴角勉强挤出的微笑）
- **特效需求**：
  - **威压符的两行浮空字**（参考《死亡笔记》L 的审判文字 + 《JOJO 奇妙冒险》替身能力的浮空字）
  - **数字滚动的机械美感**（每一秒精确切换一位 + 数字旋转出现）
  - **温度变化的全场物理感**（冰霜 → 暖风 的环境光变化）
  - **箓书飞跃**的轨迹光带（参考《哈利·波特》魔法书飞行的光迹）
- **动捕需求**：张翩翩的**"冰山女王"气场**（缓步前进 + 眼神压迫 + 说话时轻描淡写）；周澈尘**"从嚣张到客气的微表情变化"**；赵天行**"数错数字的可爱表情"**（喜剧担当）
- **建议时长**：5-7 分钟（**P0 必做核心CG之一**）
