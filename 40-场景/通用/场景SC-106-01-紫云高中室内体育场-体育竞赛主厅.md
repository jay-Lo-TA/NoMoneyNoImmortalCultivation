# 场景SC-106-01-紫云高中室内体育场-体育竞赛主厅

> **场景元信息**
> - 重要级：**P0 必做**（100 章里程碑·高一体育竞赛主场）
> - 物理位置：嵩阳市·紫云高中·最大的室内体育场（体育竞赛主厅+灵敏关+速度关+力量关）
> - 首次出现：第 106 章
> - 复用章节：第 106 章、第 107 章、第 108 章、第 109 章、第 110 章

## 🎬 场景描述

紫云高中最大的室内体育场，是**第 106-110 章高一体育竞赛的主场**——比第 105 章夜里的紫云初登场更震撼的"万米体育场馆"全貌。**这是本作 100 章里程碑 P0 必做核心场景**——张羽 4.95 级肉体强度排名第五的震撼全场。

**注意区分**：本场景是**紫云高中**+**室内体育场**+**体育竞赛（不是法赛）**，与：
- `场景SC-042-02-红塔高中室内体育场-法赛集合`（红塔+法赛+第一关飞剑）
- `场景SC-067-01-白龙高中武道竞赛场馆主厅`（白龙+武道竞赛+决赛擂台）

是三个不同的赛事 + 三种不同的场馆。

**建筑特征**（**P0 必做·宏大场面**）：
- 室内大型体育场，**亮如白昼**——顶部像挂满了一颗颗小太阳
- **可变形结构**——第 110 章灵敏关时，地面能升起墙壁，分隔成一个个小房间
- 巨大记分大屏幕（持续显示前十名）
- 数万观众席位（挤满了人）
- 紫罗兰色灯光 + 紫云药业的赞助广告
- 网红大 V（4 米以上的炼体网红 + 2 米不到的小网红）
- 西装革履的公司人（满脸晦气、班味儿）——参赛公司代表
- 看台最高处 VIP 区（紫云高中年级第二乐景辰 + 网红刚山直播）

**群像阵容**（**100 章里程碑·三大高中对决**）：
- **嵩阳高中**（张羽、白真真、何大有、叶海桃、李勇 + 王海教练 + 钱深、赵天行）— 校服最朴素、待遇最差
- **白龙高中**（玉星寒、宋海龙、楚秋河等）— 白色校服 + 胸前背后 1-10 月考排名数字 + **仿妖筋肉"人形妖兽"**外形
- **紫云高中**（乐沐岚、炼天极、乐景辰等）— 紫色校服 + 眉心紫色印记（紫圣心法越深印记越复杂）
- **红塔高中**（虎云涛、熊文武等）— 红色校服 + **黑龙血脉觉醒"黑色怪物"**外形（黑色龙鳞 + 小角 + 尾巴 + 猩红眼睛）
- **普通高中学生**——浑身上下被包裹起来像木乃伊粽子（穿洁净服避免接触）

**关键道具**：
- **巨大记分大屏幕**（显示前 10 名肉体强度、灵敏、速度、力量总分）
- **比赛规则动画视频**（在大屏幕播放）
- **法力检测贴片**（每个选手佩戴）
- **亢奋剂 / 促智剂 / 神经填充剂**（选手临时服药）
- **高速钢珠发射器**（灵敏关：从四面八方墙壁射出钢珠）
- **光环**（灵敏关：选手脚下半径 3 米的光环，逐渐缩小）

**关键视觉元素（按章节分布）**：
- **第 106 章（人声鼎沸）**：场馆全景 + 各校入场 + 大屏幕"白龙高中楚秋河 4.49 级"+ 张羽白真真进入嵩阳休息区
- **第 107 章（我能摸一下你吗）**：自助餐区 + 红塔"黑色怪物"群像 + 乐沐岚"2000 块摸手"事件
- **第 108 章（张羽 4.95 级）**：大屏幕排名公布 + **张羽 4.95 级震撼全场** + 灵界墨天逸直播视角 + 紫云乐景辰 + 网红刚山的直播解说
- **第 109 章（肉体强度排名）**：虎云涛 4.98 → 宋海龙 5.03 → 熊文武 5.05 → **玉星寒 5.07** 的连番刷新
- **第 110 章（灵敏）**：地面升起分隔墙 + 光环收缩 + 钢珠飞舞 + 张羽"全集中"1.5 分钟

## 📐 CG 制作要点

- **P0 必做核心**：第 108 章**张羽 4.95 级全场震撼 CG**——大屏幕上的名字变化 + 周围人反应（楚秋河怀疑、炼天极感慨、何大有嫉妒、王海困惑、周澈尘警惕、乐景辰分析、乐沐岚关注）的多镜头切换
- **P0 必做核心**：第 109 章**玉星寒 5.07 级登顶 CG**——大屏幕连续刷新的数字特效 + 红塔黑龙血脉的"龙吼声"震撼全场
- **P1 关键 CG**：第 110 章**灵敏关变形**——地面升起分隔墙、墙壁中喷出高速钢珠、光环逐渐收缩
- **群像视觉差异化**：
  - **白龙**：白色校服 + **仿妖筋肉外形**（宋海龙-人形巨象、楚秋河-人形犀牛）——人变妖兽
  - **紫云**：紫色校服 + **眉心紫色印记**（圆点 → 复杂竖瞳）
  - **红塔**：黑色"龙鳞"反射油亮光滑 + **小角 + 尾巴 + 猩红眼睛**（妖变强人）
  - **嵩阳**：朴素校服、待遇最差
- **大屏幕数字特效**：数字跳动的金属感、4.95、5.03、5.05、5.07 等关键数字应有不同的"敲击"音效
- **网红/公司人群像**：网red 身高 4 米以上 + 西装公司人 + 穿着洁净服的木乃伊普通高中生——三种群像剪影对比
- **演出细节**：墨天逸在灵界的"老大"模糊身影 + "嵩阳十六家格局"伏笔
