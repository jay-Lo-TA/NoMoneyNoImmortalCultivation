# 场景SC-075-01-白龙高中淘汰赛擂台-白真真vs熊文武（狂风暴雨+痔疮破防+断臂）

> **场景元信息**
> - 重要级：**P0 必做（最高潮战斗 + 黑色幽默核心）**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·八强擂台
> - 首次出现：第 75 章
> - 复用章节：第 75、76 章

## 🎬 场景描述

**P0 必做最高潮战斗 CG + 黑色幽默核心场景**。白龙高中武道竞赛**八强战**（下半赛区）— **白真真 vs 红塔高中熊文武**。**本作武道竞赛篇"以弱胜强"的最极端表现 + 妖裔"外痔"破绽首次视觉化 + 白真真断臂的"为梦想付出"CG**。

**红塔高中熊文武**（**本作最强肉体 + 300 分 + 妖裔棕熊**）：
- **身高 2 米 7 + 体重近 1 吨**的棕熊妖裔
- **和宋海龙、张羽一样获得了 300 评分**
- **肉体强度 2.5-2.6（现场高一生中最高）**
- 与虎云涛**同属"红塔高中校董的'儿子'"** + 来自妖怪保护区繁育基地
- 第 75 章开头**"父亲训子"CG**：训斥虎云涛"你若是大学巅峰，有名校文凭在手，谁还能除你儿籍？抓你进厂？"
- 比赛前**打 3 管药**（针管扎向脖子） → "肉体强度暂时提升" + 全身充血

**白真真的战斗准备**（**"灵根全开" + 破体剑气**）：
- 感受体内灵根的释放 → **大脑像是被放开了某种束缚**
- **专家级功法"破体剑气" 3 级推动**（一尺剑气从指尖蔓延而出）
- 用灵根看到熊文武体内的灵机变化 → **"全身上下毫无破绽"**（"招式中的破绽……身体上的破绽……"）

**核心剧情**（**4 段递进**）：
- **第 75 章"狂风暴雨"**：
  - 熊文武如炮弹般俯冲 → 白真真连续破体剑气刺去 → 砍在毛发上如砍在钢板上
  - 白真真"潜行"消失 → 熊文武早有准备 → 双爪带起寒光如狂风暴雨
  - 熊文武用鼻子狂嗅 → **通过气味判断白真真方位**
  - 白真真大腿、小腿、腰腹开始出现伤痕（"利爪扫过她的皮肤、肌肉，就像是滚烫的刀锋划过牛油一样"）
  - "最多再有 4、5 秒钟的时间，我就要败了。"
  - 钱深/赵天行/何大有观众席"太猛了" → "要输了……"
  - 终于她看到破绽：熊文武**表皮上密布灵机如铠甲** → **全身上下毫无破绽……不，有破绽的。**
  - 一步踏出 → **"如一只在狂风暴雨中翩翩起舞的蝴蝶"** → 一步步穿过了那扑面而来的锋寒
  - **"我不要留在一层一辈子打工……我要考上十大！我要在仙路上走得更远！"**
  - 白真真指尖剑气轰鸣 → 朝着眼中破绽一剑斩去 → **破防！**
  - 熊文武身后猛地喷出一大片血来（喷泉式）
- **第 75 章"痔疮破防 + 断臂"**（**本作最黑色幽默的 CG**）：
  - 雷钧观察："是痔疮！还是外痔！"（"妖族本来就大多是四肢爬行，拥有人形开始学习、修炼之后，却要每日久坐，痔疮一定更厉害也更大。**特别是外痔已经脱出体内，不受炼体功法的保护！**"）
  - "再加上熊文武刚刚还在剧烈运动，又打了那么多烈性药物，全身充血，这一下大出血就更厉害了。"
  - 雷钧兴奋："**竟然能找到这种破绽？真没浪费我给你补课这么久……**"（**白真真"以弱胜强"完全靠雷钧的战术辅导 + 雷钧对妖裔生理的深入研究**）
  - 熊文武双目赤红 → 一声狂吼 → **濒死野兽模式** → 高速旋转 → 漫天血水+爪影朝四面八方覆盖
  - 雷钧："**这妖怪发狂了！完全在胡乱攻击……**"（白真真腿上被攻击，行动受限）
  - 熊文武倒地 → 白真真站在原地剧烈喘息 → **她的右手臂被发狂的熊文武给扯断了。**
  - **白真真胜！**
  - 雷钧连忙捡起断臂 → 抱着白真真送到急救区
- **第 75 章末"医疗结算"**：
  - 急救医生看白真真："8.8 万走保险 + 3 万自费（祛疤美容）"
  - 白真真："自费？我买保险你要我自费三万？"
  - 急救医生："要治就快，手臂坏死严重的话，那我这可就治不了了，到时候送去医院，恐怕三十万都打不住。"

**关键道具**：
- 破体剑气的金色剑光
- 熊文武的"大出血"喷泉
- 痔疮的视觉化（**本作首次公开的"妖裔生理破绽"**）
- 白真真的**断臂**（**本作最惨烈的女性角色"为梦想付出"** CG）
- 急救医疗区

**氛围**：暗红 + 金色（白真真的破体剑气） + 暗蓝（潜行灵根） + 灰白（妖裔毛皮） — **多色对比的极限战斗**。

## 📐 CG 制作要点

- **空间设计**：
  - 八强擂台（4 个擂台之一）→ 战斗持续到擂台爆破状态
  - 熊文武的妖裔身躯（2 米 7 + 棕熊皮毛 + 散发寒光的利爪）
  - 白真真的嵩阳高中校服（被撕破 + 血染）
  - 急救医疗区
- **关键帧**（**P0 必做 / 4 段递进**）：
  - **赛前"父亲训子"CG** — 熊文武巴掌拍在虎云涛脑门上（"开除儿籍！厂里打工！"） — **本作首次"妖裔父子"父权 CG**
  - **白真真"蝴蝶穿过狂风暴雨"** — 灵根全开 + 翩翩起舞穿过漫天爪影
  - **"我不要留在一层一辈子打工……我要考上十大！"** — **白真真人物弧光最高光台词 + CG**
  - **破体剑气命中破绽** — 熊文武身后**喷泉式大出血**（**本作最震撼的出血 CG**）
  - **雷钧观察发现"外痔破绽"** — "是痔疮！还是外痔！"（**本作最黑色幽默的 CG**）
  - **熊文武"濒死野兽"高速旋转** — 漫天血水+爪影 + 雷钧"这妖怪发狂了"
  - **白真真断臂** — 雷钧**连忙捡起断臂**（**本作白真真最惨烈的付出 CG**）
  - **急救医生"讹钱"** — "8.8 万走保险 + 3 万自费"（**金融讽刺核心镜头**）
- **特效需求**：
  - 破体剑气的金色剑光
  - 熊文武的"大出血"喷泉特效（**本作最大量的流体特效**）
  - 痔疮的视觉化（**本作首次**）
  - 断臂的物理特效（**本作最惨烈 CG**）
  - 急救法力的治愈流光
- **动捕需求**：
  - 熊文武的"熊扑 + 濒死野兽"动作（**需要武打替身**）
  - 白真真的"蝴蝶穿过狂风暴雨"动作
  - 雷钧的"捡起断臂"紧急动作
  - 急救医生的"讹钱"表演
- **音频设计**：
  - 熊文武的熊吼声
  - 破体剑气的"嗡——"剑光音
  - "大出血"喷泉的"噗——"血液音
  - 雷钧"是痔疮！还是外痔！"的喊声
  - 急救医生"8.8 万走保险 + 3 万自费"报价
  - 白真真"自费？我买保险你要我自费三万？"的怒吼
- **建议时长**：8-10 分钟（**全作最长单场战斗 CG**，4 段递进）
