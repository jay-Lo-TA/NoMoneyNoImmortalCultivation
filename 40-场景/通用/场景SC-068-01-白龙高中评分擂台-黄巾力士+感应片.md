# 场景SC-068-01-白龙高中评分擂台-黄巾力士+感应片

> **场景元信息**
> - 重要级：**P0 必做**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·评分赛擂台（8个分赛场）
> - 首次出现：第 68 章
> - 复用章节：第 68、69、70 章（评分赛 + 小组赛共用）

## 🎬 场景描述

**P0 必做核心 CG 场景**。白龙高中武道竞赛的**评分赛擂台** — 8 个小组擂台同时进行评分赛，**黄巾力士**作为标靶依次与各参赛选手交手。**本作首次大规模群像战斗的视觉化**。

**建筑特征**：
- **8 个分赛场擂台**，每个擂台都贴有**感应片**（黄巾力士身上 5+3+1 分不同位置 + 选手身上扣分）
- 比赛计时**1 分钟**，黄巾力士维持**1.98 级肉体强度**出力
- 擂台边缘有"打出场外 +300 分"规则

**黄巾力士的关键视觉**（**本作首次机器人物**）：
- 2 米多高的人形机器人，**身披金色铠甲 + 头戴金色面罩 + 裸露出大理石般的肌肉**
- **印刻着密密麻麻的符文**（道教金光阵）
- 工作人员在它**胸口喷上"绿洲食品"广告 + 背部喷上"微蓝医药"广告**
- "绿洲食品 = 十大宗门幽冥宫"背景
- "微蓝医药 = 战斗用药物公司"背景

**关键选手的评分赛镜头**（**群像战斗 CG**）：
- 第 68 章白龙学生首场：1 分钟内得 128 分（"刚猛无俦"风格）
- 第 68 章**宋海龙**：抓住黄巾力士的双手，**1 秒钟甩飞出场外 → 300 分**（"轰得一声"全场震惊）
- 第 68 章**炼天极**：**拳脚如狂风暴雨**将黄巾力士压制 → 261 分
- 第 68 章**红塔高中熊**（棕熊妖裔）：**将黄巾力士扑倒在地 + 甩飞** → 300 分
- 第 68 章**乐沐岚**：**一只手按在地上 + 一脚脚踹背心感应片** → 305 分（"正好超过宋海龙 300 分" + 转身瞥宋海龙一眼）
- 第 69 章**张羽**：站在擂台边缘 → 站位诱导黄巾力士到边缘 → **一招直拳+无极云手牵引** → 摔出场外 → 300 分
- 第 69 章**白真真**：**隐藏实力 → 仅得 89 分**（战斗方式不适合硬打硬拼）
- 第 68-69 章穿插：**白真真看着黄巾力士评价** — "嵩阳高中就是逊啦"

**关键道具**：
- **黄巾力士**（本作首次非人类战斗单位）
- **感应片**（贴在不同部位）
- **8 个分赛场擂台**
- **3D 实时计分系统**

**氛围**：金光闪闪（金色铠甲） + 绿洲食品绿色广告 + 微蓝医药蓝色广告 — **金钱气息的视觉化**。

## 📐 CG 制作要点

- **空间设计**：
  - 8 个分赛场擂台同时展示
  - 黄巾力士的机甲设计：**金色铠甲 + 大理石肌肉 + 符文 + 喷涂广告**
  - 感应片贴在不同部位
  - 1 分钟倒计时
- **关键帧**（**P0 必做**）：
  - **黄巾力士登场** — 金色身影缓缓走入 + 喷涂广告 + 符文发光
  - **宋海龙 1 秒甩飞 300 分** — 全场震惊 + 评分系统闪"300"
  - **乐沐岚 305 分超越** — 镇定 + 转身瞥宋海龙一眼
  - **张羽"取巧派"300 分** — 站擂台边缘 + 提前拦截 + 摔出场外
  - **白真真"隐藏派"89 分** — 艰难战斗 + 评分系统闪"89"
- **特效需求**：
  - 黄巾力士的金色符文发光
  - 感应片被击中时的金色数字弹出
  - 评分系统的 3D 实时计分 UI
  - 张羽无极云手牵引的"借力打力"水流特效
- **动捕需求**：
  - 黄巾力士的统一动作（**需要武打替身**）
  - 宋海龙的"巨力甩飞"
  - 乐沐岚的"单手压制+踹击"
  - 张羽的"取巧诱导"
  - 白真真的"隐藏实力"（动作故意放水）
- **音频设计**：
  - 倒计时 3/2/1 的电子合成音
  - 感应片被击中的"噼啪"声
  - 计分系统的"叮"提示音
  - 全场哗然的人声背景
- **建议时长**：5-8 分钟（评分赛全过程的蒙太奇）
