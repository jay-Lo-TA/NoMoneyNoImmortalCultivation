# 场景SC-044-01-红塔高中法赛飞剑考场

> **场景元信息**
> - 重要级：**P0 必做**
> - 物理位置：嵩阳市·红塔高中·室内体育场（法赛第一关·飞剑考场）
> - 首次出现：第 44 章
> - 复用章节：第 44 章

## 🎬 场景描述

**本作第一场大规模飞剑视觉戏 + P0 必做重点场景**。

红塔高中室内体育场的飞剑考场。**18 个分赛场 + 半透明法力护盾** 隔开，每批 18 名选手随机抽签上场，操控飞剑击打飞舞在半空中的巴掌大小信号牌，速度越快、力度越准，得分越高（满分 100）。

**飞剑设定**（"九天飞剑"厂商赞助）：
- 民用级法力飞剑，**不需要学习御剑之术**
- 通体成刃，**无剑柄**（区别于传统武侠剑）
- 剑鞘是遥控器 + 法力传输器
- 通过输入法力的频率、类型、大小进行控制

**赛场高光**（按出场顺序）：
- 钱深（嵩阳高中）— 72 分（"纯真分奴"）
- 何大有（嵩阳高中）— 78 分（"校董之子"）
- 炼天极（紫云高中）— 90 分（"天勤之子"，手臂一甩）
- 乐沐岚（紫云高中）— 95 分（"冰肌玉骨" + 高种姓，当前最高）
- 白真真（嵩阳高中）— 34 分（"剑批拳批掌批刀批枪批药批分霸贷奴卡奴道术无用论嵩阳黑子她很穷"评价巨长）
- **宋海龙（白龙高中）— 100 分满分**（外置灵根加持 + 御使飞剑如闪电）
- **张羽（嵩阳高中）— 即将上场**（本章末悬念）

**关键道具**：
- 飞剑（无剑柄，通体成刃，悬浮半空）
- 信号牌（巴掌大小，飞舞）
- 18 个法力护盾分赛场
- 剑鞘（法力遥控器）
- 寒星野的"天眼符"（金色光粒 + 头顶悬浮评价）
- 宋海龙的外置灵根假发（冲天长发，拔下后变秃头，**视觉笑点**）
- 记分板（实时显示分数）

**氛围**：史诗级对决 + 财富差距直观展示 + 张羽的"穷人破局"悬念。

## 📐 CG 制作要点

- **美术风格**（**P0 重点 CG，必做**）：
  - 修仙版"奥运会飞剑项目"：现代体育馆 + 仙道科技 + 法力视觉
  - 色调：赛场白光大背景 + 飞剑寒光 + 法力护盾蓝紫
  - 飞剑美学：通体成刃无剑柄的"无柄飞剑"（区别于传统武侠剑，**视觉差异化卖点**）
- **镜头语言**（**高密度分镜**）：
  - 镜头 1：主持人念广告词（"九天剑光寒，一剑震昆墟"）
  - 镜头 2：18 个分赛场全景（法力护盾透明分格）
  - 镜头 3：钱深 72 分 / 何大有 78 分（嵩阳高中"豪门富人学生"组）
  - 镜头 4：炼天极 90 分（手臂一甩的干脆动作）
  - 镜头 5：乐沐岚 95 分（冰山美少女 + 紫色校服 + 当前最高）
  - 镜头 6：白真真 34 分（评价巨长 UI + 失落）
  - 镜头 7：**宋海龙拔下外置灵根假发瞬间**（"卧槽！高中就秃了？"的视觉笑点）
  - 镜头 8：宋海龙 100 分满分的御剑英姿（外置灵根加持 + 如闪电般的飞剑轨迹）
  - 镜头 9：寒星野天眼符"男凝"全场的视角（**主角群像头顶评价 UI**）
  - 镜头 10：张羽握紧剑鞘的特写（"所以不能玩传统的飞剑……"悬念蒙太奇）
- **特效需求**（**P0 优先级最高**）：
  - 飞剑破空特效（寒光轨迹 + 音爆）
  - 信号牌被击中的火花 + 金属声
  - 18 个法力护盾（半透明蓝紫色场域）
  - **外置灵根**视觉（冲天长发摘下瞬间的灵光散逸 + 秃头反差）
  - 天眼符 UI（金色光粒汇聚双眼 + 人物头顶悬浮文字"纯真分奴/校董之子/天勤之子/冰肌玉骨"等）
  - 主持人念广告词时的全息广告字幕
- **动捕需求**（**高强度**）：
  - **大量飞剑投射动作**（数十次，不同选手）
  - 宋海龙的拔灵根 + 御剑组合动作
  - 张羽握剑鞘 → 第 45 章延续的"破局"动作预备
- **音频设计**：
  - 飞剑破空 + 信号牌击中的清脆金属声
  - 外置灵根拔出瞬间的灵光音效
  - 观众惊叹声（钱深 72 → 何大有 78 → 炼天极 90 → 乐沐岚 95 → 宋海龙 100 的层层递进）
  - 张羽"所以不能玩传统的飞剑……"内心独白
  - 紧张 BGM（暗示贫富差距 + 张羽的破局悬念）
