# 场景07-天人演武图武学演练

> **场景元信息**
> - 重要级：**P0**
> - 物理位置：嵩阳中心大厦 999 层 · 天人演武图展示区（5 米观察范围内）
> - 首次出现：第 20 章
> - 复用章节：第 20 章（张羽双修行为 + 武学领悟）、第 21 章（九门武学演练 + 失败 + 入门）

## 🎬 场景描述

天人演武图前 5 米的"参悟区"——多名高中生在此尝试练成图中隐藏的武功（其实是一门以武道意念淬炼道心的"天武炼心诀"）。**画面要素**：
- 张羽的特殊行为：吐纳 → 站起来参悟 → 练健体三十六式 → 拿出手机刷教材公放 → 边吃自助餐边参悟——形成"一念多动"的奇观
- 武学本身：图中所藏其实是九门不同兵器/拳种（拳、掌、腿、刀、剑、指法…），每门对应一种武道意念（杀意、斗志、气魄、勇武…）
- 张羽的成功：以手机录像照搬白真真的九门武学招式，但最终领悟"重意不重招"的奥妙，凝练武道意念至 90% 程度
- 张羽的失败：因道心等级仅 1 级（要求 3 级），最终"棋差一招"未竟全功，获得"天武炼心诀 0 级（1/1）"

## 📐 CG 制作要点

- **美术风格**：动态水墨 + 仙道光效；张羽演练时身上杀意/斗志/气魄/勇武等不同色光交替浮现
- **镜头语言**：
  - 关键镜头①：张羽"一念多动"的快剪蒙太奇（吐纳 ↔ 练功 ↔ 看手机 ↔ 吃大虾），体现"摆烂思想突破仪式压迫"
  - 关键镜头②：张羽照搬白真真视频动作的尴尬姿态（保安服 + 邯郸学步）
  - 关键镜头③：张羽演练至高潮时武道意念凝练的爆发瞬间（9 种颜色光芒从体内涌出）
  - 关键镜头④："棋差一招"的失败瞬间：光芒骤然消散、画上浮现"天纵之资，棋差一招"字样
- **特效需求**：
  - **武道意念可视化**：金/红/紫/蓝等不同颜色代表不同武道意念
  - **画作反应**：张羽演练时画中人影变得越来越清晰，最终出现鲜艳色彩，如同真人
  - **羽书 UI 浮现**："天武炼心诀 0 级（1/1，8.3/8 法力，0.85/0.8 肉体强度，1/3 级道心）"
- **动捕需求**：
  - 张羽吐纳 + 健体三十六式的标准动作
  - 九门武学的完整套路（拳、掌、腿、刀、剑、指法…），需大量动捕数据
- **首登场**：第 20 章，本场景是 OP/ED 候选画面
