# 场景SC-058-01-嵩阳高中体育练功场-高二学长打压实战

> **场景元信息**
> - 重要级：**P0**
> - 物理位置：嵩阳高中·体育练功场（被改造成"高二学长指导高一实战"现场）
> - 首次出现：第 58 章
> - 复用章节：第 58、59 章（与 SC-059-01 配合构成两幕式 CG）

## 🎬 场景描述

嵩阳高中体育练功场被临时改造成**"跨年级实战指导"现场**。**蓝岭**带领的高二示范班（蓝岭、双幽朋等）整齐站立，围成半圆；高一（1）班学生则列队站在另一侧，逐一鞠躬"学长好"——这是嵩阳高中乃至昆墟高中圈层森严的弟位秩序。

### 第 58 章前半：白真真被欺
蓝岭（身材最高大、像门板一样的肌肉男，**已完成"绝对"绝育手术**，身上无任何性别特征，胳膊布满针孔）登场"指点"白真真。白真真上场即偷踹蓝岭小腹丹田要害——蓝岭躲都不躲、面不改色，反而反手捏住白真真的脚踝。蓝岭肌肉暴涨将白真真**猛地甩飞**，白真真带着残影即将撞上墙壁——**武道老师雷钧脚下砰一声巨响、电射而出**，在最后一刻接住白真真。**墙壁被撞碎**，两人飞出练功场外。

雷钧警告蓝岭"下手太重了吧"，蓝岭微笑回应"正好给学妹一个实战轻敌的教训"——继续邀战。

### 第 58 章后半：张羽拘役符登场
张羽从练功场大门外走入，**右手在黑白相间的箓书（巡察部标记）上高速划过**——

> **拘役符——发动！**

无数金色光点闪动，化为**九只金色巨掌**，分别按住蓝岭的脑袋、肩膀、双臂、双腿、腰腹——**蓝岭被死死压制**。全场学生震惊：
- "拘役符？"
- "张羽这穷鬼……他哪来的箓书施展拘役符的？"
- "他竟然买了箓书？难道他今天请假是去抵押器官的？"

**视觉重点**：
- **学弟鞠躬的秩序感**（画面定格全员鞠躬的"哥特式构图"——镜头从上往下，地面上只看得见一双双鞋尖）
- **蓝岭的体型压迫感**（门板一样的身高 + 全身肌肉的重量感）
- **白真真被甩飞的**残影慢镜头**（空中 720° 旋转 + 撞击镜头的"打脸"特效）
- **雷钧脚下爆发式救人**（脚下水泥地爆裂、身影如电射）
- **墙壁被撞碎**的**飞溅碎片慢镜**
- **拘役符的金色巨掌**（九只巨掌从虚空中凝聚 + 锁链质感 + 金色光粒扩散）
- **张羽指尖划过箓书的特写**（黑底白纹 + 指尖法力流动 + 巡察部徽记闪动）

## 📐 CG 制作要点

- **空间设计**：体育练功场 + 学弟学长分列两侧 + 中央空出对峙区——形成**"法庭审判"的舞台构图**
- **关键帧**：
  - 镜头 1：俯拍**学弟全员鞠躬的画面**（广角镜头 + 镜头缓慢下移 + 多人鞋尖入画）
  - 镜头 2：白真真偷踹蓝岭小腹——**慢镜脚部特写**
  - 镜头 3：蓝岭无反应、反而反手捏住白真真脚踝（**力量对比镜头**）
  - 镜头 4：白真真被甩飞的**720° 慢镜空中旋转** + 头发飘散 + 校服飘起
  - 镜头 5：白真真即将撞墙的**时间冻结**（最后一帧画面变红）
  - 镜头 6：雷钧脚下**水泥地爆裂** + 身影化为一道黑色光痕
  - 镜头 7：雷钧抱住白真真撞碎墙壁（**镜头跟着人物一起穿墙**——内景变外景）
  - 镜头 8：张羽**从大门走入的低角度镜头**（逆光剪影 + 身后大厅门框光线）
  - 镜头 9：张羽指尖**划过箓书的超高速特写**（指尖 + 黑白书页 + 巡察部徽记闪烁 + 法力流光）
  - 镜头 10：拘役符发动的**九只金色巨掌凝聚过程**（光粒→光圈→巨掌→锁链）
  - 镜头 11：蓝岭被按住的**面部特写**（震惊+愤怒+青筋暴起）
- **特效需求**：
  - **拘役符的金色粒子特效**（参考《鬼灭之刃》日轮刀的符咒纹路）
  - **蓝岭肌肉暴涨**的微动效（参考《进击的巨人》艾伦变身）
  - 墙壁被撞碎的**CGI 物理破坏**（混凝土碎片 + 钢筋暴露）
  - 张羽指尖的**法力流动粒子**（黑底白纹中流动的金色光带）
- **动捕需求**：白真真被摔飞的空中挣扎动作；蓝岭的"门板人"体型压迫感（武指需要展示绝对力量差）；张羽登场时的**从容淡定表情**（与平时的焦虑形成对比）
- **建议时长**：4-5 分钟（**整场 CG 是动画视觉差异化的关键场次**）
