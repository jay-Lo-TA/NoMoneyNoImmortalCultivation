# 场景SC-096-01-嵩阳高中天台-张羽邀请阿真入伙+炼天极买衣误会

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳高中·教学楼天台（放学后）
> - 首次出现：第 96 章
> - 复用章节：第 96、97 章

## 🎬 场景描述

**本场景是"张羽邀请白真真入伙黑暗补课界" + "炼天极买衣喜剧误会"的双线CG，是原作黑色幽默的精华场景。**

空间：嵩阳高中·教学楼天台（放学后）。**与 SC-038-01 嵩阳高中天台实战对练场同一物理空间，但本场景重在"对话戏+信息密度"，不是实战训练**。

氛围：放学后的私密空间。**这是两人在校内的秘密训练基地之一——过去的这些天里，两人在这里留下了无数努力的汗水。** 此刻则是"招合伙人"的天台密谈。

关键剧情节点：
- **炼天极买衣服误会**（**全章最高能的笑点**）：
  - 炼天极发来消息："你穿过的衣服，卖我一件。"（**白真真看到后脑补"张羽在卖沟子"**）
  - 白真真内心："羽子你这也走太远了，我追不上啊。"
  - 白真真："就不能找点真正的学霸吗？哪怕你找宋海龙也行……不行，宋海龙的肉体强度太高，医疗费可能都不够赔的。"
  - 白真真："如果是乐沐岚花钱的话，妈妈我都更能接受一点啊！"
  - 真相揭晓：张羽压根没卖沟子，是乐沐岚通过炼天极来买张羽的衣服（**洁癖脱敏治疗**）
  - 价码博弈：白真真建议报价 5000 → 炼天极秒同意（**黑色幽默：5000 块买件衣服，比补课一天还多**）
- **张羽邀请阿真入伙**：
  - 张羽："阿真，黑暗补课界你知道吗？"
  - 白真真："好消息，儿子没卖沟子。坏消息，儿子加入帮派，干着比抢劫还危险的工作去了。"
  - 张羽："我在那里能拿时薪 6500。"
  - 白真真："嗯，我信你，你就说要我怎么做吧。"

光照：
- **天台傍晚**：夕阳西下 + 暖橙色调 + 城市天际线作为背景
- **手机屏幕 UI 特写**：羽书界面 + 炼天极的消息框（**冷蓝色 UI 与暖橙天空形成反差**）

关键道具：
- **张羽手机**（炼天极消息 + 乐沐岚的转账记录 + 羽书 UI）
- **天台围栏 + 校园远景**（夕阳下的校园俯瞰）
- **张羽和白真真的"哥们勾肩"姿势**（白色无相云罡云雾隐隐闪动）

## 📐 CG 制作要点

**镜头语言**：
- **开场**：白真真视角看手机 → 镜头推到屏幕上"你穿过的衣服，卖我一件" → 白真真的脸震惊 → **关键帧：白真真瞳孔地震**
- **脑补蒙太奇**：
  - 张羽在白龙高中武道赛后（白龙高中厕所 10 块、巨幅广告、肌肉大汉环绕）的场景闪回
  - 镜头切回天台，白真真一脸凝重地看着张羽
- **真相揭晓**：张羽点开乐沐岚的朋友圈 + 给白真真看聊天记录 → 白真真的脸从震惊转为尴尬
- **天台对话分屏**：左半边张羽劝说 + 右半边白真真面部表情变化（**双线叙事**）
- **结尾勾肩**：两人在天台围栏前的剪影 + 夕阳西下 + 城市天际线（**兄弟情深的标准镜头**）

**特效需求**：
- **手机消息 UI**：参照现代社交软件消息气泡设计，但加上羽书品牌色
- **脑补蒙太奇**：用浮空字 + 闪白特效区别于现实时空
- **无相云罡**：张羽劝说话语时周身云雾隐隐闪动（**气场感**）

**表演要求**：
- **白真真的"卖沟子"脑补**需要细腻的表情变化——从震惊到复杂到佩服到心疼到同情的层层递进
- **张羽的"完全不知情"**需要"无辜到底"的喜剧表演
- **邪神娃娃的暗中吐槽**：可以通过张羽胸口的念珠微震动 + 字幕条形式呈现
