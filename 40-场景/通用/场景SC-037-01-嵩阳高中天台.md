# 场景SC-037-01-嵩阳高中天台

> **场景元信息**
> - 重要级：**P2**
> - 物理位置：嵩阳高中·教学楼天台
> - 首次出现：第 37 章
> - 复用章节：第 37 章

## 🎬 场景描述

嵩阳高中教学楼天台——放学后的秘密训练场。空间开阔、水泥地面、四周有铁栏杆和储物水箱。背景是嵩阳市的天际线：远处的高楼大厦、低空飞过的飞剑、夕阳西下的暖光。

**核心场景元素**：
- **张羽 vs 白真真** 私下全力对练（不同于课堂上的对练——可全力施为、不暴露底牌）
- 张羽散手 4-5 级（已突破）vs 白真真散手 + 高中基础剑术
- 拳风剑指交错，风雷之声
- 白真真剑指虚晃，张羽被「手忙脚乱」逼退
- 白真真语音指导：「不要看我的手，你要看我的腰胯、肩膀来预判我的动作。不要老想着靠自己对散手的熟悉度来进行判断……」

**视觉重点**：夕阳逆光 + 风衣飘动 + 拳脚碰撞的冲击波光效 + 两人激烈的攻防节奏 + 训诫式对话的构图（白真真站前指导，张羽低头听讲）。

## 📐 CG 制作要点

- **空间设计**：
  - 水泥地 + 铁栏杆 + 储物水箱 + 通风口
  - 远景：嵩阳市天际线、夕阳、飞剑轨迹
  - 灯光：夕阳金色 + 阴影拉长
- **关键帧**：
  - 张羽出拳 + 白真真剑指格挡的瞬间（冲击力波纹特效）
  - 张羽被白真真剑指逼得手忙脚乱时连续退步的慢镜头
  - 白真真背对夕阳站立训话，张羽低着头擦汗的剪影镜头
  - 远景：天台上的两个剪影在城市天际线前对练
- **特效**：
  - 拳风 / 剑指冲击波（淡蓝气浪）
  - 散手 4-5 级肉体爆发时青筋暴起 + 肌肉充血（写实 CG）
- **建议时长**：2-3 分钟（实战切磋 + 训诫对话，是人物关系的深化戏）
