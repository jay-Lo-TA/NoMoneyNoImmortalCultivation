# 场景SC-028-01-嵩阳高中电脑机房-道术考场

> **场景元信息**
> - 重要级：**P2**
> - 物理位置：嵩阳高中·电脑机房（道术考场）
> - 首次出现：第 28 章
> - 复用章节：第 28 章

## 🎬 场景描述

嵩阳高中的电脑机房改造为道术考场。一排排学生电脑桌整齐排列，每张桌上一台显示器。屏幕显示统一的道术考试界面（第一题：交通部万行四通大神符咒选择题）。机房环境冷白灯管照明，空气中带着电脑散热风扇的低频嗡鸣。

**关键道具**：学生电脑（显示符箓考题）、教材堆（赵天行临考翻教材）、白真真趴桌小憩、考前讲台手机 / 资料堆。

**核心剧情点**：
- 张羽回忆起符箓知识（交通部万行四通大神、调取道路监控的符咒）
- 超纲题（举报符、拘役符等没学过的内容，4 选 1 的字符乱码选项）
- 何大有炫富交卷 → 玉牌箓书 + 天知万问大神解答符（金色虚影浮现）

## 📐 CG 制作要点

- **美术风格**：校园机房 + 修真考试 + 富人炫富。色调偏冷白 + 显示器蓝光
- **镜头语言**：
  - 镜头 1：机房全景（整齐的学生电脑桌，考试氛围）
  - 镜头 2：张羽电脑屏幕特写（4 个字符乱码选项的"4 选 1"，带选择题的 UI）
  - 镜头 3：赵天行临阵翻教材的紧张表情
  - 镜头 4：白真真趴桌小憩的淡定表情
  - 镜头 5：**何大有玉牌箓书特写**（玉色表面 + 鬼画符 + 金色天知万问大神虚影浮现）
  - 镜头 6：张羽与白真真讨论超纲题（"凭什么超纲 25%"的社会议题戏）
- **特效需求**（**中优先级**）：
  - 符箓字符乱码的视觉化（建议用《哈利波特》魔法文字 + 现代颜文字混搭）
  - 天知万问大神的金色虚影（半透明、庄严神祇感）
  - 玉牌的材质感（高端玉色，带法力光晕）
- **建议时长**：2-3 分钟（含考试过程 + 考后讨论）
