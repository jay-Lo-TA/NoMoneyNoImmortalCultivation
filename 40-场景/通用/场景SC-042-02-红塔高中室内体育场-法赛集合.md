# 场景SC-042-02-红塔高中室内体育场-法赛集合

> **场景元信息**
> - 重要级：**P0**
> - 物理位置：嵩阳市·红塔高中·室内体育场（法赛集合+第一关飞剑比赛）
> - 首次出现：第 42 章
> - 复用章节：第 42 章、第 43 章、第 44 章

## 🎬 场景描述

嵩阳市三大名校之一**红塔高中**的室内体育场，作为本次高一年级法力竞赛（法赛）的举办场地。**这是 P0 必做场景**——大量法赛名场面集中在此。

**建筑特征**：室内大型体育场，顶部有半透明采光穹顶，地面铺 PU 跑道，划分为 18 个分赛场区。每个分赛场有半透明法力护盾隔开。观众席环绕上方，可俯瞰全场。背景是红塔高中的校徽和巨大记分板。

**群像阵容**（**本作首次大场面对手登场**）：
- 嵩阳高中（张羽、白真真、钱深、何大有等）— 校服无特色
- **白龙高中**（宋海龙领军 10 人）— **白色校服 + 胸前背后 1-10 月考排名数字**（辨识度极高）
- **紫云高中**（炼天极、乐沐岚等）— **紫色校服，从飞舟降临**（财富震撼）
- **红塔高中**（东道主，含**小熊猫妖裔嵩阳市人**）— 红色校服为主

**关键道具**：
- 信号牌（巴掌大小，飞舞在空中）
- 飞剑（"九天飞剑"赞助，悬浮半空，通体成刃无剑柄）
- 寒星野的"天眼符"（金色光粒+头顶悬浮文字）
- 飞舟（紫云高中登场道具）
- 魂修生控制的无人机（送道具）

**氛围**：宏大、群像、修仙版"奥运会开幕式"。从集合时的紧张 → 入场式的财富炫耀 → 飞剑比赛时的对决与高光。

## 📐 CG 制作要点

- **美术风格**（**P0 重要视觉场景**）：
  - 修仙版"奥运会场馆"：现代体育馆 + 法力科技 + 仙道美学
  - 色调：以中央白光为主 + 各校校服色块（白/紫/红/嵩阳的素色）形成视觉分组
- **镜头语言**：
  - 镜头 1：体育场大全景（俯视 18 个分赛场）
  - 镜头 2：各校入场蒙太奇（白龙整齐走位 → 紫云飞舟降临 → 红塔东道主登场）
  - 镜头 3：白龙高中白色校服 + 数字的特写（1-10 数字是视觉卖点）
  - 镜头 4：宋海龙 2 米 5 泰坦巨汉 + 冲天长发（假发/外置灵根）的特写
  - 镜头 5：紫云飞舟降临的全景（财富震撼）
  - 镜头 6：飞剑比赛的高光分镜（白真真 34 分 / 钱深 72 分 / 何大有 78 分 / 炼天极 90 分 / 乐沐岚 95 分 / 宋海龙 100 分）
- **特效需求**（**高优先级 P0**）：
  - 飞剑穿空 + 击中信号牌的火花特效
  - 18 个法力护盾分赛场
  - 寒星野的"天眼符"金色光粒 + 头顶悬浮评价文字 UI
  - 紫云飞舟的空气呼啸
  - 各校服色块和数字光效
- **动捕需求**：
  - 大量群像走位（每个校 5-10 人）
  - 飞剑投射动作（重复多次 + 击中反馈）
  - 宋海龙拔下外置灵根假发的"秃头"瞬间（视觉笑点）
- **音频设计**：
  - 入场式 BGM（史诗感）
  - 飞剑破空声
  - 观众惊叹声
  - 信号牌被击中的清脆金属声
