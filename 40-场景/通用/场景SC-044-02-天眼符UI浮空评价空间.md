# 场景SC-044-02-天眼符UI浮空评价空间

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：抽象UI层（万民部·天知万问大神·天眼符评价空间）
> - 首次出现：第 44 章
> - 复用章节：第 44 章

## 🎬 场景描述

**"天眼符"**（万民部天知万问大神法术）的视觉化 UI 层。当使用者的天眼符发动后，视野中所有人头顶都会浮现金色光粒组成的**评价文字**，包括银行资产、个人资产、学校成绩、家庭关系、信仰等级等的**综合模糊评价**。

**核心设定**：
- 使用天眼符 = 视野中所有人的"成分分析"——以"不值一提"为基础默认
- 想要"展开详情"需要更高信仰等级 + 花钱（**信仰等级 + 钱 = 评价清晰度**）
- 看全场人 = 看破产（昆墟历史上"看破产"事件有先例）
- 评价可以很简洁（4 个字"不值一提"），也可以巨长（白真真 1 行 11 项评价）

**关键 UI 元素**：
- 金色光粒（汇聚到使用者双眼）
- 头顶悬浮文字（"纯真分奴" / "校董之子" / "天勤之子" / "冰肌玉骨" / "不值一提" 等）
- 文字颜色编码：默认金色，"稀有/精英"等特殊评价可能有不同色
- "展开详情"交互（点击可展开，如白真真的"剑批|拳批|掌批|刀批|枪批|药批|分霸|贷奴|卡奴|道术无用论|嵩阳黑子|她很穷……"）

**氛围**：修仙版"个人征信系统" + "全息成分分析"—— **本作讽刺金融修仙的视觉卖点之一**。

- 各种被观察的角色（"不值一提"/稀有/精英等评价）

## 📐 CG 制作要点

- **美术风格**（**修仙讽刺美术 + UI 视觉**）：
  - 全息金色光粒 UI（《攻壳机动队》《银翼杀手 2049》）
  - 中国仙道"天眼"美学（道教符箓 + 万民部官僚风）
- **镜头语言**：
  - 镜头 1：寒星野指尖滑箓书 → 发动天眼符（箓书 UI + 念咒式发动）
  - 镜头 2：金色光粒汇聚双眼（POV 主观镜头）
  - 镜头 3：现场大量学生头顶浮现"不值一提"评价（金色字海）
  - 镜头 4：白真真头顶"展开详情"的巨长评价（**视觉高潮**）
  - 镜头 5：乐沐岚的"冰肌玉骨（展开详情）"
  - 镜头 6：宋海龙身上的灵气压力（暗示其超高级别）
- **特效需求**（**P1 重要**）：
  - 金色光粒汇聚双眼
  - 头顶悬浮文字 UI（大量 + 不同色编码）
  - "展开详情"交互式 UI 展示
  - 符箓发动的全息感
- **动捕需求**：
  - 寒星野指尖滑箓书（手部特写）
  - 双眼汇聚金色光粒的特写
- **音频设计**：
  - 符箓发动的"嗡"声 + 全息音效
  - 大量评价 UI 浮现的"叮叮叮"音效
  - "展开详情"的滑入动画音效
  - 反讽 BGM（暗示金融修仙的讽刺）
