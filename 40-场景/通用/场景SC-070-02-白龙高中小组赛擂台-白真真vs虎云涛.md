# 场景SC-070-02-白龙高中小组赛擂台-白真真vs虎云涛（潜行首战）

> **场景元信息**
> - 重要级：**P0 必做**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·小组赛擂台
> - 首次出现：第 70 章
> - 复用章节：第 70 章

## 🎬 场景描述

**P0 必做核心战斗 CG**。白龙高中武道竞赛**小组赛**的一场关键对决 — **白真真 vs 红塔高中虎云涛**。**白真真战斗灵根"潜行"的首次公开战斗** — **本作最"出其不意"的一次胜负**。

**建筑特征**：
- 小组赛擂台（8 个分赛场之一）
- 比赛规则：抽签对决，胜者与其他胜者再对决，决出 1 个 8 强
- 比赛时间：未明确 1 分钟限制（淘汰赛无时间限制）

**红塔高中虎云涛**（**本作首次虎妖登场**）：
- **红塔高中校董的"儿子"** + **来自妖怪保护区**的繁育基地
- **人形身躯 + 老虎的脑袋 + 一身老虎皮毛 + 散发寒光的锋锐利爪**（"妖怪参赛不该做个去爪手术吗？这等于自带兵器！"）
- 头上一连串成分鉴定：**保护区|龙虎妖血|药桶|钢骨**
- 雷钧翻译："来自妖怪保护区，混有龙血的虎妖血脉，吃了很多药，还经过了某种骨骼改造手术。"
- 评分赛 266 分（红塔高中第二高分）
- **六边形战士**：高力高敏高攻高防，战斗中没有表现出任何明显缺陷
- **出身**：红塔高中校董"父亲大人"每个月都会为各个种族的待孕母体播种，让繁育基地内的母体怀上"父亲大人"的孩子。**龙校董的"低分奴隶制"**——"弱者工作，强者读书"。"我不要打工！我要读书！"

**白真真的战斗灵根"潜行"**（**本作首次呈现**）：
- 灵根释放后，**空气中丝丝缕缕的灵机朝着白真真涌去**
- 灵机逐渐**凝聚在白真真的指尖**，像散发出淡淡的剑光
- **白真真在虎云涛眼中消失不见**（"潜行类的战斗灵根？"）
- 虎云涛退回擂台一角寻找白真真 → 找不到 → 抬头看天空 → 还是没有
- 突然再次出现时白真真已**一指点向虎云涛的丹田**
- 虎云涛一爪撕去 → **白真真早已看穿出招轨迹** → 剑指变化向后撤
- 虎云涛追击 → 脚下一痛 → **被白真真绊了一跤**
- **白真真 4 招完美博弈，绊摔虎云涛出场外**
- 虎云涛不服："我不服，我还没出全力！" → 裁判提着后颈肉抓出去

**关键道具**：
- 战斗灵根"潜行"的视觉化（**灵机流光 + 剑光 + 隐身效果**）
- 虎云涛的利爪（"自带兵器"）
- 白真真的破体剑气（指尖延伸的剑光）

**氛围**：暗调蓝紫色（潜行灵根的颜色） + 红色（虎云涛的攻击）— **"暗与红"的视觉对决**。

## 📐 CG 制作要点

- **空间设计**：
  - 小组赛擂台（10 米直径的圆形台）
  - 虎云涛的红塔高中校服（红色为主，**妖裔有"龙校董"logo**）
  - 白真真的嵩阳高中校服（无特色）
- **关键帧**（**P0 必做**）：
  - **虎云涛登场** — 人形虎头 + 散发寒光的利爪
  - **白真真灵根释放** — 灵机朝指尖涌去 + 剑光凝聚
  - **白真真"消失"** — 虎云涛四处寻找（观众视角：白真真还站在那里）
  - **白真真突然出现 + 一指** — 全场震惊
  - **白真真"看穿出招"** — 剑指变化后撤 + 绊摔
  - **虎云涛被裁判提着后颈肉抓出去** — **本作最爆笑镜头之一**
- **特效需求**：
  - **潜行灵根的视觉化**：白真真周围空气扭曲 + 灵机流光 + 剑光
  - 虎云涛的红色攻击轨迹
  - 虎云涛脚下被绊的瞬间慢镜
- **动捕需求**：
  - 虎云涛的"虎扑"动作（**需要武打替身**）
  - 白真真的"消失-出现-剑指"动作
  - 裁判"提着后颈肉"的滑稽动作
- **音频设计**：
  - 潜行灵根的"空气扭曲"特效音
  - 虎云涛虎吼声
  - 全场哗然
  - 虎云涛被抓住的"嗷呜"声
- **建议时长**：2-3 分钟（潜行灵根的完整 4 招博弈）
