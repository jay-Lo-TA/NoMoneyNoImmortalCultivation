# 场景SC-091-02-嵩阳北月高新技术服务业园区-暗学帮前台

> **场景元信息**
> - 重要级：**P0 必做核心CG**（暗学帮首次视觉化登场）
> - 物理位置：嵩阳市·远郊·嵩阳北月高新技术服务业园区·暗学帮前台 105 室
> - 首次出现：第 91 章
> - 复用章节：第 91、92、93、94、97、98 章（暗学帮系列 CG 的"门面场景"）

## 🎬 场景描述

**本场景是"暗学帮"组织的首次视觉化登场 CG，是原作对"帮派现代化"的黑色幽默讽刺的视觉核心。**

空间：嵩阳市远郊·嵩阳北月高新技术服务业园区。**出乎张羽意料：气派的大门 + 时不时进出的车辆 + 保安亭 + "嵩阳北月高新技术服务业园区"招牌——和想象中的阴暗潮湿帮派驻地截然相反**。

前台：约 20 平米接待区，**微笑前台 + 礼貌引导 + "请到 105 室稍作等待，我们的销售人员马上就会过去"**——比正规公司还要正规。

走廊/隔间：105 室隔壁传来**讨价还价的争吵**：
- "特么的……为了卖这套武功，我手下有 2 个兄弟被罚到破产，1 个兄弟被公司肢解后卖了，你现在还给我压价？"
- "狗眼，你跟我凶也没用，帮派里的规定就是这样，7 级出课就是这个价。"
- "规定！规定！老子以前一个人去砸金课帮的场子，一个人点了他们十六个地下补习班，你们怎么不和我说规矩？"

**黑色幽默核心：暗学帮是深海教育集团名下的帮派组织，专门干"破解对手功法、打击敌对帮派"等脏活，是大公司手里的"夜壶"。**

关键剧情节点：
- 张羽以"我是来出课的"接暗号（**邪神教的行话**）
- 张羽在 105 室展演 10 级无极云手（**乳白色罡气化为云雾环绕——首次在帮派大佬面前展示实力**）
- 黑鸦登场（销售员，文质彬彬，礼貌引导）

光照：
- **园区外**：月光 + 园区门口招牌的冷白 LED 灯
- **前台接待区**：暖黄灯光 + 米色前台桌 + 微笑前台
- **105 室**：单灯顶光 + 中年人圆桌对面 + 张羽周身云雾形成"舞台追光"

关键道具：
- **"嵩阳北月高新技术服务业园区"招牌**（黑色幽默核心道具）
- **前台桌上的"暗学帮内部宣传册"**（画面与正规补习机构宣传册一致）
- **105 室内的桌椅**（黑鸦圆桌对面 + 记录本）
- **乳白色无极云罡**（张羽展演时的核心视觉）

## 📐 CG 制作要点

**镜头语言**：
- **园区大门镜头**：仰拍园区门口的"嵩阳北月高新技术服务业园区"招牌 → 摇到张羽意外的脸（**对比构图：园区 vs 张羽贫民窟服装的反差**）
- **前台镜头**：微笑前台特写 → 镜头拉远露出气派前台 → 张羽（伪装"马云腾"形象）站在接待区
- **105 室展演镜头**：张羽背对镜头 → 周身白色云雾涌出 → 镜头绕到张羽正面 → 露出坚毅的脸 + 10 级无极云手特写
- **隔间争吵镜头**：105 室门紧 → 门外走廊传来激烈争吵声 → 镜头特写门板震动的灰尘 → 门突然被关上 → 走廊恢复安静（**声音设计 > 画面设计的叙事手法**）

**特效需求**：
- **10 级无极云罡**首次以"商业展演"形态呈现（参考 SC-088-02 实战课的凶猛版本 → 本场景改为"压迫感弱化、教育感强化"版本）
- **暗学帮"企业内部管理" UI**：保安头顶数字（0、-122、-123、-124）以**AR 浮空字**形式呈现（参考场景 SC-059-02 威压符 UI）

**黑色幽默动画**：
- 园区门口的保安敬礼 vs 张羽想象中帮派的凶神恶煞（**强烈的视觉反差喜剧**）
- 前台小姐"我们的销售人员马上就会过去"vs 张羽以为会有打手拎着砍刀过来（**职业化 vs 黑帮化的反差**）
