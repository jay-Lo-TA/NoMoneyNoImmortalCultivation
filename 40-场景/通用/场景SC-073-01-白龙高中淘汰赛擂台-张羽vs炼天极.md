# 场景SC-073-01-白龙高中淘汰赛擂台-张羽vs炼天极（八强战+浴血奋战）

> **场景元信息**
> - 重要级：**P0 必做（武学突破 + 浴血）**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·八强擂台→4强擂台
> - 首次出现：第 73 章
> - 复用章节：第 73、74 章

## 🎬 场景描述

**P0 必做最高潮战斗 CG**。白龙高中武道竞赛**八强战**（第 73 章末）+ **张羽浴血奋战续战**（第 74 章整章）— **张羽 vs 紫云高中炼天极**。**本作武道竞赛篇最高光战斗 + 张羽无极云手 4→5 级突破的"战斗 CG" + 张羽"皮下罡气"战术的视觉化首次揭示**。

**建筑特征**：
- 八强擂台（4 个擂台之一）→ 战斗持续进入"擂台爆破"状态
- 比赛时间：长（突破极限的 30 秒 + 战斗总结 + 急送医疗）

**紫云高中炼天极**（**本作战斗狂人 + 广告带货主播**）：
- 紫云高中高一战斗王牌，**实力仅次于乐沐岚**
- **广告主合同**：1 分钟内必须解决张羽，否则扣广告费
- 修炼的功法：**专家级功法"爆裂波动拳" 3 级推动**（参悟自炸药爆炸，气流冲击波的拳法；**转化出的特种法力被称为"爆炸拳劲"**）
- 比赛前**打药 + 直播带货**：喝战斗亢奋剂、吃胶囊、注射战斗针筒
- 比赛前**租赁战斗灵根"涌动"**（灵机注入体内主动参与法力运转）
- **带伤也要赢**的广告主压力

**核心剧情**（**4 段递进**）：
- **第 73 章"广告直播带货 + 战斗起手"**：
  - 炼天极从背包里掏出各种药物 → 裁判竟然不阻止（"这还是昆墟吗？"）
  - 炼天极边打药边直播（"这个战斗亢奋剂我已经吃了三个月了"）
  - 紫云人拍摄广告 + 钱深等人看广告拍摄
  - 张羽"你特么能不能别尊重我了，直接小看我、看不起我不行吗？"
  - 比赛开始 → **爆裂波动拳的"爆炸拳劲"无法被无相云罡牵引**（"张羽的力量不如我强大，法力不如我雄厚，应该很快就抵挡不住了。"）
  - **张羽以不灭印法牵引挪移炼天极的法力**（"竟然能够牵引我的拳劲？这说明他除了无极云手之外，还擅长另一门在法力上能借力打力的高深武学。"）
- **第 74 章"浴血奋战 + 突破极限"**：
  - 炼天极发动战斗灵根"涌动"（灵机朝双手汇聚）
  - 爆裂波动拳 + 战斗灵根 = **"人形大炮"** → 爆炸冲击波隔空轰击张羽
  - 张羽被炸得浑身是血 → **仍然猛地撞碎冲击波反击**（"明明已经伤得这么重了还要继续战斗？你不怕破产吗？"）
  - **张羽的"皮下罡气"战术首次揭示**：无极云手 10 级 → 无相云罡能运转到身体各个位置而不伤害身体分毫 → 隐藏皮下 → 形成"护甲" → 表面血肉模糊但**实际不伤筋动骨**
  - 炼天极越急越用力 → 越凶猛 → **从张羽身上返回的力量越是汹涌** → 累计伤势
  - 炼天极"呕吐" → 右臂重创 → 继续战斗
  - 紫云人："广告主那边很不满意，如果剩下 30 秒钟内无法击败张羽，他们就要撤广告了……" → 炼天极爆发
  - **爆裂波动拳"自毁模式"**：以灵机和法力爆炸推动 + 以手臂重伤追求威力极限 → 右臂猛然膨胀
  - 双方拳掌剧烈爆炸 → 张羽再次被炸出血 → 推向擂台边缘 → **连退 4 步后稳稳站定**
  - 张羽微笑："谢谢你在这一战中给我的压迫，**终于让我的无极云手突破极限，达到了 4 级**"（第 73 章末尾实际是 5 级突破：原本只是覆盖在双掌部位的无相云罡突然一阵暴涨，蔓延到了手肘部位）
  - 炼天极苦笑坐倒
- **第 74 章"医疗结算"**：
  - 炼天极被急救队抬走
  - 张羽浑身是血也送去治疗 → "**等等！你先看看我这伤多少钱治？保险又能报多少？**"（**张羽怕赢了破产**）
  - 急救医生："伤这么重？这不治个百八十万对得起他的医学证吗？" → 检查发现"就点皮肉伤"
  - **5 千检查费 + 1 万 5 治疗费 + 1 万药费** → **保险覆盖**
  - 张羽："这么贵？" → "不治好点，你怎么参加接下来的比赛了？反正都走保险，你怕什么？"
  - 张羽心中叹："唉，这下保险费要涨了。"
- **第 74 章"楚秋河担架"**：
  - 楚秋河在担架上装晕（"这炼天极有点本事啊……"）
  - 担架上的学生战报："我赢了紫云的人，嵩阳高中的白真真打赢了虎云涛，然后她和我打……我输给了她。"
  - 楚秋河听到"输给了白真真"——"还好不是我一个人输" → "**张羽……你特么的再给我多赢几场啊！最好再赢几个白龙高中的人。**"（**本作最搞笑的反派角色楚秋河彻底成为张羽隐藏粉丝**）

**关键道具**：
- 炼天极的爆裂波动拳（爆炸气浪特效）
- 战斗灵根"涌动"（灵机朝双手汇聚的流光）
- 皮下罡气（**本作首次视觉化**：皮肤下淡白色流光）
- 直播设备（紫云人拍摄广告）
- 急救医疗区

**氛围**：暗红 + 紫色（紫云高中） + 白色无相云罡 — **极限战斗的视觉冲击**。

## 📐 CG 制作要点

- **空间设计**：
  - 八强擂台（4 个擂台之一）
  - 炼天极的紫云高中校服（紫色 + 直播设备）
  - 急救区在擂台旁
  - 担架设备
- **关键帧**（**P0 必做 / 4 段递进**）：
  - **广告直播带货** — 炼天极边打药边直播 + 紫云人拍摄广告（**本作首次"打药直播带货"CG**）
  - **爆裂波动拳 + 战斗灵根"人形大炮"** — 灵机朝双手汇聚 + 爆炸冲击波隔空轰击
  - **张羽"撞碎冲击波"反击** — 浑身是血（"明明伤得这么重还要继续战斗？你不怕破产吗？"）
  - **皮下罡气战术** — **本作首次视觉化揭示**：皮肤下淡白色流光（**本作最关键的"伪横练"战术 CG**）
  - **炼天极"自毁模式"** — 右臂猛然膨胀 + 漫天血雾 + 极限一拳
  - **张羽无极云手 4→5 级突破** — **从双掌蔓延到手肘**（"在绝境之下，他的无极云手又突破了？5 级了？"）
  - **张羽"赢了担心破产"** — 急救区"等等！你先看看我这伤多少钱治？保险又能报多少？"（**金融讽刺核心镜头**）
  - **楚秋河"装晕暗中为张羽加油"** — "**张羽……你特么的再给我多赢几场啊！最好再赢几个白龙高中的人。**"
- **特效需求**：
  - 爆裂波动拳的爆炸气浪（**本作首次"现代武学"特效**）
  - 战斗灵根"涌动"的灵机流光
  - **皮下罡气的视觉化**（**本作首次**：皮肤下淡白色流光）
  - 张羽无极云手 4→5 级突破的"云罡蔓延"
  - 急救法力的金色流光
  - 担架上的血迹（"流体特效"）
- **动捕需求**：
  - 炼天极的"爆裂+自毁"动作（**需要武打替身**）
  - 张羽的"浴血撞碎冲击波"动作
  - 急救医生的"报价 + 治伤"表演
  - 楚秋河"装晕"的喜剧表演
- **音频设计**：
  - 爆炸声（爆裂波动拳）
  - 灵机流光音（战斗灵根"涌动"）
  - 直播带货的"亲们看好了"广告语
  - 张羽"等等！你先看看我这伤多少钱治？"的喊声
  - 楚秋河"张羽……你特么的再给我多赢几场"的内心独白
- **建议时长**：8-10 分钟（**全作最长单场战斗 CG**，4 段递进）
