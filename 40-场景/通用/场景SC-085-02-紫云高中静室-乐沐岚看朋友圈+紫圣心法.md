# 场景SC-085-02-紫云高中静室-乐沐岚看朋友圈+紫圣心法

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市·紫云高中·乐沐岚私人静室
> - 首次出现：第 85 章
> - 复用章节：第 85、86 章

## 🎬 场景描述

紫云高中内部，乐沐岚的私人静室。**注意**：本场景是紫云高中内部，**与场景SC-024-02-嵩阳高中静室2号（嵩阳高中的静室）是不同的物理位置**——本场景属于乐沐岚个人，揭示紫云高中"高种姓"学霸的私人空间。

**建筑特征**：
- 约 20 平米私人静室，色调**纯净**（白 + 紫 + 金）
- 墙面是磨砂玻璃 / 软包吸音
- 中央有紫玉色的蒲团 / 软榻
- 空气中有淡淡的紫圣心法的灵气（紫色光晕）
- 紫玉色的箓书悬浮在乐沐岚掌心

**时间**：第 85 章末尾 + 第 86 章初，乐沐岚修炼紫圣心法 + 看张羽朋友圈的时段。

**关键视觉元素**：
- **紫圣心法的"有钱人圣洁、智慧、善良、强大，穷人污秽、愚蠢、邪恶、弱小"** 心法运转特效
- **银行余额 / 股票盈利的"数字"**（心法运转的"燃料"）
- **张羽朋友圈"被偷拍"**："又来到白龙高中啦，高一年级第一的宋海龙邀请我们来指导教学。"
- **乐沐岚的"分级克制"思考**：
  - "花钱让张羽来被我接触一下，摸一下" → 恶心 → 退回
  - "先去观摩两人的训练" → 恶心 → 退回
  - "先买几件张羽的衣服试试看" → 恶心 → 退回
  - **"反复思考后：先把张羽的朋友圈照片下载"**
- **从"接触身体"到"看照片"的分级脱敏**

**核心剧情**：
- 乐沐岚散去紫圣心法 + 努力停下"有钱人圣洁"的思想
- 原本心法：银行余额越大 + 股票盈利越多 → 心法运转越畅快 → 道心提升
- 但让她对"贫穷+穷人"的厌恶也越来越重
- 擂台上遭遇张羽 + 思想转变需要
- 加上张羽好友 → 点开朋友圈
- 第一条："又来到白龙高中啦，高一年级第一的宋海龙邀请我们来指导教学" → "宋海龙竟然花钱找穷人张羽来指导？"
- 第 86 章续：分步脱敏（"花钱让张羽被我接触一下" → 退回 → "先买衣服" → 退回 → "先下载照片"）

**氛围**：私人静谧 + 思想洁癖 + 紫圣心法视觉化 + 阶级差距讽刺。

## 📐 CG 制作要点

- **空间设计**：
  - 20 平米私人静室
  - 色调：白 + 紫 + 金（紫云高中的"高种姓"视觉）
  - 中央紫玉色蒲团
  - 紫圣心法灵气（紫色光晕弥漫）
- **关键帧（**P1 必做 CG**）**：
  - **紫圣心法运转的"数字"特效**：
    - 乐沐岚脑海中浮现银行余额、股票盈利
    - 数字越大 → 紫色光晕越浓
    - 同步浮现"圣洁、智慧、善良、强大"的字（与"污秽、愚蠢、邪恶、弱小"对称对比）
  - **散去心法 + 思想挣扎**：
    - 紫光逐渐散去
    - 乐沐岚面部表情挣扎（"有钱人怎么能害怕穷人？"）
  - **"分级脱敏"蒙太奇**：
    - 4 个分镜快速切换：
      1. "花钱让张羽被我接触一下，摸一下"（乐沐岚伸手的特写）→ 想到擂台 → 手缩回
      2. "先去观摩两人的训练"（站在远处看的剪影）→ 想到擂台 → 身体后仰
      3. "先买几件张羽的衣服"（手摸衣服特写）→ 想到张羽的汗渍 + 鸡皮疙瘩
      4. **"先下载张羽的朋友圈照片"**（手指轻点 → 朋友圈页面浮现）
  - **朋友圈页面浮空**：
    - 张羽 + 白真真在白龙高中门口的自拍
    - 文字："又来到白龙高中啦，高一年级第一的宋海龙邀请我们来指导教学。如果有想要指导的朋友，随时都可以联系。"
    - 乐沐岚的表情"目光一亮"（"40 万都不要，但花钱请上课却愿意吗？"）
- **特效需求**（**高优先级**）：
  - 紫圣心法的紫色光晕
  - 银行余额 + 股票盈利的"金融 UI"（数字跳字 + 金光）
  - "圣洁/污秽"等字的对称浮现
  - 朋友圈页面的浮空展示（**与 羽书 UI 不同的"社交媒体"风格**）
  - "分级脱敏"蒙太奇的快速切换
- **动捕需求**（**高优先级**）：
  - 乐沐岚的"洁癖挣扎"表演（4 个等级的分层情绪）
  - 散去心法的"放松 + 犹豫"表情
  - 朋友圈的"目光一亮"瞬间
- **音频设计**：
  - 静室的安静 + 紫圣心法的低频共鸣
  - 银行余额"叮"声
  - 朋友圈的"翻页" 音效
  - "分级脱敏"4 段快速对白
- **建议时长**：2-3 分钟
