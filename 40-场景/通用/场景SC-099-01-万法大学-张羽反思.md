# 场景SC-099-01-万法大学-张羽反思

> **场景元信息**
> - 重要级：**P1**（**张羽道德抉择关键场景 + 邪神反衬核心 CG**）
> - 物理位置：嵩阳市·嵩阳高中周边/上学路上
> - 首次出现：第 99 章（张羽背着白真真去学校的路上 + 邪神独白）
> - 复用章节：第 99/100 章

**首次出现原文（第 99 章）**：张羽背着白真真跑向学校的路上，张羽反思道——"虽然心中很爽，但张羽明白肚子里这灵根是白真真变得更强，考上十大的关键"+"回去以后就跟大家说我把租赁的灵根借给你用了"+"隐藏灵根、隐藏自己、武道修炼加速、实战看破绽、法力提炼加速以及她最近常用的炼体辅助功能"+"张羽也是听得双眼放光，恨不得立刻苦修一番"。

## 🎬 场景描述

**嵩阳市通勤路上/嵩阳高中周边**——张羽与白真真对"真灵根"命名的轻松场景，与邪神"杀人夺宝"建议形成强烈反差的反思时刻。

**核心空间元素**：

1. **张羽背着白真真奔跑**（**第 99 章 P1 必做 CG**）：
   - 跑车腿全力运转 + 白真真搂着张羽后背
   - 一边玩着箓书一边说灵根妙用
   - "灵机嗖嗖地往体内灌来，让张羽感觉身体一阵充实"
2. **张羽反思与拒绝**：
   - "行了阿真，这灵根你赶紧拿回去吧，别进进出出给弄坏了"
   - 张羽强忍不舍，把"价值几个亿"的真灵根还给白真真
3. **邪神独白与咒骂**（**第 99 章 P1 必做蒙太奇**）：
   - 伪装念珠的邪神全程旁观
   - 内心的咒骂："蠢货！立马动手杀人夺宝啊！"
   - "特么的，这就是穷鬼的劣根性，老是相信朋友、感情、信任这种虚无缥缈的东西"
   - "钱！只有钱才是最重要的啊！"
4. **灵根命名场景**：
   - 张羽提议："真真灵根" → 白真真："好土"
   - 改为"真灵根"
   - 决定对外宣称是"租赁的灵根"
5. **回到学校**：
   - 张羽开始翘法力课/实战课
   - 严老师/雷钧允许他缺课
   - 获得大量自由苦修时间
