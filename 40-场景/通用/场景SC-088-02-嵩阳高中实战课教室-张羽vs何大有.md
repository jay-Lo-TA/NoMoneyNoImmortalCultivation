# 场景SC-088-02-嵩阳高中实战课教室-张羽vs何大有

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市·嵩阳高中·实战课教室（武术教室）
> - 首次出现：第 88 章
> - 复用章节：第 88 章

## 🎬 场景描述

嵩阳高中实战课的教室——专门用于实战对练的武术教室。**注意**：本场景是**实战课教室**（**专门用于实战对练的固定教室**），**与场景SC-029-01-嵩阳高中武道实战考场-方格擂台（月考用方格划分）、场景SC-012-01-嵩阳高中体育练功场（体育课练功场）、场景SC-058-01-嵩阳高中体育练功场-高二学长打压实战（跨年级指导）、场景SC-059-01-嵩阳高中体育练功场-周澈尘对峙+威压符（高阶对峙）是不同的空间**——本场景是**日常实战课的固定教室**。

**建筑特征**：
- 教室约 50-80 平米
- 地面铺专业练功垫（与方格擂台不同的"软垫"风格）
- 墙上贴有"实战对练注意事项"
- 雷钧（刀疤中年壮汉）在讲台附近监控
- 学生们分散对练

**时间**：第 88 章末尾，雷钧实战课的最后一段对练。

**关键视觉元素**：
- **雷钧的"教训何大有"目的**：
  - 何大有"仗着暴涨的肉体强度横冲直撞，根本不将我教导的实战技巧放在眼里"
  - 雷钧"派出自己手下大将张羽，打算让对方知道战斗技巧和武技的重要性"
- **何大有的"暴涨 vs 张羽的技巧"**：
  - 何大有的肉体强度已经远在张羽之上（移植手术后的"有钱人 1.5+"）
  - 张羽的实战能力"太强"——何大有阴沉着脸
- **何大有"盘算"**："经过这段时间的植入和适应训练，我的肉体强度应该已经远在张羽之上了。不过这小子的实战能力太强……"
- **张羽的"秒杀"**：
  - 张羽一掌拍来 → 何大有下意识伸手一挡 → 张羽抓住何大有手
  - 拼力量 → 何大有另一只手也抓住 → 法力碰撞爆发
  - 何大有一声暴喝想举起张羽丢出界外 → "对方的双手坚若磐石，在运劲下纹丝不动"
  - **何大有的震惊**："你竟然敢和现在的我比拼力量？" → 但力量不济

**核心剧情**：
- 雷钧宣布分组："张羽，何大有，你们两个一组切磋"
- 张羽的"王者打倔强青铜"内心
- 何大有的阴沉脸 + "盘算"内心
- 张羽一掌拍来 → 双方抓住手 → 拼力量
- 何大有想举起张羽 → 张羽"坚若磐石" → 何大有震惊
- 故事在何大有的震惊中结束（留下悬念）

**氛围**：实战课日常 + 张羽的"绝对力量压制" + 何大有的"有钱也买不到技巧"反差 + 雷钧的"教训"目的。

## 📐 CG 制作要点

- **空间设计**：
  - 50-80 平米武术教室
  - 专业练功垫（与方格擂台不同的"软垫"）
  - 墙上的"实战对练注意事项"
  - 雷钧在讲台监控
- **关键帧（**P1 必做 CG**）**：
  - **雷钧宣布分组**："张羽，何大有，你们两个一组切磋"
  - **张羽的"王者打倔强青铜"**内心独白（特写）
  - **何大有的阴沉脸 + 盘算**：
    - 何大有"经过这段时间的植入和适应训练，我的肉体强度应该已经远在张羽之上了"
    - 阴沉的表情 + 内心盘算的浮字
  - **张羽一掌拍来**：
    - 张羽的"突然袭击"动作
    - 何大有下意识伸手一挡
  - **双方抓住手**：
    - 张羽抓住何大有手
    - 何大有另一只手也抓向张羽
    - 双方"手抓在一起"的特写
  - **法力碰撞爆发**：
    - 双方体内的法力涌出
    - 碰撞时的气浪
  - **何大有的"暴喝举"**：
    - 何大有一声暴喝 → 双臂肌肉暴起
    - 想把张羽举起丢出界外
  - **张羽的"坚若磐石"**：
    - 张羽"纹丝不动"的特写
    - 何大有的震惊（"什么！"）
  - **何大有的"你竟然敢"内心**：
    - 浮字"你竟然敢和现在的我比拼力量？"
  - **故事在何大有的震惊中结束**：
    - 镜头定格在何大有的震惊表情
- **特效需求**（**高优先级**）：
  - 法力碰撞的气浪特效
  - 何大有"举人"动作的肌肉暴起
  - 张羽"坚若磐石"的**金色罡气护体**
  - 何大有的震惊"什么！"浮字
- **动捕需求**（**高优先级**）：
  - 雷钧的"分组宣布"姿态
  - 何大有的"阴沉盘算"表情
  - 张羽的"一掌拍来"动作
  - 双方"抓住手"的擒拿
  - 何大有的"暴喝举"动作（双臂肌肉暴起）
  - 张羽"纹丝不动"的稳定
  - 何大有的"震惊"表情
- **音频设计**：
  - 雷钧的宣布声
  - 双方"抓住手"的闷响
  - 法力碰撞的气浪声
  - 何大有的暴喝
  - 何大有的震惊"什么！"
- **建议时长**：1-2 分钟
