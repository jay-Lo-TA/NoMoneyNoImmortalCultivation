# 场景SC-009-03-羽书 UI 浮空展示空间

> **场景元信息**
> - 重要级：**P2**
> - 物理位置：虚实叠加·羽书浮空 UI 空间

## 🎬 场景描述

这是主角张羽随时可用的"羽书面板"虚实叠加空间。当张羽心念一动，羽书就在他面前浮空展开，显示各种数据：道心等级、法力值、肉体强度、武功等级（健体三十六式/散手/高中基础剑法）、道术等级（基础吐纳法/基础炼心法/周天采气法）等。背景会根据张羽所在位置变化（食堂、广场、宿舍、学校等），前景永远叠加半透明的羽书 UI。该场景是"角色 status"的核心展示窗口，也是张羽做决策（找工作、修炼、参赛）的视觉锚点。

## 📐 CG 制作要点

- **空间设计**：半透明浮空 UI + 现实背景虚化
- **羽书 UI**：清晰的等级/数值/进度条，需要中文字体
- **关键帧**：从"健体三十六式 2 级（17/20）"到"健体三十六式 3 级（0/30）"的转化瞬间；"周天采气法 1 级 → 4 级"的多级跳转
- **动效**：等级提升时数字 + 进度条动效 + 微金光晕
- **建议时长**：每章 5-30 秒（蒙太奇特写镜头）
