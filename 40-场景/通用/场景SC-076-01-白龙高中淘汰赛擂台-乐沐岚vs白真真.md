# 场景SC-076-01-白龙高中淘汰赛擂台-乐沐岚vs白真真（四强战2）

> **场景元信息**
> - 重要级：**P0 必做（四强战 + 投降 + 金融核心）**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·四强擂台
> - 首次出现：第 76 章
> - 复用章节：第 76 章

## 🎬 场景描述

**P0 必做四强战 CG + 金融讽刺核心场景**。白龙高中武道竞赛**四强战**第 2 场 — **乐沐岚 vs 白真真**。**白真真"穷到无法参赛"主题的视觉化核心 + 乐沐岚"完美剑阵"的首次全面展示**。

**紫云高中乐沐岚**（**本作女性最高种姓"冰肌玉骨" + 紫云高中年级第一**）：
- **从开学那天起便始终是年级第一**（**紫云高中的"低分奴隶制"**）
- **定下的每日规矩**："所有高一学生在她面前都必须保持安静，650 分以下的贱民学生没她允许，不准和她说话。"
- **专家级功法"星斗剑术" 5 级推动**（**可跨越 5 米距离伤敌**）
- **"我的星斗剑术自成一阵，剑气森严有度，招式毫无破绽。除非有强横的护体功法强行冲撞过来，不然同级别的高手是不可能接近我的。"**

**核心剧情**（**白真真"穷到无法参赛"主题**）：
- 第 76 章开头**急救区结算**：
  - 白真真拒绝自费 3 万祛疤 → 9.8 万走保险 → **全身留下疤痕**
  - 白真真"技术还不错，感觉用个 8、9 成力问题不大。"
  - 雷钧面色复杂："你还打算继续参赛？" + "但你保险已经差不多扣完了，那个乐沐岚的打法我看过，**剑气锋锐，攻击范围也广，你跟她打……医疗费的账单很容易就能涨到十几万，甚至是几十万。**"
  - 雷钧劝："你现在已经进了四强，放在简历上也是很不错的成绩了，何必冒险继续比赛？"
  - 雷钧隐含想法："你毕竟不能考十大顶尖大学，武道竞赛的第一其实对你来说并没有那么重要。"
  - 张羽一瘸一拐走来："老雷你放心，我打的时候注意着呢，没伤到要害，都是些皮肉伤。"
  - 张羽："最多能从我姐那边借个 20 万，你看够不够你打下一场？"
  - 白真真："**羽子，你说我们要是有钱人多好。**"（**本作金融讽刺主题的"穷是实力"台词**）
  - 雷钧走后白真真看张羽："这一剑可能要花三、四十万……"
  - "要用几十万去冒险吗？"
  - "**她有勇气穿透熊文武狂风暴雨般的攻势，剑斩对方的身体破绽。但当要用几十万去赌一个胜利时，她感到了犹豫。**"
- **四强战**：
  - 乐沐岚看着白真真手脚上的丑陋疤痕 — "为什么留疤在身上？"
  - 白真真："没钱治啊。"
  - 乐沐岚震惊："这就是穷的样子吗？太可怕了。"（**本作最冷血的高种姓台词 + CG**）
  - 比赛开始 → 乐沐岚指尖连点 → **一道道锋锐剑气已经隔空斩出**
  - **5 级的星斗剑术跨越 5 米距离** → 白真真面色一变
  - 白真真试破体剑气格挡 → **每一道剑气之间更像是排兵布阵** → 复杂联系 → 围杀
  - 白真真进入潜行 → **格挡剑气、剑气和剑气的碰撞暴露了身形**
  - "对方的剑气实在太过迅捷和密集，以她的速度和爆发力，还没办法纯靠移动来进行闪躲。"
  - 看着白真真左支右拙的模样，乐沐岚："我的星斗剑术自成一阵，剑气森严有度，招式毫无破绽。"
  - 白真真心中一叹："**唉……擂台上直接用潜行还是太吃亏了，只有第一战用的时候最出其不意，后面对手有了防备就没那么有用了。**"
  - "破绽是有的。" → 但"如果我硬上的话，就算能赢也会被重伤吧？"
  - "**这一剑可能要花三、四十万……**"
  - **"要用几十万去冒险吗？"** → **白真真犹豫了**
  - "**于是几秒钟后，眼看着自己便要被剑气彻底包围之时，白真真选择了认输。**"
  - 白真真下擂台时眼中闪过释然："终究还是实力不足，太穷了。"（**本作白真真人物弧光的"现实投降"CG**）
  - "算了，把这破绽告诉羽子去吧，让好大儿替我教训这女人，他应该不用花那么多钱了。"
- **紫云高中休息区**：
  - 乐沐岚回到紫云高中休息区，看到跪在地上的炼天极
  - 乐沐岚直接坐在炼天极的背上："这便是她对炼天极失败的惩罚，接下来的三天时间里对方只要在她面前出现，都得当她的板凳。"
  - 乐沐岚对白真真评价："她没钱治疗了，**虽然就算有钱治疗也就是多浪费点时间**。"（**最冷血的高种姓台词**）
  - 炼天极心道："穷人输给富人，**理所应当**。"

**关键道具**：
- 星斗剑术的剑气（**本作首次冰蓝色剑阵**）
- 白真真的破体剑气（被星斗剑阵压制）
- 紫云高中的紫红色休息区
- 炼天极"当板凳"

**氛围**：冰蓝色（星斗剑术） + 暗红（白真真挣扎） + 紫色（紫云高中） — **冷暖对比的剑阵对决**。

## 📐 CG 制作要点

- **空间设计**：
  - 四强擂台（**整场比赛只有 1 个主擂台** — 第 76-77 章连续两场）
  - 紫云高中紫色休息区
  - 急救医疗区
- **关键帧**（**P0 必做 / 5 段递进**）：
  - **第 76 章开头"白真真断臂后讨价还价"** — "自费？我买保险你要我自费三万？"（**金融讽刺核心镜头**）
  - **白真真"羽子，你说我们要是有钱人多好"** — **本作金融讽刺主题的核心台词 + CG**
  - **乐沐岚"为什么留疤在身上？"白真真"没钱治啊"** — **高种姓 vs 穷人的阶级 CG**
  - **星斗剑术 5 米剑阵** — **冰蓝色剑气 + 排兵布阵 + 围杀**（**本作首次完整呈现"剑阵"概念**）
  - **白真真"擂台投降"** — "终究还是实力不足，太穷了。"（**白真真人物弧光的关键 CG**）
  - **乐沐岚坐炼天极当板凳** — **本作最爆笑的高种姓奴隶制 CG**
  - **乐沐岚"她没钱治疗了"** — **最冷血的高种姓台词 + CG**
- **特效需求**：
  - 星斗剑术的冰蓝色剑气（**本作首次冰蓝色特效**）
  - 星斗剑阵的"围杀"特效
  - 白真真破体剑气被压制
  - 断臂疤痕的视觉化
  - 紫云高中紫色调
- **动捕需求**：
  - 乐沐岚的"星斗剑术指法"（**需要武打替身 + 手指动捕**）
  - 白真真的"挣扎" + "投降"动作
  - 乐沐岚"坐炼天极当板凳"动作
  - 雷钧"讨价还价"对话表演
- **音频设计**：
  - 星斗剑术的"嘶——"冰蓝剑光音
  - 白真真"自费？"的怒吼
  - 雷钧"十几万甚至几十万的债务"语重心长
  - 张羽"你说我们要是有钱人多好"的叹息
  - 乐沐岚"为什么留疤在身上"的冷血台词
- **建议时长**：5-7 分钟（**金融讽刺核心 + 白真真人物弧光**）
