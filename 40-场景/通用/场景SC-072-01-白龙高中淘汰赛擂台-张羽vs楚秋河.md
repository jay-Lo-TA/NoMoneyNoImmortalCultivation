# 场景SC-072-01-白龙高中淘汰赛擂台-张羽vs楚秋河（卡车冲锋拳）

> **场景元信息**
> - 重要级：**P0 必做**
> - 物理位置：嵩阳市·白龙高中·地下一层武道竞赛场馆·淘汰赛擂台
> - 首次出现：第 72 章
> - 复用章节：第 72 章

## 🎬 场景描述

**P0 必做核心战斗 CG**。白龙高中武道竞赛**淘汰赛** 16 强战 — **张羽 vs 白龙高中楚秋河**。**张羽 vs 三大名校的首次正面对决** — 对方"用卡车冲锋拳 + 战斗补剂 2.2 级肉体强度"压制，被张羽以"无相云罡 10 级 + 全集中状态"反制。

**核心剧情**（**武学对决 + 金融讽刺双核心**）：
- 楚秋河打药（战斗补剂，**肉体强度临时提升到 2.2**）
- **专家级功法"卡车冲锋拳" 2 级推动**：古有武道强者观飞鸟走兽，山河日月而创造功法。如今自有各方强者观车水马龙，高楼大厦而创造出这个时代的功法。
- 楚秋河如一头猛兽 → "卡车冲锋"姿态（"犹如一辆大卡车般撞向张羽"） → 擂台砰砰作响
- 张羽的应对：**天武炼心诀 10 级"全集中状态"**（四周围一切瞬间进入极为缓慢的状态）
- 张羽以无相云罡正面拍 → 楚秋河的力量如一大群野猪蜂拥而来
- 张羽的"借力打力"：看清拳力走向 → 一点一点搅动无相云罡 → 牵引对方的力量 → 滑到一旁
- 楚秋河每一拳都被弹开 → **中门大开** → 张羽钻入怀里
- 噼啪炸响声中张羽-1 一连三击（胸口、肋下、小腹）
- 楚秋河打药感觉不到痛 → 热血上涌 → 双手合抱张羽 → 张羽**卡着极限距离**躲过
- 张羽连续命中 → 楚秋河"内脏好像出血了" → 一刹那僵直
- 张羽抓住僵直 → 上腹、胸口、肩、腰连续重击 → 楚秋河"变成巨大沙包" → 软软倒地
- 楚秋河脑补"输给了三大高中以外的学生"的黑暗未来 → 活活吓晕
- **此战张羽得胜！**

**白龙高中竞赛教练的慢镜头分析**（**本作首次战斗复盘 CG**）：
- "这个张羽一开始就站在了擂台边缘的位置。"
- "为的就是让黄巾力士主动出击之后，主动来到擂台边缘。"
- "接着他用了一招普通的直拳，然后就有意思了。"
- "他在那一瞬间，判断出了黄巾力士的出手，然后提前打断，再用无极云手牵引黄巾力士的力量，最终把黄巾力士摔向了几步之外。"
- "而这几步之外因为他的站位关系，正好就是界外。"
- "**前前后后，每一招都算计得恰到好处**……"
- 宋海龙反应："还以为这小子和法赛的时候一样是匹黑马，应该有点实力。结果喜欢用这些小聪明，真是没意思。"

**关键道具**：
- 卡车冲锋拳 2 级的"重卡气浪"特效
- 战斗补剂的注射器（楚秋河赛前打药）
- 慢镜头回放系统

**氛围**：暗调+卡车引擎的轰鸣 + 张羽白色无相云罡的"水球"质感 — **重 vs 柔的视觉对决**。

## 📐 CG 制作要点

- **空间设计**：
  - 淘汰赛擂台（**16 强阶段为 8 个擂台之一**）
  - 楚秋河的白龙高中校服（白色 + 胸口"排名数字"）
  - 战斗补剂注射器
  - 慢镜头回放系统（白龙高中教练旁的全息投影）
- **关键帧**（**P0 必做**）：
  - **楚秋河打药"战斗补剂"**（打药推进 2.2 级肉体强度）
  - **卡车冲锋拳"如大卡车撞向张羽"** — **本作首次呈现"现代城市武学"**（观卡车而创造的功法）
  - **天武炼心诀 10 级"全集中状态"** — 周围慢镜 + 张羽的"水球"无相云罡
  - **张羽"借力打力"** — 楚秋河每一拳都被弹开
  - **张羽钻入怀里"一连三击"**
  - **慢镜头回放系统**（白龙高中竞赛教练的分析镜头，**本作首次战术复盘 CG**）
- **特效需求**：
  - 卡车冲锋拳的"重卡气浪"特效（**本作首次"现代城市武学"特效**）
  - 战斗补剂的"血液循环"发光
  - 无相云罡"水球"质感
  - 慢镜回放系统的"全息投影"特效
- **动捕需求**：
  - 楚秋河的"卡车冲锋"动作（**需要武打替身**）
  - 张羽的"借力打力"动作（**本作张羽战斗风格的视觉化确立**）
  - 慢镜头回放系统的"分析"动作
- **音频设计**：
  - 卡车引擎的轰鸣（卡车冲锋拳发动时）
  - 战斗补剂的"嗞——"注射声
  - 慢镜回放系统的"滴答滴答"时间音
  - 楚秋河脑补"黑暗未来"的心跳声
- **建议时长**：2-3 分钟（战斗 + 慢镜回放 + 担架蒙太奇）
