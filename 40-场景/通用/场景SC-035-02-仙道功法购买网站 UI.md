# 场景SC-035-02-仙道功法购买网站 UI

> **场景元信息**
> - 重要级：**P2**
> - 物理位置：抽象 UI 层（昆墟仙道功法购物网站）
> - 首次出现：第 35 章
> - 复用章节：第 35、36 章

## 🎬 场景描述

张羽实名登录的昆墟仙道功法购物网站。视觉风格介于淘宝/京东和修仙秘籍之间：UI 框架是现代电商，但商品列表全是「长春益寿功」「搬砖王详解挖掘掌」「姹女大法男女通用版」「九天速成八部符箓」「不周搬山经」「心元大手印」「周天星斗剑」「光明心经」「天魔寄生法」等荒唐又真实的功法名。

**视觉重点**：
- 顶部推荐栏：「您的好友正在看：姹女大法男女通用版，大梦心经睡觉也能练功，九天速成八部符箓」
- 热销排行榜：挖掘掌、吊车爪、跑车腿等打工功法（揭示昆墟打工人真实生态）
- 中部推荐：「您的竞争对手正在下单：法力加分方略、基础八部符咒通用字典、天魔寄生法」（大数据刺激消费）
- 底部推荐：观看此功法的人还在看：反寄生法、净化神功诀、斩天魔之卷

整个 UI 风格是本作「黑色幽默 + 修仙讽刺」的核心载体之一，强烈建议采用手绘/手写体 + 浮空画轴风格，与现实电商拉开距离。

## 📐 CG 制作要点

- **UI 风格**：淘宝/京东骨架 + 修仙细节（图标用玉简、竹简、符箓、灵气图标）
- **关键帧**：
  - 张羽滑过排行榜时，热销功法名滚动（挖掘掌、吊车爪、跑车腿一闪而过）
  - 切换筛选条件（18-20 岁 → 功法等级：进阶级 → 专家级）的 UI 动效
  - 「您的竞争对手正在下单天魔寄生法」的弹窗出现在屏幕上，张羽手指停在屏幕上方
  - 「观看此功法的人还在看：反寄生法、净化神功诀」的列表展开动效
- **UI 元素清单**：
  - 顶部 logo（昆墟官方认证）
  - 搜索栏（带灵气光晕的输入框）
  - 推荐栏 / 热销榜 / 新品榜 / 筛选器（年龄 / 功法等级 / 价格）
  - 商品卡：功法名 + 价格 + 销量 + 标签（基础级 / 进阶级 / 专家级）
  - 购物车 / 收藏夹
- **动捕需求**：张羽滑手机的手指动作 + 表情（先嫌弃、后心动、后怀疑大数据杀熟）
- **建议时长**：90-120 秒（作为内心戏的主舞台）
