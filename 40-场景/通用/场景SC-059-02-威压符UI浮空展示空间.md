# 场景SC-059-02-威压符UI浮空展示空间

> **场景元信息**
> - 重要级：**P1 视觉差异化UI**
> - 物理位置：抽象UI层（威压符浮空展示空间）
> - 首次出现：第 59 章
> - 复用章节：第 59 章

## 🎬 场景描述

威压符是张翩翩跨越灵界施展的**特殊符箓**——恭请神力，**将施术者的"月考分数"和"银行余额"以浮空大字形式展示出来**，以威慑对手。

**两行字的内容**：
- **第一行**：「白龙高中高三年级月考总分：699 分」
- **第二行**：「银行余额：2」 → 逐位跳动 → **最终停在 25 亿**

**UI 设计核心**：
- **字体**：黑体加粗 + 镀金描边 + 顶部微光（参考《死亡笔记》L 的浮空文字）
- **布局**：上方为分数行（单行单数字），下方为存款行（多位数逐位跳动）
- **数字跳动节奏**：每秒精确切换一位 + 数字旋转入场 + 切换瞬间金光闪烁
- **反隐藏防护**：当他人用天眼符观察施术者其他信息时，显示「**你不配知道**」红色大字警告

**视觉重点**：
- **逐位跳动**的机械感 + **金光流动**的奢华感
- 全场观众**表情切换**的蒙太奇（钱深狂喜 / 赵天行数错 / 何大有震惊 / 蓝岭沉默）
- 张翩翩施术时的**"轻描淡写"**（表情不变 + 嘴角微勾）

## 📐 CG 制作要点

- **空间设计**：抽象UI层 —— 浮空于施术者头顶上方 1 米处的**两层数字字阵**，可旋转一定角度
- **关键帧**：
  - 镜头 1：威压符发动 → 张翩翩头顶金光闪起
  - 镜头 2：**699 分浮空登场**（特写 + 黑底金边 + 镀金光晕 + 慢速入场）
  - 镜头 3：**2 → 25 → 250 → 2500 → 2万5 → 25万 → 250万 → 2500万 → 2亿5 → 25亿** 的逐位跳动（每一秒切换一位的精确节奏）
  - 镜头 4：何大有发动天眼符 → 镜头切换至天眼符视角 → 显示「**你不配知道**」五个红色大字
  - 镜头 5：跳到 25 亿后金光**辐射全场**（地面反光 + 学生面部金光映照）
  - 镜头 6：张翩翩**嘴角微勾**的轻描淡写表情
- **特效需求**：
  - **镀金数字字阵**：参考《死亡笔记》浮空字 + 《JOJO 奇妙冒险》替身能力字阵
  - **数字逐位跳动的金光闪烁**：每一位数字独立粒子 + 切换瞬间光线扩散
  - **"你不配知道"的红色警告字**：参考游戏中的"权限不足"提示
- **动捕需求**：张翩翩施术时的"女王式轻蔑"微表情（眼睛斜视 + 嘴角勾起的弧度）
- **建议时长**：30-60 秒（**视觉差异化卖点场**——可以单独截取为短视频传播）
