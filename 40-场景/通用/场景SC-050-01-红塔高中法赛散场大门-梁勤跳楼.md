# 场景SC-050-01-红塔高中法赛散场大门-梁勤跳楼

> **场景元信息**
> - 重要级：**P0 必做**（**1-200 章季终钩子**）
> - 物理位置：嵩阳市·红塔高中·室内体育场散场大门外
> - 首次出现：第 50 章
> - 复用章节：第 50 章

## 🎬 场景描述

**1-200 章结尾、26 集半年番的季终悬念**。法赛颁奖结束后，各校开始散场。张羽+白真真与嵩阳高中的众人从会场大门走出去时，**一道黑影从体育场的顶部落下来**——"那是……是个人？"

**这是 P0 必做场景**——黑影是新角色伏笔，制造"下季核心谜题"。

**建筑结构**：
- 红塔高中室内体育场的**散场出口大门**（高耸金属门 + 上方透光顶棚）
- 大门外是**红塔高中校园空地**（白天 / 正午阳光）
- 周围有各家学校的学生、老师、教练、裁判、评委（红塔高中尤多）
- 体育场顶部高度数十米（跳楼者有足够的坠落空间）

**关键道具**：
- **数道罡气**（红塔高中老师们的反应）——瞬间化为**柔软的垫子**接住跳楼者
- **梁勤（跳楼者）**——被罡气托着缓缓送到地面的"被救者"
- 张羽手机中的"电子协议"（前十名无极云手使用权，奖金两万）

**名场面**（**本章高潮 + 季终定格**）：
1. **张羽看手机电子协议**（白真真吐槽"真抠门" + 张羽奖金 6 万 3 千多）
2. **各校散场**（嵩阳高中的众人走向大门）
3. **黑影从顶部落下**（张羽视角："是个人？"）
4. **数道罡气瞬间化垫**（红塔老师"爱护校园环境，人人有责"）
5. **梁勤被救**（闭目待死 → 被托到地面 → 痛苦："为什么要救我！"）
6. **梁勤身份揭露**（连输法赛赌局 + 父亲催收电话 + 绝望中跳楼）
7. **老师冷面回复**（"本校校区是禁止跳楼区域" + "失足？那也要赔钱"）
8. **梁勤的绝望挣扎**（"等找到机会了，我换个地方跳" / "这破世道，连想要免费跳个楼的地方都难找"）
9. **嵩阳高中的反应**（"张羽觉得那个被罡气包裹起来的人有些眼熟"——**重要伏笔**）
10. **季终定格**——张羽一行继续走远，镜头缓缓上升俯瞰红塔高中 + 梁勤

**氛围**：
- 前半段：**松弛庆祝**（颁奖、奖金、散场）
- 后半段：**骤转惊悚**（黑影落下、罡气化垫、跳楼者身份揭露）
- 收尾：**黑色幽默**（跳楼都要赔钱、禁止跳楼区域）+ **悬疑伏笔**（张羽觉得眼熟 + 跳楼者真实身份 + 嵩阳高中的未来冲突）

## 📐 CG 制作要点

- **美术风格**：与 SC-042-02 / SC-044-01 一致但**光线转变**（室内颁奖 → 室外正午阳光） + 情绪转变（庆祝 → 骤停）
- **镜头语言**（**季终级别的镜头调度**）：
  - **特写**：张羽手机屏幕的"电子协议" + "奖金到款 20,000"
  - **全景**：各校散场，嵩阳高中的队伍走向大门
  - **慢镜头 + 仰角**：黑影从体育场顶部落下（**季终标志性镜头**）
  - **俯拍 / 慢镜头**：数道罡气从四面八方升起、交织、化为"柔软的垫子"
  - **特写**：梁勤闭目待死的脸 → 被托住时的茫然 → 睁开眼的痛苦
  - **对切镜头**：张羽抬头看黑影 + 老师出手化罡气 + 梁勤被救起
  - **季终定格镜头**：镜头缓缓升高俯瞰红塔高中 + 张羽一行远去 + 梁勤一人被老师围住（**画面定格 3-5 秒**，暗示"这才是故事的开始"）
- **特效需求**（**本季最关键的视觉符号**）：
  - **黑影坠落的运动模糊**（不要让观众立刻看清脸，留下悬念）
  - **数道罡气**（白/青/金多种颜色）瞬间化为"床铺般柔软"的可视化
  - **梁勤被罡气托住**的"棉花糖"质感
  - **季终字幕** + **BGM**（建议使用悬疑 + 空灵的钢琴 + 大提琴）
- **动捕需求**：
  - **跳楼**的自由落体动作（慢镜头多次重复）
  - **罡气化垫**时的老师"双手平推"姿势
  - 梁勤**闭目等死** → **茫然睁眼** → **痛苦控诉**的情感层次
  - 张羽**回头一瞥**的关键表情（"有些眼熟"）
- **BGM 提示**：前半段轻快 → 黑影落下时**骤停**（1 秒静音）→ 罡气化垫的**英雄式鼓点**→ 梁勤被救后的**冷面幽默** → 季终镜头的**空灵长音**
