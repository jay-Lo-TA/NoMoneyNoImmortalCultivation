# 场景SC-090-01-张羽出租屋-邪神合作+黑暗补课界提议

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：昆墟第一层·张羽贫民窟出租屋
> - 首次出现：第 90 章
> - 复用章节：第 90、91 章（张羽夜奔前的准备阶段）

## 🎬 场景描述

**本场景是"邪神合伙人"关系确立的关键CG，也是"黑暗补课界"主线的破题CG。**

空间：昆墟第一层·张羽贫民窟出租屋（同 SC-089-01）。**但娃娃从破布猫大小，缩成拇指大小的黑色念珠，张羽用一根细绳将其套在脖子上——这是邪神在凡人社会的"伪装身份证"。**

氛围：从 SC-089-01 的对峙紧张 → 转为"精打细算的生意谈判"——娃娃语气转柔、用胡萝卜吊人；张羽则完全不吃这套，坚持"你在我手里，我才安全"。

关键剧情节点：
- 娃娃撤回"下跪"和"放开你"两个愿望（**第一次向张羽让步**）
- 娃娃**缩小成念珠挂在张羽脖子上**（**本作首次邪神"实体寄宿"CG**）
- 娃娃**透露嵩阳高中背后+白龙高中背后各藏了一只邪神**（**新副本伏笔**）
- 娃娃**传授"黑暗补课界"概念**——非法补课、不买传播权、反而赚更多（**金融讽刺核心场景**）

光照：夜晚出租屋，灯泡顶光，**娃娃在张羽掌中时为单一聚光效果**（类似珠宝广告镜头）。
关键道具：
- **黑色念珠**（邪神伪装形态，挂在张羽脖子上的"项圈"——既是寄生也是护身符）
- **张羽手机**（最后打开购物网站查询"挖掘掌、跑车腿"价格）
- **无相云罡**（张羽捏娃娃时涌出，10 级无极云手的"武装恐吓"）

## 📐 CG 制作要点

**镜头语言**：
- **变身特效**：猫咪大小布娃娃 → 一帧帧缩小 → 黑色念珠（**参考皮克斯《变身国王》的视觉缩放语言**，但要更工业化）
- **念珠特写**：挂在张羽脖子上的黑色念珠 vs 张羽眼睛（**双重机位：第一视角看向念珠、第三视角看向张羽胸前的"项圈"**）
- **手机 UI 浮空**：张羽打开购物网站，挖掘掌 / 跑车腿的功法价格以场景 SC-009-03 羽书 UI 浮空风格呈现

**特效需求**：
- **邪神缩小化**实体变身特效（参考《犬夜叉》四魂碎片变化为玉的视觉）
- **愿望撤回仪式特效**：张羽体内两道寒意同时消散的"反向烟雾"
- **嵩阳 + 白龙两所高中背后隐藏邪神的"暗色剪影闪回"**（世界观的恐怖氛围铺垫）

**动画节奏**：本场景是"金融讽刺 + 黑色幽默"密集区，需要节奏紧凑、对话信息密度高。可借鉴《瑞克和莫蒂》式的快节奏叙事。
