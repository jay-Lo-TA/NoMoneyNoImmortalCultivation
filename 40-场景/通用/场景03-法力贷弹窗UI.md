# 场景03-法力贷弹窗UI

> **场景元信息**
> - 重要级：**P0**
> - 物理位置：抽象UI层（法力贷应用界面 / 修仙贷推送弹窗）
> - 首次出现：第 1 章
> - 复用章节：第 1 章（及后续所有涉及法力贷 APP 的章节）

## 🎬 场景描述

法力贷 / 修仙贷的应用界面与推送弹窗。这是本作第一视觉差异化卖点，需要贯穿全剧。UI 设计融合修真符咒美学与现代金融 APP 视觉：
- **主界面**：法力值可视化（灵气体量计）、借贷额度、还款日息
- **弹窗**：新用户 30 天免息、日息半天功力的"修仙贷"宣传
- **签协议界面**：法力贷 / 修仙贷申请流程（绝育/器官抵押等特殊条款）
- **催款界面**：第 2 章后段频繁出现的逾期催收 UI

**关键视觉元素**：法力值（绿色灵气流）、还款日（仙道纪年 + 现实日历双标）、勾选条款（仿传统符咒阵图）、广告（仙人代言新用户免息）。

**氛围**：克制冷静的金融科技感 + 修真诡异感的混搭。需要让观众一眼看出"修真被彻底金融化"的世界观核心。

## 📐 CG 制作要点

- **美术风格（核心视觉差异化）**：
  - 主色：墨黑底 + 金色灵气流 + 红色警示
  - 字体：仿宋体（传统） + 等线体（现代）混排
  - 图标：符文化（莲花、灵剑、铜钱、勾玉）
  - 动效：灵气流动 + 数字滚动 + 符文闪烁
- **镜头语言**：
  - 镜头 1：手机屏幕全屏 UI 展示
  - 镜头 2：UI 与现实人物叠化（体现"修真被金融化"的隐喻）
  - 镜头 3：弹窗弹出瞬间的快速变焦
- **特效需求**：
  - 灵气粒子流动（绿色/金色/红色）
  - 数据滚动效果（法力值、还款日、利率）
  - 弹窗弹出 + 消失的过渡动画
- **UI 一致性**：必须与"羽书 UI"（场景14）形成视觉对比
  - 法力贷 UI = 冷色调金融 + 修符文
  - 羽书 UI = 暖色调古老 + 现代数据
