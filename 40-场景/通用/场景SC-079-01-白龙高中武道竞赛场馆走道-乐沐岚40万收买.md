# 场景SC-079-01-白龙高中武道竞赛场馆走道-乐沐岚40万收买

> **场景元信息**
> - 重要级：**P1**
> - 物理位置：嵩阳市·白龙高中·武道竞赛场馆内走道
> - 首次出现：第 79 章
> - 复用章节：第 79 章

## 🎬 场景描述

白龙高中武道竞赛场馆内、连接场馆主厅与休息区的一段没什么人的人行走道。走廊灯光冷白，靠墙一侧有长椅和垃圾桶，墙上是武道竞赛的赞助商海报（绿洲食品 + 微蓝医药）和各校晋级对阵表。走道尽头是通往场馆主厅的双开防火门，门缝透出主厅的嘈杂与欢呼。

**时间**：第 79 章 决赛开始前约 20 分钟，乐沐岚主动约张羽见面的间隙。

**关键视觉元素**：
- **冷白顶灯 + 赞助商海报**（商业氛围）
- **长椅、空垃圾桶、墙面对阵表**（冷清）
- **门缝透出主厅的暖光**（战场 vs 走廊的反差）

**核心剧情**：
- 乐沐岚开门见山："20万，我买你输，如何？" → 涨价到 40 万
- 张羽内心戏：40万 = 多少功法、灵根、符箓、训练器具 + 张翩翩的回复"竞赛第一自然比竞赛第二更有可能上十大"
- 仪式"哇哇叫"逼张羽拒绝
- 张羽回复："行啊，直接打我卡里吧" → 但乐沐岚提出签正神契约 → 张羽沉默 → 乐沐岚离去
- 张羽心中涌起"丢了一笔巨款的心痛"→ 运天武炼心诀斩断杂念 → "把乐沐岚那死有钱人按在地上狠狠暴打"

**氛围**：冷清走廊 vs 热闹主厅的反差；两个阶级（紫云学霸 vs 嵩阳穷人）的金融对决；黑色幽默+阶级讽刺。

## 📐 CG 制作要点

- **空间设计**：
  - 冷白顶灯的走廊
  - 墙上的赞助商海报（绿洲食品/微蓝医药的 logo）
  - 走道与场馆主厅的双开防火门（门缝透出主厅的暖光和欢呼声）
  - 长椅、垃圾桶、对阵表
- **关键帧（**P1 必做 CG**）**：
  - **乐沐岚开门见山**：直接画面切割，张羽 vs 乐沐岚 一米距离对视
  - **金额跳字**："20 万" → "40 万" 的数字以 UV 字特效浮空呈现（可参考 威压符 UI 浮空风格，但**金融乌托邦色调**——金 + 暖橙）
  - **张羽的"心痛"特效**：脑海中浮现功法、灵根、符箓、训练器具的购买清单 + 余额"4 万 5"被 40 万覆盖
  - **仪式"哇哇叫"** UI 浮现（与 羽书 UI 不同的"金融威胁"风格）
  - **天武炼心诀斩断杂念**：张羽闭眼，脑海中数字爆裂，化为"打"字
  - **门缝透出主厅光**：暗示"里面才是战场"
- **特效需求**：
  - 浮空金额数字（镀金逐位跳动）
  - 购买清单快速蒙太奇（功法/灵根/符箓图标 + 价格）
  - 张羽道心运转时的"斩字"特效
  - 门缝暖光（视觉锚点）
- **动捕需求**：
  - 乐沐岚的"自信报价"面部表情（"我出 40 万，你就该答应"）
  - 张羽的"心动→挣扎→被仪式催促→拒绝"情绪层次
- **音频设计**：
  - 走廊冷清的回声
  - 门缝透出的主厅欢呼
  - 浮空金额的"叮叮"声（金融数字音）
  - 仪式"哇哇叫"音效
