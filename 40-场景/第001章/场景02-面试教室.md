# 场景02 · 面试教室

> **所属章节**：[[30-剧情/章节摘要/第001章-面试]]
> **CG 重要级**：**P0**
> **预计时长**：5-6 分钟

## 📖 场景摘要

三位面试官审问张羽。谈到绝育手术、器官抵押贷款、性别转换等特殊入学条件。这是本作世界观的核心展示——金融化的仙道学院。

## 👥 出场角色

[[20-角色/主角/张羽]], 20-角色/配角/人物-42-嵩阳高中面试官三人

## 🎬 镜头设计

- 全景：教室空间（现代 + 符咒装饰）
- 中景：面试官三人组（仙门级气场）
- 中近：面试官提问（面部捕捉）
- 近景：张羽回答（礼貌笑容 + 紧张）
- 特写：金融 UI 弹窗（绝育、器官抵押、性别转换）

## ✨ 特效 / CG 重点

- 金融 UI 系统（法力贷广告）
- 现代教室 + 符咒融合装饰

## 🎨 道具

- 教室桌、面试官文书、金融 UI 屏幕

## 📝 CG 番剧改编要点

⚠️ 必做 P0：开篇定调。面试官面部捕捉需威严感。绝育/器官抵押等内容可用金融 UI 弹窗展示。
