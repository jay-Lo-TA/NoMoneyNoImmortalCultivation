## 📖 章节摘要

**P0 章节**：体赛结束后玉星寒主动来嵩阳休息区找张羽议价"陪练"（11000 元/小时成），白真真好友位标价 500 元被砍到 100 元；揭出李雪莲监管下玉星寒"贪墨补课费"的核心黑幕（合同要求 15000 实报 11000）、星火真人&李雪莲后悔没签张羽；张羽感叹"课多得安排不过来"，末尾张翩翩桥洞问出"两个月后让你上二层"。

**关键事件线**：

1. **白龙高中休息区·玉星寒让张羽**（P0 CG 开头）
   - 玉星寒看着天空柱第一的张羽："怪不得当初能被李师姐看重，虽然弟子选择上最终还是被我淘汰了，但也确实有着可取之处。"
   - 玉星寒看向宋海龙心想："可以换个陪练了。"
   - 宋海龙感受到玉星寒目光，心中后悔："我也许也该换个陪练对象了……想着自己之前不应该退掉张羽这边的陪练的"

2. **嵩阳休息区·嵩阳不跪白龙**（P0 CG 名场面）
   - 玉星寒主动走来嵩阳休息区
   - 嵩阳学生被白龙年级第一光环"逼得连连后退"
   - 女生小声问："白龙高中的习俗是要下跪的吧？我们要不要下跪迎接？"
   - 男生答："这里可是紫云高中，你们知道紫云拜见年级第一的礼节是什么吗？"
   - 何大有金句（P0 OP/ED 候选）：
     > "这里是嵩阳高中的休息区，那脚下就是嵩阳高中的土地，最多给这白龙人鞠个躬就行了。"
   - 白真真内心："一帮废物，还得是她这位体育竞赛第五的站出来和玉星寒交流"

3. **白真真好友位 500 / 玉星寒月薪 5000**（P0 CG 单元）
   - 白真真："好友位一个500。"
   - 玉星寒："那算了……其实我也没有那么想加你。"
   - 白真真连忙："那100，100行不行？"
   - 揭示玉星寒"既贫穷又富有"：星火真人&李雪莲投入无数资源，但每月可支配零花钱 5000 块
   - "500块的白真真好友位，李雪莲可能不会批，那对他来说就太贵了"

4. **玉星寒 11000 元/小时陪练张羽**（P0 CG 重点）
   - 玉星寒："陪我训练，一个小时给你6000，干不干？"
   - 张羽："15000一小时。"
   - 玉星寒："最多7000。"
   - 张羽："7000不可能，至少也要12000，我总不能比宋海龙低吧？"（宋海龙是 10000/小时）
   - 玉星寒："那11000，最多就这个价了，你不干我走了。"
   - 张羽："行行行，11000干了。"

5. **李雪莲监管"补课费"贪墨学费核心黑幕**（P0 揭露）
   - 玉星寒给李雪莲打电话申请 15000 陪练费："我跟他商量的是一小时15000的费用，你看能不能批一下？"
   - 李雪莲："可以，记得每次把你们的收费记录，还有交手的过程录下来上报给我。"
   - 玉星寒内心独白："这是他贪墨学费的机会。他自有办法应对。"
   - 关键伏笔：玉星寒曾请宋海龙陪练也是为了"贪墨学费"
   - 实际成交 11000，上报 15000，差价 4000/小时

6. **李雪莲&星火真人后悔没签张羽**（P0 CG 单元）
   - 李雪莲："就是当初那个时候，也许该将这个张羽一起签下来的。"
   - 星火真人回复："那个张羽看来是很不错，我当初是有点小觑他了，如果手头宽裕的话，确实该把他签了。"
   - 星火真人："但既然我们手头的资源只够培养一名弟子，就只能选择最优秀最有潜力的那一个。"
   - 星火真人："好了雪莲，过去的事情已经过去，不用患患得失，徒增烦恼。"

7. **张羽"课安排不过来"**（P0 核心）
   - 玉星寒走后，张羽又收到宋海龙陪练申请
   - 接着又收到乐沐岚："我能请你来陪我练习地面战吗？"
   - 张羽感叹（P0 OP/ED 候选）：
     > "唉，这课多得安排不过来，钱也要赚不过来了。"
   - 白真真："羽子，你要补不过来，那我能帮你代课啊。我现在可不是两个月前武道赛时候的我了。"
   - 张羽内心：白真真进步速度比宋海龙、乐沐岚都快，给他们做陪练没问题
   - 张羽内心：阿真也很需要这笔钱"未来才能跟我一起考进十大"
   - 张羽内心：姐姐当初要求两人包揽第一第二名，"今天晚上问问姐姐"

8. **末尾·张翩翩桥洞问出"两个月后上二层"**（P0 钩子）
   - 当晚比赛结束，张羽/白真真/张翩翩约见于桥洞之下
   - 三人站在河道旁，随意闲聊
   - 张翩翩突然问出（P0 章末悬念）：
     > "张羽，给你一个机会，两个月后让你一起和我上二层的话，你愿意吗？"

**关键数据**：玉星寒陪练 11000 元/小时（上报 15000）/ 白真真好友位 500 砍到 100 / 玉星寒月薪 5000 / 宋海龙陪练 10000 元/小时 / 两个月后上二层

**关键角色**：张羽 / 白真真 / 玉星寒 / 张翩翩 / 宋海龙 / 乐沐岚 / 何大有 / 王海 / 赵天行 / 李雪莲 / 星火真人
