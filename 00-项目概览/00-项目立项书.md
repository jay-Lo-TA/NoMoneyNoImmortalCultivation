# 📋 项目立项书 · 没钱修什么仙番剧

> **角色**：CG 番剧制片人
> **状态**：v0.1 草稿（待制片人细化）

## 1. 项目定位

### 1.1 一句话定位
将熊狼狗修真文明小说《没钱修什么仙？》改编为 26 集标准 TV 番剧，呈现"修真+现代金融"独特世界观。

### 1.2 项目类型
- **形态**：标准 TV 番剧（半年番）
- **单集时长**：15 分钟
- **总集数**：26 集
- **总时长**：约 390 分钟（6.5 小时）

### 1.3 目标观众
- **核心**：18-30 岁年轻人
- **次核心**：修真/玄幻题材爱好者
- **辐射**：泛二次元、金融题材跨界观众
- **原书粉照顾度**：待定（建议保留关键爽点 + 适当优化人设）

## 2. 核心卖点（USP）

| 维度 | 卖点 |
|---|---|
| **世界观** | 修真 + 现代金融体系（法力贷、天庭金融监管）—— **全网唯一** |
| **视觉** | 昆墟 36 层 + 18 层超级建筑 + 现代城市修仙 |
| **冲突** | 修仙 + 贫困 + 债务三角，张力天然 |
| **角色** | 张羽（极致逆袭 + 努力型男主）、白真真（进化型灵根女主） |

## 3. 改编范围

### 3.1 来源
- 原作：1-200 章（本次分析范围）
- 全本：1-906 章（远期可规划第 2 季）

### 3.2 第 1 季覆盖范围
建议覆盖原作 **1-200 章**（筑基资格考试前），对应 26 集内容。

### 3.3 重要级体系

| 档位 | 定义 | 应对 |
|---|---|---|
| **P0** | 主角、必出场场景、关键战斗、必建模角色 | 必须 1-2 集内出初稿 |
| **P1** | 重要配角、主要场景、关键法术 | 优先级次于 P0 |
| **P2** | 一般场景、过渡镜头 | 可简化、外包 |
| **P3** | 一次性背景 NPC、远景剪影 | 可延后/简化为剪影 |

## 4. 制作关键挑战

### 4.1 世界观可视化
- "法力贷"UI 设计（金融 APP + 修真符咒融合）
- "羽书"（外挂系统）UI 设计
- 昆墟建筑的视觉风格（金融大厦 + 仙山融合）

### 4.2 法术特效
- 普通法术（炼气期）：特效密度低
- 进阶法术（筑基+）：特效密度高，需更多预算
- 终极法术（金丹+）：关键战斗，必做 P0

### 4.3 等级视觉差异化
- 炼气期：朴素的"高中生修行"
- 筑基期：开始有"法力"感
- 金丹+：进入真正的"仙侠感"

## 5. 立项核心建议

### ✅ 立项
- 世界观独特性强，**强立项价值**
- 主角人设有吸引力（贫困 + 努力 + 复仇）
- 修真 + 金融的跨界题材有破圈潜力

### ⚠️ 风险
- 第 1 季只有"筑基前"内容，主角光环不够强
- 法术特效 vs 预算平衡
- 原书粉 vs 泛二次元的平衡
