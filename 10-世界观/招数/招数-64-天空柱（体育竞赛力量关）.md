# 招数-64-天空柱（体育竞赛力量关）

> **招数元信息**
> - 招数名：天空柱（紫云高中·体育竞赛第四关设备）
> - 所属体系：体育竞赛/力量测试/大型设备
> - 重要级：**P0 必做核心**（张羽 12531 公斤震撼 + 玉星寒 10000 公斤）
> - 首次出现：第 116 章（规则介绍）
> - **首次出现原文（第 116 章）**：第一轮选手各自上报目标重量，"然后依次登台擎起天空柱。就这样一共进行三轮比试"——**最终按三轮中的最优成绩评分**。
> - **关键原文（第 117 章）**：熊文武第二轮挑战 10000 公斤——"黑色龙鳞就好像是被撑开的衣服一样，裂开一道又一道血红的口子"。

## 📖 招数描述

紫云高中室内体育场的**标志性大型力量测试设备**——一根可调重量的巨型立柱，选手站在下方用双手撑住柱体底部。本作第 116-120 章体育竞赛第四关（力量关）的核心舞台。

**核心机制**：
- **三轮比赛制**：
  - 第一轮：热身 + 判断极限
  - 第二轮：接近极限的保底成绩
  - 第三轮：冲击极限 / 放手一搏
- **盲报机制**：每轮上报目标重量时**不知道其他选手的重量**（避免心理战）
- **不可用法力**：纯靠肉体力量
- **最终成绩**：三轮中的**最优成绩**

**重要比赛结果**：
| 选手 | 最高成绩 | 章节 |
|---|---|---|
| 玉星寒 | 第一轮未爆全力 | 117 章 |
| 熊文武（红塔） | 10000 公斤（第二轮拼命）| 117 章 |
| 白真真 | 8050 公斤（疤痕裂开流血） | 117 章 |
| 张羽 | **12531 公斤（精确到个位数）** | 118-119 章 |

## 🎨 视觉化要点

- **天空柱视觉**：**巨大的立柱缓缓压下**，像整个天空塌下来
- **张羽下方视角**："**抬头望去便有一种整个天空都被遮蔽的感觉**"（116 章）
- **天空柱数字**：柱体表面浮现大号金色重量数字（2000 → 12531）
- **熊文武 10000 公斤爆发**：**黑色龙鳞撑开+血雾+青筋暴起**（强烈血腥 CG）
- **张羽 12531 公斤全集中**：黑白世界 + 单指按住 + 双掌抗住山呼海啸（118 章）

## 💥 CG 特效需求

- **天空柱从天而降**：宏大场面 + 下方人员渺小对比
- **重量数字浮动 CG**：从 2000 到 12531 的渐变
- **张羽全集中战斗状态**：黑白两色 + 心脏跳动特写 + 筋肉一节节绷紧
- **白真真疤痕裂开流血**：红色疤痕如龙爪捏紧 + 血肉撕裂
