# 元磁针

> **首次出现原文已包含在原文件中**（子代理已带原文引用）


> **仙韵集团研发的辅助修炼设备** | 机械臂操控 + 法力检测贴片 | **"自动寻功"系统**

## A. 基础档案

- **法器名**：元磁针
- **类别**：辅助修炼法器（机械 + 仙道）
- **所属**：仙韵教育集团（万法宗下属）研发部
- **首次出场**：第 11 章
- **功能**：在法力运转路线上进行刺激，引导修炼者识别和纠正路线
- **操控方式**：机械臂自动操作
- **配合设备**：法力检测贴片（贴在双手、双脚、胸口、后背）

## B. 法器描述

- **外观**：银针状 + 连接线路
- **运作原理**：
  - 检测贴片时时监控修炼者体内法力运转状况
  - 元磁针根据法力运转数据，在机械臂操作下，对修炼者法力运转路线进行精准刺激
  - 通过刺激身体各部位，引导法力沿正确路线运转
- **目标**：让初学者能够练成复杂的"试验功法113号"
- **现实类比**：张羽称之为"自动寻路……哦不，自动寻功"
- **精度问题**：助手建议"元磁针指引精度需提升"，说明现有设备对初学者仍不够友好

## C. 元磁针检测室场景

### 场景要素
- **检测室**：现代化仙道研发场所
- **法力检测贴片**：贴在学生双手双脚胸口后背等位置
- **机械臂**：白色机械臂操作元磁针
- **数据显示屏**：
  - 各学生法力运转的实时情况
  - 练功图（汇聚学生各部位法力数据，描述法力运转路线）
  - 绿色 = 正确路线 / 红色 = 错误路线

### 试验过程
- **15 名学生**试功
- **法力路线错误率**：多数学生 → 错误越来越多 → 失败
- **坚持到底的学生**：3 人（其中张羽错误率高但坚持）
- **失败学生症状**：
  - 面色铁青透不过气
  - 惨叫一声疼晕过去
  - 吐出大口血
- **研究人员态度**："老师"（研究负责人）→ "死几百人又如何了？"

## D. 视觉化要点

### 元磁针本身
- **银针**：细长、表面光滑
- **连接线路**：从机械臂到学生身体
- **浮空**：机械臂操控，针尖可悬空接近学生身体
- **法力反馈**：针尖周围可能有淡青色法力波动

### 检测贴片
- **形状**：薄圆片或方形贴片
- **颜色**：银白色
- **位置**：双手、双脚、胸口、后背
- **功能**：检测法力运转数据 → 反馈给屏幕

### 机械臂
- **颜色**：白色或银白色
- **结构**：关节式工业机械臂
- **精度**：高精度定位
- **速度**：根据学生运功情况实时调整

### 显示屏
- **风格**：现代医疗设备 + 仙道符咒
- **数据流**：绿色（正确）红色（错误）流动的法力路线
- **练功图**：2D 的人形轮廓 + 法力运转路线叠加

## E. CG 特效需求

- **元磁针银针特效**：针尖的寒光 / 接近身体时的微弱电流感
- **机械臂运动**：关节转动的精度 + 银针入体的短暂画面
- **法力检测贴片**：贴片闪烁的频率 + 数据上传的光线
- **运功图可视化**：学生体内法力流动的绿色 / 红色线条
- **错误率警报**：红色高亮 + 警报音
- **法力失控特效**：红色血雾 + 体内溢出

## F. 与试验功法113号关系

- 元磁针是为试验功法 113 号专门研发的辅助设备
- 两者配套使用：**113 号提供路线 → 元磁针提供刺激**
- 设计理念："**113 号和元磁针本来就是为初学者研发出来的**"
- 失败原因：元磁针精度不足 + 113 号路线太复杂 → 初学者仍然失败

## G. CG 制作要点

### P1 关键CG（第 11 章元磁针检测室）
- **试功现场氛围**：紧张 + 痛苦 + 部分学生坚持的毅力
- **元磁针机械臂特写**：精准 + 仙道美学
- **运功图显示屏**：绿色 / 红色流动的法力路线（**本作"内化可视化"首次登场**）
- **失败者特写**：吐血 / 昏迷 / 痛苦表情

### 视觉差异化
- 仙韵集团办公大楼的现代仙道风格
- 元磁针的银白色机械臂美学
- 法力检测贴片的多点布置

### 配乐建议
- 紧张 + 压抑的电子音
- 元磁针刺激时的"叮"短音
- 失败者吐血时的低沉闷响

## H. 招数等级与价值

- **不属于"功法"**：是"辅助法器"
- **价值**：研发成本极高（仙韵集团级别）
- **使用场景**：仅限仙韵集团研发部试验
- **市场化前景**：若精度提升 + 113 号简化，可能成为初学者修炼辅助器材
