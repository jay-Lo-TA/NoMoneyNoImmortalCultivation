# 外置灵根（外接型灵根）

> **法赛财富差视觉化** | 宋海龙 100 分的根源 | 季终钩子金句触发器

## A. 基础档案

- **招数名**：外置灵根（External Spirit Root / 即插即用灵根）
- **类别**：道术装备 / 灵根外接器
- **首次出场**：第 44 章（宋海龙 100 分 + 拔"假发"）
- **CG 优先**：**P0 必做**（财富差距视觉化标志 + 笑点 + 季终钩子回响）

## B. 形态与原理

- **形态**：**宋海龙的"冲天长发"其实是外置灵根的伪装**
- **使用方法**：
  1. 装在头上
  2. **即插即用，随拔随换**
  3. 法力暴涨 + 灵机吐纳 + 法力提炼
- **与普通灵根区别**：
  - 学校里旧版本灵根 → 内置，**永久**
  - 外置灵根 → 外接，**临时**（但可随时更换更高级）

## C. 视觉化要点

- **宋海龙拔假发瞬间**（**法赛名场面 P0**）：
  - 张羽："卧槽！高中就秃了？"
  - 白真真："那不是假发，那是外置灵根。"
  - 张羽："外置灵根？我买不起，不过秃顶和假发我都能满足你。"
- **灵根启动效果**：
  - 灵机骚动（空气压力变化）
  - 法力如臂指使
  - 头顶冲天长发飘动（视觉财富差）

## D. CG 特效需求

| 元素 | 制作规格 |
|---|---|
| 假发拔下 | **P0 名场面**——特写 + 三人反应镜头 |
| 灵根装上 | 头顶发光 + 灵机涌动特效 |
| 100 分刷分 | 飞剑如闪电 + 全场惊叹 |
| 财富差对比 | 寒星野天眼符 UI + "校董之子"等评价 |

## E. 关联场景 / 角色

- **场景**：`[[40-场景/通用/场景SC-044-01-红塔高中法赛飞剑考场]]`（P0）
- **角色**：宋海龙（拥有）、白真真（**"这是我最想要的生日礼物"**）、张羽

## F. 重要细节

- **宋海龙用法**：**不是为了法力暴涨，是为了"锻炼法力控制力"**
  - 装上灵根 → 法力暴涨 + 难以控制
  - 强者把"难以控制"当作**训练机会**
  - 寒星野："100 分只是因为总分是 100 分"
- **白真真羡慕**："这是我最想要的生日礼物" + 想偷走的心理活动
- **张羽安慰**："外置灵根我买不起，不过秃顶和假发我都能满足你"
- **金句回响**：第 50 章季终，**张羽 + 白真真离开时，恰好看见从顶部落下的梁勤**——两人同是买不起外置灵根的穷人，**命运暗合**
