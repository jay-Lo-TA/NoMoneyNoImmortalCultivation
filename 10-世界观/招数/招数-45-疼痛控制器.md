# 疼痛控制器

> **嵩阳市月考·道心考试核心道具** | 手机形态 | 疼痛档位0~N级 | 张羽挺到37级极限

## A. 基础档案
- **道具名**：疼痛控制器（手机形态）
- **类别**：灵界道心考试设备 / 嵩阳市月考专属
- **首次出场**：第 31 章
- **使用者**：嵩阳市全体月考生（含张羽、白真真）
- **所属场景**：`[[40-场景/通用/场景SC-031-01-灵界-道心考试疼痛考场]]`

## B. 形态与机制
- 外观类似智能手机，正面屏幕显示疼痛等级档位（0~N）
- 仅有 "+" 和 "-" 两个按钮，分别上调 / 下调疼痛等级
- 灵界同步感知：按键 → 身体立即感受到对应等级疼痛（无皮外伤，但神经级痛觉）
- 6 小时考试，疼痛等级越高、维持时间越长，排名越高

## C. 疼痛等级阶梯
- **等级 1**：钢针刺入手掌
- **等级 2**：双手各一根钢针贯穿
- **等级 37**：双手双脚各贯穿 37 根钢针（张羽在此达到极限）
- **等级 N+**：超出极限 → 意识晕厥 → 强制踢出灵界 → 成绩作 0

## D. 考试规则
- 排名制：分数取决于"疼痛等级 × 维持时长"曲线
- 嵩阳月考 → 全市排名打分；高考 → 全昆墟一层排名打分
- 一旦晕厥 = 0 分处理（嵩阳高中"零分即至高羞辱"）

## E. CG 必做提示
- **P1 必做 CG**：第 31 章张羽道心翻盘
  - 控制器 UI 浮空数字跳动（0 → 37 → 60 分钟倒计时）
  - 等级 1~37 疼痛强度渐变特写（肌肉扭曲 + 血管暴起 + 汗珠写实）
  - 张羽狞笑瞬间：基础炼心法失效 → 天武炼心诀意念浮现 → 绝世武将虚影闪回
  - 李星宇分身注视 + 半空实时排名榜闪烁

## F. 视觉化要点
- 控制器屏幕用全息符文 + 数字双重显示
- 疼痛等级提升伴随身体发光颜色变化（淡红→深红→紫黑）
- 张羽在 37 级时的"狞笑"是本作核心面部表演 CG
