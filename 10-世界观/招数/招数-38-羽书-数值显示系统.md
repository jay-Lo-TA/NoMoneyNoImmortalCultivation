# 羽书·数值显示系统

> **首次出现原文已包含在原文件中**（子代理已带原文引用）


> **张羽的"金手指"外挂系统** | **本作"修真+游戏化"视觉差异化卖点** | 第 2 章首次登场

## A. 基础档案

- **法器名**：羽书
- **类别**：主角外挂法器 + 数值显示 UI
- **所属**：张羽专属（穿越自带 / 邪神仪式赠予）
- **首次出场**：第 2 章（羽书仪式 + 数值浮现）
- **视觉呈现**：浮空的法术符文 + 游戏化的数字面板
- **核心功能**：实时显示修炼进度 + 法力值 + 功法等级 + 加点选项

## B. 羽书的核心功能

### 数值显示功能
- **法力值显示**：
  - 第 11 章："周天采气法 0 级（0/1）" → "周天采气法 1 级（0/20）"
  - 第 12 章："周天采气法 1 级（11/20）" → "1 级（16/20）" → "2 级（0/40）"
  - 法力："7.7 → 7.8 → 8.3"
- **功法升级路径**：
  - 周天采气法：1 级（0/20）→ 2 级（0/40）→ 3 级（0/60）→ ... → 10 级（0/200）
  - 健体三十六式：1 级（0/10）→ 2 级（0/20）→ ... → 10 级（0/100）
- **走神率统计**：第 13 章法力课走神率 0%（张羽全勤第一）
- **现实货币**："存款 13700 → 3700 → 1700"（法力贷还款 + 神经填充剂消费）

### 经验灌输功能（关键 P0 卖点）
- **第 11 章**：张羽练成周天采气法 1 级后，"成百上千遍运转此功的记忆从他脑海中狂涌而出"
- **效果**：瞬间拥有大量修炼经验，相当于"习练此门功法已数月"
- **副作用**：身体因经验灌输而承受痛苦（"五内俱焚"）
- **意义**：让张羽在短时间内快速提升，是张羽"挂逼"特性的核心

### 加点功能（专精切换）
- **第 11 章**：张羽"将羽书的专精从健体三十六式切换到了周天采气法"
- **效果**：专精切换后，该功法升级效率显著提升
- **代价**：切换后其他功法升级效率降低
- **决策逻辑**：周天采气法练成后，健体 36 式暂时够用，专注周天采气法

### 任务倒计时
- **仪式力量倒计时**：吐纳 / 参悟等任务的倒计时数字
- **第 11 章**："请遵守仪式约定，努力完成愿望，不要故意偷懒拖延，10"
- **第 19-20 章**：双仪式倒计时（吐纳 + 参悟）轮番逼迫张羽

## C. 视觉化要点

### 羽书UI的视觉风格
- **浮空呈现**：法术符文组成的浮空界面
- **修真 + 游戏化**：
  - 类似 RPG 游戏的属性面板
  - 但符文风格融入传统仙道
  - 数字字体 + 修真符咒混排
- **半透明**：法术符文的发光字体
- **位置**：通常浮在张羽视野前方
- **交互**：张羽内心默念即可切换 / 升级 / 加点

### 数值显示的特效
- **升级时的数字跳动**：
  - "周天采气法 1 级（0/20）→ 1 级（11/20）" → 数字渐变 + 金光一闪
- **新功法习得**：金色文字浮现 + 文字扩散特效
- **经验灌输**：
  - "成百上千遍记忆狂涌" → 金色碎片从虚空飞入张羽脑内
  - 张羽表情：痛苦 + 忍耐
- **法力变化**：浮空法力值实时刷新

### 加点切换界面
- 羽书 UI 上功法列表
- 选中某功法时，金光环绕
- 切换时金光一闪 + 旧专精褪色

### 仪式倒计时
- 红色倒计时数字
- 寒意 + 张羽感受到的催逼感
- 数字从 10 → 0 倒数

## D. CG 特效需求

- **羽书 UI 浮空展示**：法术符文 + 数字 + 半透明
- **数字跳动特效**：金币滚动 + 金色渐变
- **经验灌输特效**：金色记忆碎片 / 漩涡 / 涌入脑内
- **法力值刷新**：浮空数字实时变化
- **仪式倒计时**：红色数字 + 寒意氛围
- **加点切换金光**：选中功法时的金色光环

## E. 羽书的"金融 UI"扩展

### 法力贷弹窗
- **第 1 章首次登场**：法力贷 APP 弹窗
- **功能**：借款 / 还款 / 利息计算
- **UI 风格**：现代金融 APP + 修真符咒
- **弹幕式广告**："你欠 X 功力"

### 灵币交易
- **货币类型**：灵币（普通修士交易）类似现代纸币 + 符咒
- **数字余额**：APP 显示存款 / 借款 / 利息
- **催收短信**：第 18 章"欠呗欠款"+ 第 20 章白真真的"800 万欠款"幻境

## F. CG 制作要点

### P0 视觉差异化卖点
- **本作"修真+游戏化"的视觉核心**：所有数值显示都通过羽书 UI
- **首次登场（第 2-5 章）**：建立读者 / 观众对羽书 UI 的期待
- **持续强化（第 11-20 章）**：周天采气法升级 / 法力提升 / 加点切换
- **画面差异化**：现代修真番中罕见的高密度数字 UI

### 与"传统修真"的对比
- 传统修真：内观感应 / 师父口传心授
- 羽书系统：数据化 / 视觉化 / RPG 式
- **视觉冲突感**：传统水墨 + 现代 UI 数字 = 本作独特风格

### UI 设计参考
- **游戏 UI**：《原神》属性面板 / 《最终幻想》状态栏
- **修真 UI**：《凡人修仙传》灵兽属性 + 数值
- **金融 UI**：《赛博朋克 2077》账户管理 + 信用评分

## G. 羽书的"代价"

- **邪神仪式附身**：羽书是张羽与邪神仪式的连接
- **倒计时机制**：逼迫张羽学习修炼，否则受到寒意 + 数字倒数惩罚
- **本质**：张羽穿越后与邪神签订契约，邪神通过羽书监视张羽
- **隐藏真相**：羽书来源 / 邪神身份是 200 章后的核心悬念

## H. 与张羽关系

- 张羽穿越自带 → 第 2 章羽书仪式浮现 → 第 5 章完善数值系统
- 张羽依赖羽书：
  - 监控修炼进度（防止走火入魔）
  - 加点切换（专精效率提升）
  - 经验灌输（练成功法后瞬间获得大量经验）
- 张羽的"挂逼"特性 = 羽书 + 仪式倒计时 + 经验灌输
