# 康灵针（白龙高中特供）

> **法赛财富差视觉化** | "一针康灵入丹田，我命由我不由天"

## A. 基础档案

- **招数名**：康灵针（Kangling Needle / 灵机精华浓缩针）
- **类别**：物品 / 药物
- **首次出场**：第 46 章（法赛第二关·化解考场）
- **CG 优先**：P1（白龙高中财富差 + 贵族装备 CG）

## B. 设定

- **外观**：保温盒 + 两排针剂 + **散发荧光的针管**
- **功效**：极速恢复体内法力（化解同时补充）
- **使用**：对着丹田位置一针扎下
- **价格**："我命由我不由天，这辈子不知道能不能打得起康灵针"

## C. 视觉化要点

- **保温盒开启**：寒气扑面 + 荧光反射
- **针剂取出**：两排整齐 + 散发寒光
- **扎针瞬间**：宋海龙（1 号）第一个拿起 + 胸口数字 1
- **围观反应**：其他学校学生"满脸嫉妒" + "这辈子不知道能不能打得起"

## D. CG 特效需求

| 元素 | 制作规格 |
|---|---|
| 保温盒 | 金属保温盒 + 寒气飘出 |
| 针剂 | **P1**——荧光反射的细节 + 高科技质感 |
| 扎针动作 | 1 号宋海龙 + 顺序感（按胸口数字） |
| 围观反应 | 其他人眼睛发光的特写 |

## E. 关联场景 / 角色

- **场景**：`[[40-场景/通用/场景SC-046-01-红塔高中法赛第二关化解考场]]`（P1）
- **角色**：宋海龙（1 号）、寒星野（教练）、其他学校学生
- **对比招数**：`[[10-世界观/招数/招数-14-紫云公司拓脉注射液]]`（紫云赞助）

## F. 重要细节

- **白龙高中标配**：宋海龙等十人按胸口数字 1-10 顺序取针
- **紫云高中装备**：拓脉注射液（拓宽经脉）+ 一把药吞服
- **红塔高中装备**：兽药（给熊不凡"妖裔学生"专用）
- **嵩阳高中**：张羽"假装吃药去厕所"
- **金句回响**："一针康灵入丹田，我命由我不由天"——穷人的呐喊
