## 🎯 收录标准

凡是符合以下任一标准的物件，应优先归类到 `25-道具/`：

1. **角色专属金手指**（如羽书——只有张羽能使用）
2. **角色专属契约/印记**（如邪神布娃娃、掌心灵界印记——绑定特定角色）
3. **关键官方信物**（如张翩翩给张羽的工作箓书）
4. **可复用的功能性装备**（如灵界面罩、元磁针——多次出现在不同场景）
5. **P0 必做 CG 道具**（番剧制作中需要专门建模/特效设计的核心物件）

不属于本目录的物件：

- ❌ 通用武器/防具（散布在 `10-世界观/招数/`）
- ❌ 一次性龙套物品（如某章用过的临时道具）
- ❌ 已经作为角色本体特征的东西（如白真真的真灵根算"属性"而非"道具"，可酌情入库但优先级低）
