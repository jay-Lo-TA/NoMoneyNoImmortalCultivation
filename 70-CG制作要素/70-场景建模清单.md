# 🏙️ 场景建模清单

> 重要级体系：**P0 必须 / P1 重要 / P2 普通 / P3 背景**

## P0 核心场景（必做）

| 场景 | 重要级 | 出现集数 | 备注 |
|---|---|---|---|
| 嵩阳高中面试现场 | P0 | 1 | 开场 |
| 万法大学主楼（外观） | P0 | 1+ | 多集 |
| 张羽宿舍 | P0 | 多集 | 日常戏 |
| 玄冥重水考试场 | P0 | 25-26 | 关键高潮 |
| 金融系办公区 | P0 | 多次 | 链枢神君主场 |
| 福姬仪式空间 | P0 | 10 | 邪神契约 |

## P1 重要场景

- 各系教室（土木 / 炼器 / 道术 / 金融）
- 筑基考试场地
- 战斗场地（多次复用）

## P2 普通场景

- 食堂、街道、商店
- 路人甲背景
- 过渡场景

## P3 背景

- 远景剪影
- 一次性背景
- 蒙太奇

## 建模规范

- **P0 场景**：高细节 + 可拆分
- **P1 场景**：中细节 + 模块化
- **P2/P3**：低细节 + 复用
