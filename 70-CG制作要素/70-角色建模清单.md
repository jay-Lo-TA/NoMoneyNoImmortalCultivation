# 👤 角色建模清单

> 重要级体系：**P0 必须 / P1 重要 / P2 普通 / P3 背景**

## P0 核心角色

| 角色 | 建模难度 | 备注 |
|---|---|---|
| 张羽 | 高 | 主角，需多套服装（校服 / 战斗 / 变身后） |
| 白真真 | 高 | 女主，造型需唯美 |
| 高崇光 | 中 | 化神级法骸装备（300+件，简化表示） |
| 萧玉容 | 中 | 炼器系主任，磁极气场 |
| 链枢神君 | 中 | 反派，金融系 + 邪教风格 |
| 道乾坤 | 中 | 仙气造型 |
| 福姬 | 中 | 邪神风格 |

## P1 重要配角

- 灵岳真君

- 夜星璃

## P2 普通配角

- 群众演员
- 各系主任

## P3 背景

- 路人甲
- 一次性背景 NPC

## 建模规范

- **主角分辨率**：高模 + 法骸 / 服装切换
- **次主角分辨率**：中模 + 简化版
- **P3 角色**：低模 + 剪影处理
